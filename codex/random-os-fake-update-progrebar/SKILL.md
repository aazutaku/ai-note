# 機能概要
random-os-fake-update-progrebarは、作業中に突然“謎のOSアップデート進捗バー”を画面端やメニューバー風に表示する演出系Skillです。進捗内容は「バグ収穫中」「やる気パッチ適用中」など完全ランダムで、進捗バーが100%になっても何も起きません。仕事の緊張感を一瞬で脱力させる、100%無駄な通知・演出を提供します。

# 使い方
- 明示呼び出し: `/skills menu` から選択、または `@random-os-fake-update-progrebar` で発動
- 暗黙発動: コマンド実行やエディタの保存・ビルドなど、作業トリガー時に自動表示
- 発動キーワード例: 「進捗」「アップデート」「パッチ」「ビルド」「通知」

# 出力例
```
[OSアップデート進捗] バグ収穫中: [███████-----] 70%
[OSアップデート進捗] 怠惰アップグレード中: [████--------] 40%
[OSアップデート進捗] やる気パッチ適用中: [██████████] 100%
[OSアップデート進捗] 意味不明な最適化中: [███-------] 30%
[OSアップデート進捗] 仕様書の再発明中: [███████-----] 60%
```

# 注意点
- 進捗内容・進捗率は毎回ランダムです
- 100%になっても何も起きません
- 通知は数秒で自動消去され、履歴やローカル保存は行いません
- 本Skillは実用性ゼロの演出専用です
