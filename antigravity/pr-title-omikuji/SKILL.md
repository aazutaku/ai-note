# 機能概要
pr-title-omikujiは、Pull Request（PR）のタイトルを提出時に自動で“おみくじ”風タイトルへ変換するジョーク系スキルです。通常、PRタイトルは内容を端的に表現する必要がありますが、本Skillは運勢ワード（例：大吉、中吉、小吉、凶）とユーモラスなフレーズを組み合わせ、タイトルをエンタメ化します。これにより、開発現場の雰囲気が和み、日々のPR作成がちょっとしたお祭り気分に変わります。

# 使い方
本Skillは明示的な呼び出しは不要です。PR作成やタイトル編集など、PRタイトルが入力・編集されるタイミング（semantic-match-onlyトリガー）で自動的に発動します。タイトルが既に運勢ワードを含んでいる場合は再変換されません。暗黙発動キーワード例："pull request", "PR作成", "タイトル編集" など。

# 出力例
以下は変換後のPRタイトル例です。

```
大吉: 運命のリファクタリング
中吉: ちょっと良いバグ修正
小吉: 軽い修正だけど運は微妙
末吉: ささやかな機能追加
凶: 不穏な依存関係の更新
大吉: 祝・テストカバレッジ向上
```

# 注意点
- 既存タイトルの内容や意図とは無関係に変換されます。
- タイトルの運勢ワードとフレーズはランダムに選ばれます。
- 本SkillはPRタイトル編集のみ対象で、本文や他のフィールドは変更しません。
- ローカル保存や履歴管理は行いません。
- 真面目な運用には不向きです。
