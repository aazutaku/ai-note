# 概要
本Skillは、作業の合間や気分転換を目的とした“偽ウイルスアラート通知”をデスクトップに表示するジョーク系ツールです。実際のウイルス検出やシステム変更は一切行いません。

# 公式ドキュメント抜粋
通知にはPythonの[plyer.notification](https://plyer.readthedocs.io/en/latest/#plyer.notification)を利用。Windows/macOS/Linuxの主要デスクトップ通知に対応しています。

# 利用例
- 長時間のコーディングで集中が切れた時の気分転換
- チーム内のジョークとして
- オンライン配信や勉強会での演出

# 注意点
- 本物のウイルス通知と誤認されないよう、内容は明らかにネタ・冗談で構成されています
- 通知はローカルPCでのみ表示され、履歴やログは保存されません
- 通知頻度は作業の邪魔にならないよう、デフォルトで15分〜1時間間隔

# 設計方針
- 実在APIのみを使用し、環境構築不要（pip install plyer のみ）
- ユーモア重視、悪質な演出やシステム影響ゼロ
- スキル本体はCLIサブコマンド形式で明示・暗黙両対応
