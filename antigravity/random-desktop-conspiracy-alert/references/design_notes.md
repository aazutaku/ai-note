# 概要
このSkillは、開発現場での気分転換やチームの雰囲気づくりを目的に、突発的な陰謀論風通知をデスクトップに表示します。内容は完全なジョークであり、実害や実際の警告はありません。

# 公式ドキュメント抜粋
- Python subprocess: https://docs.python.org/3/library/subprocess.html
- Linux notify-send: https://specifications.freedesktop.org/notification-spec/latest/

# 利用例
長時間のコーディングや単調な作業の合間に、意味不明な通知で気分転換を図ることができます。チームで導入すれば、会話のきっかけや笑いを誘発する小ネタとしても活用可能です。

# 注意点
- 通知頻度は1時間に1回に制限されています。
- 通知内容はシステムに保存されませんが、履歴はホームディレクトリにログとして残ります。
- OSの通知機能が必要です。Windowsでは標準対応していません。

# 設計方針
ユーザー体験を損なわないよう、過剰な通知や誤作動を避ける設計とし、通知内容のバリエーションや履歴管理機能も実装しています。
