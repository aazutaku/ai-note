# 概要
random-os-fake-error-prophecyは、作業中のコマンド実行や編集操作のたびに、未来を予言する架空のOSエラーメッセージをランダムで表示することで、ユーザーの気分転換やコミュニケーション促進を狙った演出系Skillです。

# 公式ドキュメント抜粋
本Skillは実際のエラー通知とは異なり、システムやファイルには一切影響を与えません。出力は全て架空の予言であり、履歴はユーザーホーム配下のJSONファイルに最大50件まで保存されます。

# 利用例
- ターミナルでgitやmakeコマンドを実行した際に、突然予言メッセージが表示される
- コードレビューやpush時に、未来の出来事を仄めかす警告が出て話題になる

# 注意点
頻度は30%程度に抑え、作業フローを妨げない設計です。実用的なエラー通知と混同しないよう、現実にはあり得ない内容・表現に限定しています。

# 設計方針
トリガーキーワード検出、履歴保存、サマリ表示などを備え、拡張性と安全性を意識しています。
