# 概要
terminal-boss-bgm-suggesterは、日々のターミナル作業に遊び心を追加するために設計されました。最初のコマンド実行時のみランダムなBGMを提案することで、過剰な通知や環境破壊を防ぎつつ、作業の始まりに小さな演出を提供します。

# 公式ドキュメント抜粋
YouTube Data API v3（https://developers.google.com/youtube/v3）は、動画の検索や取得に利用できますが、本Skillでは事前に選定した公式動画リンクを利用し、APIキーや外部通信を不要としています。

# 利用例
- 朝イチのコマンド実行時に「情熱大陸」などを提案
- 明示的に`/terminal-boss-bgm-suggester force`で気分転換

# 注意点
- 履歴はローカル（~/.claude_terminal_bgm_history.json）にのみ保存され、プライバシーに配慮
- 複数端末利用時は端末ごとに発動

# 設計方針
- シンプルなローカルJSONで日付判定
- テキスト通知のみで音声再生やポップアップは行わない
- 曲リストは著名なBGMを厳選し、リンク切れ時は随時修正可能
