# 概要
random-os-fake-mascot-notifierは、作業中のユーザーに対し、完全ランダムなタイミングで架空のOSマスコットが通知を送ることで、作業の合間の現実逃避や気分転換を促進します。通知内容・キャラクターともにバリエーション豊富で、通知理由やタイミングもカオスに設計されています。

# 公式ドキュメント抜粋
通知表示にはPythonのplyerパッケージ（https://plyer.readthedocs.io/en/latest/）を利用し、クロスプラットフォームでデスクトップ通知を実現しています。乱数生成は標準のrandomモジュールを使用。

# 利用例
- 長時間のコーディングや単調作業の合間に、無意味だが癒される通知が届きます。
- ターミナルから `python mascot_notifier.py daemon` で常駐させると、一定間隔で自動通知。
- `log` サブコマンドで1回だけ即時通知も可能。

# 注意点
- plyerの通知は一部Linux環境やWSLでは表示されない場合があります。
- 通知は完全にランダムで、実用的な情報やアドバイスは含まれません。
- 履歴はメモリ上のみで永続化されません。

# 設計方針
「役に立たない通知」「完全ランダム」「ゆるキャラ感」を重視し、実在のOSや企業キャラとは無関係なオリジナルマスコットを多数収録。通知頻度は15分〜1時間の範囲で自動調整し、うるさすぎない設計としています。
