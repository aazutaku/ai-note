# 概要
commit-dajare-generatorは、コミット時に自動でダジャレ混じりのメッセージを生成するスキルです。主に日本語圏の開発現場で、コミュニケーションの活性化や雰囲気の緩和を狙っています。

# 公式ドキュメント抜粋
- git公式: https://git-scm.com/doc
- Python subprocess: https://docs.python.org/ja/3/library/subprocess.html
- pykakasi(日本語形態素解析): https://github.com/taizan-hokuto/pykakasi

# 利用例
- チーム開発でコミット履歴をユーモラスにしたい場合
- 朝会やレビューで「今日のダジャレ」をネタにしたい場合
- コードレビューの緊張を緩和したい場合

# 注意点
- ダジャレの質や“寒さ”は保証されません。状況に応じてご利用ください。
- 英語圏のファイル名やdiffには一部対応が限定されます。
- ログファイル(.commit_dajare_log)はローカル保存のみ、共有リポジトリには含めないことを推奨します。

# 設計方針
- ファイル名・diff内容からキーワード抽出→辞書ベースでダジャレ生成
- 拡張性を考慮し、辞書や生成ロジックは容易に追加・修正可能
- CLIサブコマンドでログ閲覧・サマリーもサポート
