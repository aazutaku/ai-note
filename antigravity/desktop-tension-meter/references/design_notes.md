# 概要
このSkillは、ユーザーの作業中にデスクトップへ無意味な「緊張度」テロップを表示することで、遊び心ある演出体験を提供します。根拠のないラベルをあえて用いることで、仕事やPC作業に謎のプレッシャーを演出します。

# 公式ドキュメント抜粋
本SkillはPython標準のTkinterを利用し、ウィンドウの位置・表示・非表示制御を行います。詳細: https://docs.python.org/3/library/tkinter.html

# 利用例
- 長時間の作業に飽きたときの気分転換
- オフィスでの話題作り
- 意味のない演出で集中を妨げる（逆効果もあり）

# 注意点
生体情報や実際のストレス値は取得しません。通知が鬱陶しい場合はスクリプトを停止してください。ログはローカルにのみ保存され、外部送信はありません。

# 設計方針
シンプルな構成とし、OSやユーザー環境に依存しない設計を心がけています。今後はユーザーアクティビティの検知範囲拡張や、通知演出のカスタマイズも検討可能です。
