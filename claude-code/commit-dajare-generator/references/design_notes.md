# 概要
commit-dajare-generatorは、エンジニアのコミット作業を楽しくするため、変更内容やファイル名から理系・開発系ダジャレを自動生成するSkillです。

# 公式ドキュメント抜粋
- git commit: https://git-scm.com/docs/git-commit
- コミットメッセージのガイド: https://chris.beams.io/posts/git-commit/

# 利用例
- チームの雰囲気を和ませたい時
- レビュー時の話題作り
- 日々のコミットに遊び心を加えたい場合

# 注意点
- ダジャレの質や好みは個人差が大きく、全ての現場で歓迎されるとは限りません。
- 除外パスやバイナリファイルには配慮していますが、完全な精度ではありません。

# 設計方針
- ファイル名やdiffからキーワード抽出し、テンプレートと組み合わせてダジャレ生成
- 無理やり感も含めて“ネタ枠”として楽しむ設計
- コミット前にメッセージ選択・編集が可能なインタラクティブUI
- ローカルで完結し、履歴や個人情報は保存しません
