# 機能概要
boss-key-instant-hideは、作業中に見られたくない画面（エディタやターミナル等）を、コマンド一発で即座に非表示にし、天気予報やエクセル風の無難なウィンドウへ切り替えるスキルです。上司や親が近づいてきた際に「今ここで見られたくない！」という状況を回避できます。さらに、偽の進捗バーやダミーデータ表示など、演出も選択可能です。データ損失や実害なく、安心して使える設計です。

# 使い方
このSkillは明示的な呼び出しは不要です。以下のようなキーワードが会話やコマンドに現れた際、Antigravityが自動で発動します。
- 「上司が来た」「親が来た」「画面を隠したい」
- 「ボスキー」「一発で画面を切り替え」
- 「進捗バーを偽装」「天気予報を表示」

# 出力例
```
[INFO] 全エディタウィンドウを最小化しました。
[INFO] Excel風ウィンドウを前面に表示しました。
[INFO] ダミー進捗バー: [███████-----] 70%
[INFO] 5秒後に元の状態に戻せます。
```

# 注意点
- 本Skillはローカル環境のウィンドウ制御API（pygetwindow, pyautogui, tkinter等）を利用します。
- 実際のデータや作業内容は保存されません。必要に応じて事前保存を推奨。
- サポートOSはWindows/Mac/Linuxですが、ウィンドウ制御の挙動はOS依存です。
- 仮想環境やリモートデスクトップでは一部機能が制限される場合があります。
