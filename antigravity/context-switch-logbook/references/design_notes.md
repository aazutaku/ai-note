# 概要
このSkillは、作業中のエージェントやタスクの切り替え履歴を漏れなく記録し、後から振り返るためのコンテキスト管理を支援します。CLI操作と自動発動の両対応で、日々の作業フローを可視化します。

# 公式ドキュメント抜粋
Antigravityのコンテキスト管理APIやログ保存方針に準拠し、ユーザーのローカル環境下で安全に動作します。

# 利用例
- タスクAからBへ移動した理由を後で確認したい
- 複数ブランチを並行作業する際の切り替え履歴を残したい
- チーム内で作業経路の共有やレビューに活用

# 注意点
ログはユーザーのホームディレクトリ配下にローカル保存され、外部同期や自動バックアップはありません。機密情報の記載には注意が必要です。

# 設計方針
シンプルなテキストログ形式で、CLIからの柔軟な記録・参照・要約が可能。今後はAPI連携やグラフィカルな可視化も検討しています。
