# 機能概要
commit-mythical-creature-announcerは、Gitコミット時や明示的な呼び出しで、毎回異なる“架空の神獣”と“理不尽な開発運勢”をターミナルに表示するジョーク系Skillです。開発現場にカオスと笑いをもたらし、日々のコミット作業を少しだけ楽しくします。神話生物のイラスト風アスキーアートと、意味不明な運勢コメントがセットで出現し、飽きずに楽しめます。

# 使い方
- 明示呼び出し: `/commit-mythical-creature-announcer` をターミナルで実行
- 暗黙発動: Gitコミット時、または「コミット」「運勢」「神獣」などのキーワードを含む操作時に自動発動

# 出力例
```
🐉 ドラゴンが眠りについた…今日は進捗が静かに進むでしょう。
─────────────────────────────
          __====-_  _-====__
        _--^^^#####//      \\
       -^##########// (zzz) \\
      _/############//       \\
     /#############((     .   ))
    -###############\\     //
   -#################\\_ //
  _#/|##########/\######( )
 |/ |#/#\#/\#/\#/\#/\#|\|
─────────────────────────────
本日の開発運勢: コードレビューは神話級の難易度。バグはそっとしておこう。
```

# 注意点
- 表示内容は完全ランダムで、実際の運勢や進捗とは一切関係ありません。
- ローカル環境でのみ動作します。履歴や神獣の記録は自動保存されません。
- 除外パス: .git, .claude/skills/配下以外には影響しません。
