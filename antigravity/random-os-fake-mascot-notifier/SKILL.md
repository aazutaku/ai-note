# 機能概要
random-os-fake-mascot-notifierは、作業中のユーザーに対し、架空のOSマスコットがデスクトップ通知で突然現れ、全く役に立たない激励や警告をランダムに発します。通知内容・キャラクター・タイミングは完全にカオスで予測不能。現実逃避や気分転換、作業の合間のちょっとした癒しを提供し、単調な作業環境に遊び心をもたらします。

# 使い方
このSkillはAntigravityのsemantic-match-onlyトリガーで自動発動します。明示的な呼び出しは不要です。『応援』『やる気』『通知』『OS』『マスコット』などのキーワードや、長時間の無操作が検出された場合に自動で通知が表示されます。設定やカスタマイズは不要で、インストール後すぐに機能します。

# 出力例
```
[通知] カーネルくん: システムの裏側から君を応援してるよ！
[通知] ビットちゃん: ビットの海であなたの成功を祈っています。
[通知] メモリ犬: メモリの片隅で見守っています。休憩も大事！
[通知] タスクキャット: タスク管理は計画的に。たまには深呼吸。
[通知] バグバード: バグは友達、怖くないよ！
```

# 注意点
- 通知はデスクトップ領域に表示されますが、環境によっては通知APIの制限で表示されない場合があります。
- 通知頻度は自動調整され、うるさすぎないよう制御されています。
- 通知内容やキャラクターは完全ランダムで、実在のOSやキャラクターとは一切関係ありません。
- ローカルに通知履歴や個人情報は保存しません。
