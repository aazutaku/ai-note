# 概要
prompt-branch-suggesterは、Gitリポジトリの作業ブランチ名やコミットメッセージ、直近の差分ファイルをもとに、Claude Codeの指示ファイル分岐や一時プロンプトセットを自動生成・提案するスキルです。手作業によるプロンプト編集の手間を減らし、分岐の抜け漏れやヒューマンエラーを防ぎます。

# 公式ドキュメント抜粋
- [Claude Code公式](https://github.com/anthropic/claude-code)
- [GitHub Commits API](https://docs.github.com/ja/rest/commits/commits)

# 利用例
feature/add-loginブランチでログイン機能を追加した場合、差分ファイルやコミット内容から「ログイン機能の追加・検証」に特化した指示セットを自動で提案。リファクタやホットフィックス時も同様に適切な分岐指示を生成します。

# 注意点
- 既存の指示ファイルは上書きしません
- 差分取得にはGitコマンドが必要です
- 除外パスは自動で無視されます

# 設計方針
分岐ルールや指示テンプレートは今後の運用やプロジェクトごとに柔軟に拡張可能です。安全性重視のため、生成ファイルは一時的なものとし、手動でのレビューを推奨します。
