# 概要
このSkillは、現実には存在しないOSやシステムの起動画面を毎回ランダム生成し、作業開始時の非日常感や遊び心を演出することを目的としています。実用性はほぼありませんが、日々のルーチンにちょっとした刺激を与えます。

# 公式ドキュメント抜粋
- Python標準ライブラリのみで実装
- argparseによるCLIサブコマンド対応
- 完全ランダム生成で再現性なし

# 利用例
- ターミナルやエディタ起動時の自動演出
- /skills random-os-startup-movie の明示呼び出し
- チームの朝会やハッカソンのネタとして

# 注意点
- 実在のOSや本物の進捗内容は一切表示しません
- ログや履歴は保存されません
- 進捗アニメーションはテキストのみ

# 設計方針
- 100%ランダム生成による毎回異なる体験
- 再現性よりも“未来感”“謎感”を重視
- シンプルな構成で拡張性あり
