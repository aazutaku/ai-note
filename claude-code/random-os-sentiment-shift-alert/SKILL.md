# 機能概要
このSkillは、OSの人格が突如変化したかのようなシュールな通知をデスクトップにランダム表示します。例えば「本日よりツンデレモードです」や「しばらく詩人になります」など、実際には何も変わらないにも関わらず、OSのキャラ設定が急変した旨を本気で宣言します。作業に没頭しているとき、不意打ちで現れる不可解な通知が、日常の単調さを和らげ、思わず笑ってしまう体験を提供します。

# 使い方
- 明示的な呼び出し: `/random-os-sentiment-shift-alert`
- 暗黙発動キーワード例: 「通知」「キャラ変」「人格変化」「OSの気分」「モード切替」などが会話やコードコメントに含まれると自動発動します。

# 出力例
```terminal
[通知] OSキャラ変更のお知らせ
本日よりツンデレモードでご案内します。ご理解とご協力をお願いします。

[通知] システム人格切替
しばらく詩人になります。操作は五七五でお願いします。

[通知] OSより
今からやたら上から目線で案内します。ついてこれますか？

[通知] キャラ設定変更
本日より「やる気ゼロ」モードです。指示は控えめに。
```

# 注意点
- 通知は実際のOS設定や作業内容に一切影響しません。
- 通知内容は毎回ランダム生成されますが、実際のシステム動作は変化しません。
- 通知履歴はローカル保存されません。
- macOS/Linuxのnotify-sendやosascriptなど、標準的な通知APIを利用しています。
