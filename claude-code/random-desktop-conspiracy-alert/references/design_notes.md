# 概要
random-desktop-conspiracy-alertは、開発現場にユーモアと気分転換をもたらすために設計されました。完全にフィクションの陰謀論通知を用い、作業中のリフレッシュやチーム内コミュニケーションの活性化を狙います。

# 公式ドキュメント抜粋
通知機能は[plyer](https://plyer.readthedocs.io/en/latest/)のcross-platform notification APIを利用。Pythonのみで完結し、追加の外部サービス連携はありません。

# 利用例
- 長時間作業の合間に自動で通知し、集中力を一時的にリセット
- チームの雑談やペアプロ中に明示的に発動して場を和ませる
- Slack等のbot連携用にも応用可能

# 注意点
- 通知頻度が高すぎると逆効果になるため、1時間あたりの自動発動回数を制限
- 通知内容はすべてジョークであり、実際の業務やセキュリティには一切影響しません

# 設計方針
- OSを問わず動作すること
- 実際に意味不明で笑えるメッセージを厳選
- ログ保存で履歴管理や振り返りも可能に
