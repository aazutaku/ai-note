# 機能概要
このSkillは、あなたの作業中に突如“架空のOSマスコット”がデスクトップ通知で現れ、全く役に立たない激励や謎の警告をランダムに発します。カーネルくんやメモリ犬、ビットちゃんなど、毎回異なるマスコットが登場し、内容も完全に予測不能。通知タイミングもカオスで、現実逃避や気分転換に最適です。単調な作業や長時間のコーディング時に、思わずクスッと笑える一言が届きます。

# 使い方
- 明示呼び出し例: `/random-os-fake-mascot-notifier` を実行すると即座に通知が表示されます。
- 暗黙発動キーワード例: 「休憩」「疲れた」「集中」「やる気」「通知」「OS」「マスコット」などの文脈や、一定時間作業が続いたタイミングで自動発動します。

# 出力例
```
[通知] カーネルくん: 「今日もバグと戦ってるね！」
[通知] ビットちゃん: 「あなたのRAMが心配です」
[通知] メモリ犬: 「スワップが溢れても僕はそばにいるよ」
[通知] シェル猫: 「コマンド打ちすぎ注意！」
[通知] スレッド鳥: 「マルチタスクはほどほどに」
```

# 注意点
- 通知はデスクトップ領域に直接表示されます。通知の履歴はローカルには保存されません。
- 通知頻度は自動で調整され、うるさすぎないよう制御されています。
- マスコットやメッセージ内容は毎回完全ランダムです。実在のOSやキャラクターとは無関係です。
- Linux/macOS/Windowsの主要デスクトップ環境で動作しますが、通知APIの制約で一部環境では表示されない場合があります。
