# 概要
本Skillは、作業中にユーザーを唐突に祝福することで気分転換やユーモアを提供することを目的としています。通知内容は完全にランダムなネタ系メッセージで、作業効率や生産性への直接的な影響はありませんが、開発現場の空気を和らげる効果が期待できます。

# 公式ドキュメント抜粋
- Linux: notify-send (freedesktop.org)
- macOS: osascript (AppleScript)
- Windows: win10toast (PyPI)

# 利用例
- 長時間のコーディングやビルド作業の合間に、突如祝福されて笑いが生まれる
- チームで使うことで、通知をネタにしたコミュニケーションが活発化

# 注意点
- 通知APIはOSごとに異なり、Linuxではnotify-sendコマンド、macOSではosascript、Windowsではwin10toastを利用します。
- 一部の環境やリモートセッションでは通知が表示されない場合があります。
- 通知頻度は2分～15分のランダム間隔で自動調整され、スパム化を防ぎます。

# 設計方針
- 冗長な通知や過剰な演出を避け、適度な頻度で“忘れた頃に祝福”される体験を重視しました。
- メッセージは今後カスタマイズや追加も容易な設計です。
