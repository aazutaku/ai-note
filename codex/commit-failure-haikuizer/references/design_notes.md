# 概要
commit-failure-haikuizerは、Git操作失敗時のエラー内容を日本語俳句（五・七・五）に変換して表示するジョーク系Skillです。日々の開発現場に遊び心を加え、エラー時の心理的負担を和らげることを目的としています。

# 設計方針
- エラー内容からファイル名やリポジトリ名などの「対象語」を抽出し、ランダムな季語や感情ワードと組み合わせて俳句を生成します。
- 俳句テンプレートを複数用意し、毎回異なるパターンを出力します。
- 明示呼び出し（コマンドライン引数）と、標準入力による暗黙発動の両対応としています。

# 利用例
- git pushやcommit失敗時に、エラー出力をパイプで渡して俳句化
- CI/CDの失敗通知メッセージに組み込むことで、チームの雰囲気を和ませる

# 注意点
- 完全な五・七・五を保証するものではありませんが、雰囲気重視です。
- ログ保存はオプションで、デフォルトではローカルに保存しません。
