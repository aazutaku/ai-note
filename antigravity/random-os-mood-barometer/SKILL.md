# 機能概要
random-os-mood-barometerは、日々のコマンド実行やエディタ操作のたびに“謎のOS風 気分バロメーター”を画面端やメニューバー風に自動表示するスキルです。判定内容は完全ランダムで、「絶好調エンジニアモード」「バグ吸引フェイズ」「アイデア枯渇ゾーン」など、実際の生産性や気分とは無関係な“今のモード”を演出します。作業中の気分や雰囲気にちょっとしたカオスと遊び心を加え、単調な開発作業にアクセントを与えます。

# 使い方
このSkillは明示的な呼び出しは不要です。Antigravityがコマンド実行やエディタ操作など、semantic-match-onlyなトリガーイベントを検知した際、自動的にバロメーターを表示します。たとえば「git commit」「python main.py」などのコマンド実行時や、ファイル保存時に発動します。明示呼び出しはサポートしていません。

# 出力例
```
[OS-Mood Barometer] 2024-06-10 14:32:11
絶好調エンジニアモード：やる気MAX！今こそ難題に挑戦する時。

[OS-Mood Barometer] 2024-06-10 14:35:02
アイデア枯渇ゾーン：コーヒーブレイク推奨。新しい刺激を求めましょう。

[OS-Mood Barometer] 2024-06-10 14:41:17
バグ吸引フェイズ：今日はバグが寄ってきそうな予感…慎重に！
```

# 注意点
- 出力は標準出力または通知バー風のテキスト表示のみです。
- 実際の生産性や気分とは無関係なランダム演出です。
- ローカルに履歴や個人情報は保存しません。
- 邪魔にならないよう、表示タイミングや場所は調整可能です。
