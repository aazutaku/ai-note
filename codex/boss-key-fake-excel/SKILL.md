# 機能概要
boss-key-fake-excelは、作業中の端末やエディタ画面を一瞬で“なんちゃってExcel”風のウィンドウに切り替えるエンタメ系Skillです。急な上司や同僚の来訪時など、見られたくない作業内容を即座に隠し、まるで業務用Excelを操作しているかのように装えます。開発者同士のイタズラや、秘密主義な作業環境の演出にも最適です。元の画面へもワンアクションで戻れるため、日常の“逃げ”をスマートにサポートします。

# 使い方
- 明示呼び出し例: `/skills boss-key-fake-excel` または `@boss-key-fake-excel show`
- 暗黙発動キーワード例: 「やばい」「隠す」「Excel画面に切り替えて」「フェイク画面」
- ショートカット (例): `Ctrl+Shift+B` で即座にフェイク画面表示
- 画面復帰: フェイク画面上の`Esc`キー、またはウィンドウ右上の`×`ボタンで元の作業画面に戻ります

# 出力例
```
[INFO] boss-key-fake-excel: フェイクExcel画面を表示しました。
[INFO] ショートカット: Ctrl+Shift+B で起動、Escで復帰。
[DEBUG] Window ID: 0x04200007
[DEBUG] 元ウィンドウの状態を保存しました。
[INFO] 復帰操作を検知、元の作業画面へ戻ります。
```

# 注意点
- 本Skillはローカル環境でのみ動作します。
- Excel風画面は本物のExcelではなく、あくまで“それっぽい”静的なGUIです。
- マルチディスプレイや特殊なウィンドウマネージャ環境では動作保証外です。
- フェイク画面の内容はカスタマイズ可能ですが、保存や編集機能はありません。
