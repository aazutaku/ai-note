# 概要
terminal-samurai-announcerは、CLIコマンド実行時に侍風のテキストを自動挿入し、作業を演出するPythonスクリプトです。

# 公式ドキュメント抜粋
コマンドラッパーとして動作し、主要なコマンド名ごとに複数の侍ナレーションをランダム出力します。ON/OFF切替やセリフ表示、状態確認などのサブコマンドも備えています。

# 利用例
- `python samurai_announcer.py run ls -l` → 侍ナレーション付きでls実行
- `python samurai_announcer.py on/off` → グローバルで演出の有効/無効を切替
- `python samurai_announcer.py list --cmd git` → git用セリフ一覧

# 注意点
- 除外ディレクトリでは発動しません
- コマンドの標準出力は変更せず、前後に侍コメントを添えます
- 複数コマンドの連続実行にはシェルラッパーやエイリアス活用を推奨

# 設計方針
ユーザーの既存作業フローを壊さず、必要に応じて即座にON/OFF切替可能な設計。カスタムセリフや追加コマンド対応も容易に拡張できます。
