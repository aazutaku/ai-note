# 概要
このSkillは、実際のシステム操作を一切伴わず、ユーザーに仮想的な再起動進捗バーを演出することを目的としています。主に冗談やリフレッシュ、チーム内のアイスブレイク用途を想定しています。

# 公式ドキュメント抜粋
標準ライブラリのみを使用し、外部依存を排除。進捗バーの出力は標準出力に限定し、ファイルやシステム設定の変更は行いません。

# 利用例
- チームの朝会で「今日は全員リセット！」と表示
- 長時間作業後の気分転換に
- オンラインイベントでのジョーク演出

# 注意点
本Skillは演出専用であり、誤解を招くような実システム操作は一切行いません。業務システムや本番環境での誤用に注意してください。

# 設計方針
進捗メッセージは毎回ランダム生成され、ユーザー体験の多様性を重視。CLIサブコマンドで柔軟な利用が可能です。
