# 概要
random-os-startup-movie Skillは、ターミナルやエディタの起動時に毎回異なる“偽OS起動画面”をテキストで演出することを目的としています。ユーザーの日常作業に無駄な未来感や緊張感を加える、完全にエンターテインメント志向のSkillです。

# 公式ドキュメント抜粋
このSkillはPythonの標準ライブラリのみで構成され、外部APIやネットワーク通信は一切行いません。CLIサブコマンドとして `show`（1回表示）、`batch`（複数回表示）、`list`（利用可能なOS名や進捗メッセージの一覧表示）をサポートします。

# 利用例
- ターミナル起動時の自動演出
- チームメンバーへの“未来的なジョーク”として
- 作業前の気分転換やアイスブレイク

# 注意点
現実のOSやシステムには一切影響を与えません。演出内容は完全に架空であり、実用的な機能はありません。ログや履歴も残しません。

# 設計方針
毎回異なる内容を生成するため、OS名・進捗メッセージ・進捗バー・終了メッセージをランダムに組み合わせています。演出の“無駄さ”と“未来感”を重視し、実用性は意図的に排除しています。
