# 機能概要
このSkillは、ターミナルの環境変数（例: MOOD）に応じて、コマンド実行時やプロンプト表示時に「今日の気分」マークを自動表示します。作業開始時に `export MOOD=やる気MAX` のように気分をセットするだけで、以降のターミナル操作時にその内容が画面上に現れます。チームで同じ端末を使う場合やペアプロ時にも、今の気分が一目で分かるため、コミュニケーションのきっかけや雰囲気作りに役立ちます。

# 使い方
1. ターミナルで `export MOOD=元気` など任意の気分を設定します。
2. このSkillを有効化すると、各コマンド実行時に気分マークが自動で表示されます。
3. 気分を変更したい場合は再度 `export MOOD=燃え尽き寸前` のように上書きしてください。

**明示呼び出し例:**
- `export MOOD=迷走中`
- `python terminal_mood_indicator.py`

**暗黙発動キーワード例:**
- "プロンプトに気分を表示"
- "環境変数 MOOD を使って演出"
- "コマンド実行時に状態を演出"

# 出力例
```
============================
今日の気分: 迷走中
============================
$ ls
Documents  Downloads  Music
============================
今日の気分: やる気MAX
============================
$ git status
On branch main
nothing to commit, working tree clean
```

# 注意点
- 気分は環境変数 MOOD で管理されます。未設定の場合はデフォルト表示。
- シェルのプロンプト自体を変更する場合は、.bashrc/.zshrc への追加が必要です。
- このSkillはローカル端末でのみ動作し、リモートセッションや一部の仮想端末では制限があります。
