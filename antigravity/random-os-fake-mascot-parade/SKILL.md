# 機能概要
このSkillは、作業の合間や手が止まった瞬間などに、架空のOSマスコットたちが突如デスクトップ通知で現れ、意味不明な激励や警告、謎の助言を残していく演出を提供します。通知の内容は毎回完全にランダムで、実在しないOS風キャラクター（例: カーネルくん、メモリねずみ、シェルパンダ等）が、ユーザーの作業状況に関係なく唐突に登場します。実用性は一切なく、作業の合間に「なぜこれが…」と自分の正気を疑いたくなるような、遊び心あるネタSkillです。

# 使い方
このSkillは明示的な呼び出しは不要です。Antigravityが「集中が切れた」「コマンド待ち」「長時間無操作」などの状況を検知した際、自動的に発動します。暗黙発動キーワード例: "通知", "OSマスコット", "応援", "警告", "意味不明な助言" など。

# 出力例
```
[通知] カーネルくん: 「プロセス管理は心の整理から！」
[通知] メモリねずみ: 「バグ撲滅運動中。メモリリークにご用心！」
[通知] シェルパンダ: 「一服推奨。コマンド履歴も休ませよう。」
[通知] デバイスタコ: 「USBを抜く前に、心の準備を！」
[通知] スワップペンギン: 「スワップしすぎ注意！たまには休憩を。」
```

# 注意点
- 本Skillは完全なジョーク用途であり、作業効率やOS運用には一切寄与しません。
- 通知内容・キャラ名・セリフは毎回ランダム生成されますが、通知頻度は過剰にならないよう内部で調整されています。
- ローカル保存や履歴機能はありません。通知は一時的なものです。
- システム通知APIを使用するため、Linux/macOS/Windowsの主要デスクトップ環境で動作しますが、サーバやCUI専用環境では通知が表示されません。
