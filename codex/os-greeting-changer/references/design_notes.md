# 概要
os-greeting-changerは、ターミナル起動時や明示呼び出し時にだけ動作し、現実には存在しないOSの起動メッセージをランダム表示するSkillです。実用性よりも遊び心やリラックス効果を重視しています。

# 公式ドキュメント抜粋
本Skillは、ファイルやシステム設定に一切変更を加えず、出力のみを行う安全設計です。履歴はセッション内のみ保持し、永続保存は行いません。

# 利用例
- 毎朝ターミナルを開くたびに異世界OSのメッセージで気分転換
- チームのペアプロやリモート会議で話題作りに
- /skills menu から明示的に呼び出してジョークを楽しむ

# 注意点
本Skillは完全にジョーク用途です。実作業やファイル操作に影響を与えませんが、業務用途での誤解を避けるためにも注意喚起を行っています。

# 設計方針
- 実在しないOS名や演出を中心に、現実では絶対に見られない内容を厳選
- シンプルな構成で、どの環境でも安全に動作するよう配慮
- 公式: https://ai-note.tech/docs/skills/os-greeting-changer
