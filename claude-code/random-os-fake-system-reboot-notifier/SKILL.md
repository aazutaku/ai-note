# 機能概要
このSkillは、作業中のPCに“謎のOS再起動予告”をデスクトップ通知で表示します。通知内容は毎回ランダムに生成され、例えば「重要なお知らせ：5分後にシステムは謎の理由で再起動します」「理由：カーネルの気まぐれ」など、実害のないジョークメッセージが突然現れます。これにより、オフィスやチームの雰囲気を一瞬ピリッとさせつつ、すぐに笑いを誘う演出が可能です。実際の再起動やシステム操作は一切行われません。

# 使い方
- 明示呼び出し例：`/random-os-fake-system-reboot-notifier`
- 暗黙発動キーワード例：「再起動予告」「OSジョーク通知」「システムメンテナンス風通知」などが会話に含まれる場合、自動でSkillが提案・発動されます。

# 出力例
```
[通知] 重要なお知らせ：5分後にシステムは謎の理由で再起動します。
理由：カーネルの気まぐれ

[通知] システムメンテナンス：7分後に自動再起動予定です。
理由：管理者の気まぐれ

[通知] 重大なアップデートのため、3分後に再起動します。
理由：宇宙線の影響
```

# 注意点
- 通知はローカルPCの通知API（Windows: Toast, macOS: terminal-notifier, Linux: notify-send等）を利用します。
- 実際の再起動やファイル操作は一切行いません。
- 通知が表示されない場合は、OSの通知設定や権限を確認してください。
- ネタ通知のため、業務システム等での誤用に注意。
