# 機能概要
このSkillは、コミット時に変更内容やファイル名から理系ギャグ漫画家が降臨したかのごとく、無理やりなダジャレコミットメッセージを自動生成します。真面目な開発現場にユーモアを注入し、チームの雰囲気を一瞬で和ませることができます。コミットメッセージのマンネリ化を打破し、エンジニア同士の会話のネタや、レビュー時のアイスブレイクにも最適です。

# 使い方
- 明示呼び出し: `/commit-dajare-generator` または `/skill commit-dajare-generator`
- 暗黙発動: commit, 変更, ファイル名, メッセージ生成 などのキーワードを含むコミット操作時、またはsemantic commit時に自動発動します。
- コミット前に、生成されたダジャレメッセージが表示され、採用・編集・却下が選択可能です。

# 出力例
```
$ git add src/math_utils.py
$ git commit
[commit-dajare-generator]
生成候補: 「数学的に証明しちゃった、証明書だけにshow me!」
生成候補: 「utilsを修正したら、うっつーるほど美しくなった！」
生成候補: 「バグを消したら、バグっときた！」
選択: 1
コミットメッセージ: 数学的に証明しちゃった、証明書だけにshow me!
```

# 注意点
- 生成されるダジャレはファイル名やdiffの内容に依存しますが、無理やり感が強い場合もあります。
- 一部のパス（.git/やビルド成果物）は除外対象です。
- ローカルでのみ動作し、生成メッセージは保存されません。
- チームの雰囲気や文化によっては不適切な場合もあるため、利用シーンにはご注意ください。
