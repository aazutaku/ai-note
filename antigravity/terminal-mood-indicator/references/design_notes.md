# 概要
このSkillは、ターミナルの環境変数（MOOD）を利用し、作業者の気分をコマンド実行時に表示することで、作業現場の雰囲気やコミュニケーションを活性化することを目的としています。

# 公式ドキュメント抜粋
- Python os.environ: https://docs.python.org/3/library/os.html#os.environ
- subprocess.run: https://docs.python.org/3/library/subprocess.html

# 利用例
- 朝イチで `export MOOD=やる気MAX` をセットし、以降の作業で自分やチームの気分を可視化
- 気分履歴を後で振り返って自己管理や雑談のネタに

# 注意点
- 環境変数の反映はプロセスごとなので、setコマンドは現在のシェルには直接影響しません（export推奨）
- 一部のリモート端末や特殊なシェルでは表示が制限される場合があります

# 設計方針
- 実在APIのみ利用し、依存を最小限に抑えています
- コマンドラッパーとしても、単独で気分表示用にも使える柔軟設計
- 履歴ファイルはホームディレクトリ直下に保存し、ユーザーごとに分離
