# 概要
このSkillは、デスクトップの片隅に完全ランダムな天気アイコンを表示することで、作業環境に遊び心や意外性を加えることを目的としています。天気や気分の実態とは無関係な演出で、会話や作業の合間に小さな話題やリフレッシュ効果をもたらします。

# 公式ドキュメント抜粋
- PySimpleGUI: https://pysimplegui.readthedocs.io/
- Pillow: https://pillow.readthedocs.io/

# 利用例
- チーム開発中、画面右下に突如「大雪」や「雷」アイコンが現れ、雑談やアイスブレイクのきっかけに。
- シリアスな会議中に「晴れ」マークが表示され、場の空気が和む。

# 注意点
- 実際の天気APIや感情認識APIは一切使わず、完全にランダムな演出です。
- デスクトップの描画位置は環境により微調整が必要な場合があります。
- アイコン画像はローカル生成または同梱されますが、Pillow未インストール時は色付き四角で代用します。

# 設計方針
- シンプルかつOS横断的な実装のためPySimpleGUIを採用。
- 起動のたびに毎回異なる天気を表示し、繰り返し利用時も飽きさせない工夫を重視しています。
