# 概要
`os-greeting-changer`は、ターミナルやコンソール起動時に現実には存在しないOS名や起動演出をランダム表示することで、開発現場に遊び心とリラックスをもたらすスキルです。実用性よりもユーモアや気分転換を重視した設計です。

# 公式ドキュメント抜粋
UNIX系のfortune(1)コマンドや、各種OSの起動メッセージ演出を参考にしています。ユーザーの作業やファイルには一切影響を与えません。

# 利用例
- ターミナルを開くたびに異なる“偽OS”メッセージで気分転換
- チーム内で「今日のOSは何だった？」と話題に
- 長時間のコーディングの合間に一息つくきっかけ作り

# 注意点
- 出力は標準出力のみで、ログやファイルには記録されません
- 実在しないOSや演出が含まれるため、誤解を招かないよう業務用途には注意
- 本Skillは完全に無害な表示専用です

# 設計方針
- 実在しないOS/バージョンや理不尽な演出を意図的に混ぜ、現実では絶対に見られない体験を重視
- コマンドライン引数でメッセージ数や一覧表示、要約も可能にし、柔軟な利用を想定
- シンプルな構成で安全性・拡張性を確保
