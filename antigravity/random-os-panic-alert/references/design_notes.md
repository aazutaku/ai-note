# 概要
このSkillは、現実の障害通知と絶対に混同しない“珍妙なOSパニックアラート”をデスクトップ通知で表示することで、作業中のユーザーの集中力を意図的に攪乱し、気分転換やユーモアを提供します。通知内容は毎回ランダム生成され、深刻そうで全く深刻でない点が特徴です。

# 公式ドキュメント抜粋
- Windows: win10toast (https://github.com/jithurjacob/Windows-10-Toast-Notifications)
- macOS: osascript (AppleScript経由)
- Linux: notify-send (libnotify)

# 利用例
- 長時間のコーディングや単調作業時の気分転換
- チーム内でのジョーク通知として
- 仮想環境やリモートワーク中の“謎イベント”演出

# 注意点
- 本Skillは本物の障害通知ではありません。誤認防止のため、現実の通知で使われない表現・内容のみを採用しています。
- 通知間隔は自動調整され、過度なスパムを防止します。
- ログや履歴はローカル保存されません。

# 設計方針
各OSの標準通知APIを活用し、追加ライブラリが必要な場合は明示的に案内します。Skillの目的は“気を抜く瞬間”の提供であり、現実のシステム運用には一切影響を与えません。
