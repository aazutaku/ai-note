# 概要
このSkillは、作業時に毎回異なる“謎のOSアップデート進捗バー”をランダム表示し、ユーザーの緊張を和らげることを目的としたユーモア系演出スキルです。

# 公式ドキュメント抜粋
Pythonの標準ライブラリ（random, time, argparse, sys）を利用し、外部依存を排除しています。

# 利用例
- コマンド実行やエディタ保存時に自動発動
- `/skills menu` から明示的に呼び出し
- ターミナルで `python random_os_fake_update_progrebar.py run -n 3` など

# 注意点
- 進捗内容・進捗率は完全ランダムで、実際のシステム更新とは無関係です
- 100%になっても何も起きません
- 履歴やログ保存は行いません

# 設計方針
- 冗長な通知を避け、演出は数秒で自動消去
- コマンド/エディタ操作時の脱力系演出として設計
- 拡張性を考慮し、進捗内容リストは容易に追加可能
