# 機能概要
terminal-mood-barometerは、ターミナルでコマンドを実行するたびに「気分指数」とユーモラスなコメントを完全ランダムで表示するスキルです。真面目な作業中にも、毎回異なるやる気スコアや小ネタが割り込むことで、作業の雰囲気を和らげたり、ちょっとした遊び心を提供します。コマンドライン作業の単調さを打破し、チームや個人の気分転換や話題作りにも役立ちます。

# 使い方
- 明示呼び出し例: `/terminal-mood-barometer` で現在の気分指数を表示
- 暗黙発動例: `git commit` や `ls` など任意のコマンド実行時に自動で発動（設定による）
- 発動キーワード: "コマンド実行", "やる気", "気分指数", "バロメーター"

# 出力例
```
[Terminal Mood Barometer]
やる気指数: 87
コメント: 今日もコーディング日和！

[Terminal Mood Barometer]
やる気指数: 12
コメント: カフェイン補給推奨。無理せず休憩を。

[Terminal Mood Barometer]
やる気指数: 54
コメント: まあまあの調子。あと一息！
```

# 注意点
- 気分指数とコメントは完全にランダム生成で、実際の作業状況や感情を反映しません
- 履歴やスコアはローカル保存されません
- 出力は標準出力のみで、他のファイルやシステム設定は変更しません
- コマンドの実行結果を妨げることはありませんが、出力が多い場合は見落としに注意
