# 概要
context-switch-logbookは、作業の切り替え履歴を時系列で記録し、後から思考経路やタスクの流れを可視化するためのSkillです。CLIベースで直感的に操作でき、記録・閲覧・要約が可能です。

# 公式ドキュメント抜粋
公式: https://ai-note.tech/skills/context-switch-logbook
「エージェントやブランチ切り替え時に記録を残すことで、作業の振り返りや抜け漏れ防止に役立ちます。」

# 利用例
- ブランチ切り替え時に `--from`/`--to`/`--reason` を指定して記録
- 日次で `list` や `summary` で作業履歴を確認
- 誤記録時は `delete-last` で修正

# 注意点
- ログはローカル保存のみ。クラウド連携や外部API連携はありません。
- 機密情報の入力には注意してください。

# 設計方針
- シンプルなjsonl形式で高速な追記・読み取りを実現
- CLIサブコマンドで用途別に柔軟に対応
- ローカルファイルのみで完結し、環境依存性を最小化
