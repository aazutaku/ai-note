# 概要
本Skillは、作業効率を意図的に妨害するエンタメ系Skillとして設計されています。完全に実務と無関係な禅問答風メッセージを、OSの通知機能を通じて表示します。

# 公式ドキュメント抜粋
PythonからクロスプラットフォームでOS通知を送るには、`plyer.notification`モジュールを利用します。Windows, macOS, Linuxで動作しますが、環境依存の制約も存在します。

# 利用例
- コマンドラインで `python zen_master_alert.py start --interval 300` と実行すると、5分ごとに禅通知が表示されます。
- `python zen_master_alert.py once` で1回だけ通知。
- `/skills fake-os-zen-master-alert` で明示的にSkillを呼び出せます。

# 注意点
- plyerが未インストールの場合は標準出力に出力されます。
- 通知内容は完全に無意味・実務無関係であることを保証します。
- バックグラウンド実行や自動停止は未対応です。

# 設計方針
Skillの目的は「集中を乱す」ことなので、通知内容は深い意味を持たず、禅問答風に徹底しています。通知頻度やON/OFFはユーザ制御可能とし、履歴保存は行いません。
