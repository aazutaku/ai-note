# 概要
このSkillは、従来の“ボスキー”とは異なり、画面を業務用に偽装するのではなく、完全に意味不明なダミー画面をランダム生成して表示することを主眼としています。主にジョークやカオスな演出用途を想定しています。

# 公式ドキュメント抜粋
ウィンドウ制御にはPythonのtkinter（標準GUIライブラリ）を利用し、全画面表示やESCキーによる復帰を実装しています。乱数生成や文字列生成にはrandom, stringモジュールを使用。

# 利用例
- 上司の接近を検知した際に自動でダミー画面を表示
- コマンドラインから `python random_boss_key.py show --timeout 5` で手動発動
- 内容確認用に `test` サブコマンドで端末出力

# 注意点
リモートデスクトップやCLI専用端末では全画面表示が制限される場合があります。Skill発動中の作業状態は保存されませんのでご注意ください。

# 設計方針
「怪しまれるほど意味不明」な画面生成を重視し、毎回異なる内容を高速に描画します。ウィンドウ復帰の確実性とユーザー体験のバランスに配慮しています。
