# 概要
commit-fortune-tellerは、Gitコミットやpushのタイミングでターミナルにランダムな“運勢”を表示するジョークSkillです。エンジニアの作業に笑いや話題を提供し、重苦しい開発現場の空気を和らげることを目的としています。

# 公式ドキュメント抜粋
- Python randomモジュール: https://docs.python.org/ja/3/library/random.html
- argparse: https://docs.python.org/ja/3/library/argparse.html

# 利用例
- 毎回のコミット時に自動発動し、その日の運勢やタロットカードが表示される
- `/skills menu`や明示的な呼び出しで任意のタイミングでも実行可能
- ログ機能で過去の運勢を振り返ることもできる

# 注意点
- 実際のGitリポジトリや履歴には一切影響しません
- 出力内容は完全にランダムで、現実の運勢や成果とは無関係です
- ログファイルはローカルにのみ保存され、外部送信等はありません

# 設計方針
- 軽量かつ邪魔になりすぎない実装
- 占い文面やカードの意味に遊び心を盛り込む
- 実用性よりも“混乱と笑い”を重視
