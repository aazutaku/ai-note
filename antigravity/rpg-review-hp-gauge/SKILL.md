# 機能概要
このSkillは、コードレビュー時の指摘数をRPGのHPゲージ風に可視化します。PRごとに「あなたのコードの体力」をHPバーとして表示し、指摘が多いほどHPが減少。HPが0になると「力尽きた」演出メッセージが現れます。真面目なレビュー現場に遊び心を加え、緊張感と笑いを演出します。レビュー負荷の可視化やチームの雰囲気改善にも役立ちます。

# 使い方
明示的な呼び出しは不要です。Antigravityが「HPゲージ」「体力バー」「RPG風レビュー」「指摘数可視化」などのキーワードを含むレビュー状況やPRコメントを検出した際、自動的に発動します。レビュー指摘が多いPRや、レビュー進捗を可視化したい場面で活躍します。

# 出力例
```
[HPゲージ] あなたのコードの体力: [██████░░░░] 60/100
指摘数: 8 (最大HP: 100)

[HPゲージ] あなたのコードの体力: [░░░░░░░░░░] 0/100
指摘数: 20 (最大HP: 100)
力尽きた... 次回の冒険に期待しよう！
```

# 注意点
本Skillはレビュー指摘数の自動集計に依存します。除外パスや特定のファイルは設定により無視できますが、誤検出を防ぐためレビューコメントの形式に注意してください。HPゲージの閾値や最大HPはカスタマイズ可能です。ローカルに履歴を保存する場合、.agent/skills/rpg-review-hp-gauge/data/ 配下を利用します。
