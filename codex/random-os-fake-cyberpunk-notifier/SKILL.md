# 機能概要
このSkillは、あなたのデスクトップに突如として“謎のサイバーパンクOS通知”を出現させる演出系ツールです。通知内容は「ニューロン接続不良を検出」「シンセ脳波警告」「クローム義手アップデート推奨」など、実在しない未来SF世界のOSが本気で心配するようなものばかり。現実の作業に疲れたときや、気分転換、配信・実況の演出にも使えます。実用性はゼロですが、デスクトップが一瞬で近未来の電脳都市に変貌します。

# 使い方
- 明示呼び出し: `/skills menu` から本Skillを選択、または `$random-os-fake-cyberpunk-notifier` を直接実行。
- 暗黙発動: 「サイバーパンク」「SF通知」「未来っぽい演出」などのキーワードを含む会話やコマンドで自動発動。
- 通知頻度やタイミングは自動調整され、迷惑にならないよう配慮されています。

# 出力例
```
[CYBER-OS] ニューロン接続不良を検出: シナプス再構成を推奨します
[CYBER-OS] シンセ脳波警告: 脳波パターンが異常です
[CYBER-OS] クローム義手アップデート推奨: セキュリティパッチ未適用
[CYBER-OS] メモリーカプセル過熱: 冷却システムを確認してください
[CYBER-OS] ナノマシン自己診断エラー: 再起動を推奨
```

# 注意点
- 実在しない通知のみを表示し、実際のシステム状態には影響しません。
- 通知はローカル環境の標準通知API（Windows: Toast, macOS: osascript, Linux: notify-send）を利用します。
- 通知履歴は保存されません。
- 音やアニメーションは含みません。
