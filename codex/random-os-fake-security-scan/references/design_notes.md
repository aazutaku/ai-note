# 概要
このSkillは、作業中のユーザーに対し、フェイクのOSセキュリティスキャン通知をデスクトップに表示することで、職場やリモートワークの雰囲気を和ませることを目的としています。

# 公式ドキュメント抜粋
通知表示にはPythonの`plyer.notification` APIを利用。plyerはクロスプラットフォームでデスクトップ通知を提供する公式ライブラリです。

# 利用例
- `/random-os-fake-security-scan scan` で即座に通知を表示
- 進捗バーや警告内容は毎回ランダム生成されるため、何度でも新鮮な演出が可能

# 注意点
- plyerはWindows/Mac/Linuxで動作しますが、サーバーやSSH環境では通知が表示されません
- 本Skillは完全なジョーク用途であり、実際のセキュリティ診断は行いません

# 設計方針
- 実在のAPIのみ使用し、CLIサブコマンド設計（scan/list/summary）で拡張性を確保
- ログや履歴を残さず、軽量・安全な実装
