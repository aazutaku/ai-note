# 機能概要
このSkillは、開発現場や作業中のPCに突如として“謎のOS緊急アラート”風のデスクトップ通知を表示します。通知内容は「危険：バグが自我を獲得しました」「注意：メモリがピクニック中」など、業務と全く関係ないカオスなメッセージばかり。真面目な空気を一瞬で和ませたり、チームの息抜きやアイスブレイクに最適です。通知は実害ゼロで、システムやファイルには一切影響しません。

# 使い方
- 明示呼び出し例：
  `/desktop-chaos-alert` または `/skill desktop-chaos-alert`
- 暗黙発動キーワード例：
  「カオスな通知」「意味不明な警告を出して」「息抜きしたい」「変なアラート」「混沌を演出」などを含む会話やリクエスト

# 出力例
```
[通知] 危険：バグが自我を獲得しました
[通知] 注意：メモリがピクニック中
[通知] 深刻：キーボードが人生に迷っています
[通知] 警告：OSが哲学的思索に入りました
[通知] 重大：CPUが詩を書き始めました
```

# 注意点
- 通知はローカルPCのデスクトップ通知機能を利用します。Linux/macOS/Windowsで動作しますが、通知許可が必要です。
- 実際のファイルやシステムには一切影響を与えません。
- 通知頻度や内容はスクリプト内で調整可能です。
- 不要な場合はスクリプトの停止やアンインストールで簡単に無効化できます。
