# 機能概要
このSkillは、あなたの実際の気分や天候とは一切関係なく、完全ランダムな“気分天気”アイコンをデスクトップの片隅に表示します。晴れや曇り、雷や大雪など多彩な天気アイコンが、作業中や会議中に突如現れることで、画面に理不尽なカオスとユーモアをもたらします。実際の天気APIや気分入力とは無縁なので、どんなシーンでも予測不能な演出を楽しめます。

# 使い方
- 明示呼び出し: `/desktop-mood-weather` を実行すると即時にランダム天気アイコンが表示されます。
- 暗黙発動: 「気分転換」「天気演出」「デスクトップ通知」などのキーワードが会話やコマンドに含まれると自動発動します。

# 出力例
```
[INFO] desktop-mood-weather: "大雪" アイコンを表示しました
[INFO] desktop-mood-weather: "雷" アイコンを表示しました
[INFO] desktop-mood-weather: "晴れ" アイコンを表示しました
[INFO] desktop-mood-weather: "曇り" アイコンを表示しました
```
（実際はデスクトップ右下に天気アイコンがポップアップ表示されます）

# 注意点
- 表示される天気は完全な乱数で決まり、実際の天候やユーザーの気分とは一切連動しません。
- アイコン画像はローカルの一時ファイルに保存されます。不要になった場合は手動で削除してください。
- Windows/macOS/Linuxの主要デスクトップ環境で動作確認済みですが、Wayland等一部環境では通知が表示されない場合があります。
