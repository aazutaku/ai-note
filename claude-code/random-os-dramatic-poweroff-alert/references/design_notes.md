# 概要
このSkillは、作業中のユーザーに対し“壮大な嘘”の電源オフ通知をランダムに表示し、緊張感を和らげる遊び心を提供します。実際のシャットダウンは一切行われません。

# 公式ドキュメント抜粋
通知表示にはPythonのplyerライブラリを利用しています。plyerはクロスプラットフォーム対応で、Windows/Mac/Linuxで同一コードが動作します。

# 利用例
- 明示的に`/random-os-dramatic-poweroff-alert`で呼び出す
- 長時間作業や「シャットダウン」などの単語が会話に現れた際、自動発動

# 注意点
- plyerが内部で利用する通知APIはOS依存です。Linuxではlibnotifyが必要な場合があります。
- 通知内容は履歴としてメモリ上にのみ保持され、ファイル等には保存されません。

# 設計方針
- 電源操作は絶対に実行しない設計
- 通知内容は毎回ランダム生成し、強烈な演出を重視
- OS依存を極力排除し、どの環境でも動作することを優先
- コマンドラインから明示/自動発動の両対応
