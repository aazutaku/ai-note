# 機能概要
このSkillは、PC作業中に突然“OSのパワーセーバー警告”を装ったデスクトップ通知をランダムに表示します。通知内容は「CPU温存のため10分間タイピング禁止」「マウス移動過多：省エネモード発動」など、根拠も効力もない省エネ指示が毎回異なる形で現れます。集中を問答無用で中断し、周囲に謎の笑いと混乱を提供。業務や勉強の合間に気分転換やネタとして使いたい方に最適です。

# 使い方
- 明示呼び出し例:
  `/random-os-fake-power-saver-alert` (またはSkillパネルから直接実行)
- 暗黙発動キーワード例:
  「省エネ」「パワーセーブ」「警告」「休憩」「通知」「集中力低下」などが会話やコード内に含まれると自動発動します。

# 出力例
```
[通知] 電力節約のため、思考も控えめにしてください。
[通知] CPU温存モード: 10分間タイピング禁止。
[通知] マウス移動過多：省エネモード発動。
[通知] 画面注視が長すぎます。目を閉じて省エネしましょう。
[通知] システム冷却中。深呼吸して待機してください。
[通知] 電力不足: しばらく作業を控えてください。
```

# 注意点
- 実際のOSやシステム設定・電源管理には一切影響しません。
- 通知はローカル環境の通知API（Windows/macOS/Linux）を利用します。
- 通知内容は完全なジョークであり、実害はありません。
- スクリプトはPython3.7以降と、`plyer`パッケージが必要です。
