# 機能概要
このSkillは、あなたが作業やコーディングに集中している最中、突如として“OSによる壮大な電源オフ予告”をデスクトップ通知として表示します。通知内容は毎回ランダムで、「システムはあと3分で壮絶にシャットダウンします（理由：宇宙エネルギー調整のため）」や「全プロセスよ、さらば…」など、完全に意味不明かつ無責任なメッセージばかり。実際にはシャットダウン等の危険な操作は一切行われず、現実には何も起きません。仕事や勉強の緊張感を一撃で吹き飛ばす、遊び心満載の演出を提供します。

# 使い方
このSkillは明示的な呼び出しは不要で、Antigravityが「シャットダウン」「電源オフ」「終了予告」などのキーワードを含む状況や、長時間作業が続いている場合などに自動で発動します。手動でテストしたい場合は、スクリプトを直接実行することも可能です。

# 出力例
```terminal
[通知] システムはあと3分で壮絶にシャットダウンします（理由：宇宙エネルギー調整のため）
[通知] 全プロセスよ、さらば…
[通知] OSカーネル、最後の咆哮をあげよ！
[通知] 電源オフまで残り42秒。運命に抗うな。
[通知] 本日はこれにて全ての稼働を終了します（理由：量子トンネルのメンテナンス）
```

# 注意点
- 実際のシャットダウンや電源オフは一切行われません。
- 通知は各OSのデスクトップ通知APIを利用して表示されます。
- ログや履歴はローカルには保存されません。
- 通知内容は毎回ランダム生成され、繰り返し同じ内容が表示されることはほぼありません。
