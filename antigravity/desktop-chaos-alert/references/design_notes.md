# 概要
このSkillは、開発現場の緊張緩和やアイスブレイクを目的に、実害ゼロのカオスな通知をデスクトップに表示します。通知内容は完全に無意味で、業務やシステムには一切影響しません。

# 公式ドキュメント抜粋
通知には plyer ライブラリを使用しています。plyerはクロスプラットフォームな通知APIを提供し、macOS/Windows/Linuxで動作します。

# 利用例
- チームの雰囲気が張り詰めているときに自動発動
- コードレビューやデバッグ中にジョークとして
- オンライン会議の合間のリフレッシュに

# 注意点
- 通知はローカルPCにのみ表示され、リモートセッションやサーバー環境では機能しません。
- 通知履歴はchaos_alert.logに保存されますが、個人情報や業務情報は記録されません。

# 設計方針
安全性と無害性を重視し、通知内容は完全にランダム生成・固定リストから選択。ユーザーがいつでも停止・履歴確認できるCLIサブコマンド設計としています。
