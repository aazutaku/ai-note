# 機能概要
このSkillは、日々のコーディングやターミナル作業に突如として現れる“未来予言型OSエラーメッセージ”をランダムに表示します。実際のエラーや警告ではなく、根拠のない未来の出来事を仄めかすことで、作業の単調さを打破し、思わず笑ってしまうような体験を提供します。例えば「エラー: 明日あなたは3回pushに失敗します」や「警告: 来週GitHubが逆行します」など、実用性は皆無ですが、作業中の気分転換やコミュニケーションのきっかけとして活用できます。

# 使い方
このSkillは明示的な呼び出しは不要です。ターミナルでコマンドを実行したり、ファイル保存・編集・ビルド・push・commitなどの操作を行うたびに、自動的に予言メッセージがランダムで表示されます。暗黙発動キーワード例: error, warning, terminal, push, commit, review。

# 出力例
```
$ git push origin main
エラー: 明日あなたは3回pushに失敗します

$ code main.py
警告: 来週GitHubが逆行します

$ pytest
注意: コードレビューで哲学的質問を受けます

$ make build
ヒント: あなたの未来のバグは既にここにあります

$ ls
警告: 2週間後にlsの出力が逆順になります
```

# 注意点
本Skillは実際のエラー通知とは異なり、システムやコードの状態には一切影響を与えません。出力はすべて架空の予言であり、実務上の判断材料にはなりません。通知頻度は高すぎないよう調整されていますが、作業フローを著しく妨げる場合はSkillの一時停止を推奨します。ローカルファイルや履歴には保存されません。
