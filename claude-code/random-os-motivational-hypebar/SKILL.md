# 機能概要
このSkillは、ターミナルやエディタでコマンドを実行するたびに、画面端やメニューバーに“OS自己肯定感ハイプバー”を出現させます。バーには「やればできる度」「バグ耐性ゲージ」「今日のやる気残量」など、毎回ランダムなパラメータ名と、まったく根拠のない数値が表示されます。進捗や成果とは無関係に、ユーザーの自己肯定感だけを謎に盛り上げるカオスな演出を提供します。

# 使い方
- 明示呼び出し: `/random-os-motivational-hypebar`
- 暗黙発動: ターミナルでコマンド実行時、またはエディタでファイル保存・ビルド・実行などのアクション時に自動発動します。
- 設定不要、Skill有効化のみで毎回自動表示されます。

# 出力例
```
[OS自己肯定感ハイプバー]
やればできる度: 97%
バグ耐性ゲージ: 12%
今日のやる気残量: 88%
カオス指数: 42%
進捗予測不能度: 76%
```

# 注意点
- 表示されるパラメータ・数値は完全にランダムで、実用性や正確性は一切ありません。
- 本Skillは通知・演出目的のみであり、進捗管理やヘルスチェック等には利用できません。
- ローカルに履歴やデータは保存されません。
- 除外パス: システムディレクトリや自動スクリプト実行時には発動しません。
