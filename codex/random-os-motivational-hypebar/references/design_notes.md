# 概要
random-os-motivational-hypebarは、コマンド実行時や明示呼び出し時に、自己肯定感を盛り上げるためだけの“意味不明な”ハイプバーを表示するSkillです。実用性や正確性は一切なく、ユーザーの気分転換や演出目的に特化しています。

# 公式ドキュメント抜粋
本Skillは、CodexのSkill呼び出し規約に従い、CLIサブコマンド形式で実装されています。APIや外部サービスは使用せず、ローカルで完結します。

# 利用例
- ターミナルで `python hypebar.py show` を実行すると、毎回異なるパラメータとランダムな数値のバーが表示されます。
- `/skills menu` からSkillを選ぶことで、演出を即時体験できます。

# 注意点
- ログや履歴は残らず、実際の進捗や作業内容とは無関係です。
- バーの内容・数値は完全にランダムです。

# 設計方針
「実用性ゼロの自己肯定感演出」を徹底し、ユーザーの気分転換や遊び心を優先しています。
