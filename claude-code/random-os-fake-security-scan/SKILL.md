# 機能概要
このSkillは、作業中のターミナルやエディタ環境に、まるで本物のOSセキュリティスキャンが始まったかのような“フェイク通知”をデスクトップ領域にポップアップ表示します。通知内容は「未知のやる気ウイルスを検出」「あなたの進捗は安全です」など完全なジョークで構成され、進捗バーや警告メッセージも毎回ランダム。職場やリモートワーク中の空気を一瞬で和ませたり、仲間を困惑させるカオスな演出に最適です。

# 使い方
明示的には `/random-os-fake-security-scan` コマンドで即時発動。暗黙的には「進捗」「ウイルス」「セキュリティ」「OS」などの単語を含む会話やコードレビュー時に自動発動します。通知はOSのネイティブ通知APIを利用し、数秒ごとに進捗率が増加する演出付きです。

# 出力例
```
[OS Security Scan] 進捗: 24% - 未知のやる気ウイルスを検出しました
[OS Security Scan] 進捗: 47% - マウスの動きが怪しい挙動を示しています
[OS Security Scan] 進捗: 68% - あなたの進捗は安全です
[OS Security Scan] 進捗: 89% - システムにカオス成分を注入中
[OS Security Scan] 進捗: 100% - 全てのファイルがジョークで保護されました
```

# 注意点
本Skillは実際のセキュリティスキャンやファイル操作は一切行いません。通知はローカル環境でのみ表示され、履歴やログは保存されません。通知API未対応の環境では動作しませんのでご注意ください。
