# 概要
このSkillは、Gitコミットやpushなどのイベントをきっかけに、完全にランダムな占い結果をターミナルに表示するジョーク系スキルです。実用性よりも、エンジニアの作業中に笑いや緊張緩和をもたらすことを目的としています。

# 公式ドキュメント抜粋
Git公式: https://git-scm.com/doc
タロットカードの意味: https://www.angelfire.com/space/tarot/

# 利用例
- デプロイ前の緊張を和らげたい時
- チームでのペアプロやモブプロのアイスブレイク
- 毎日のコミットにちょっとした遊び心を加えたい時

# 注意点
- 実際のコミット内容や運勢とは無関係です。
- ローカル環境でのみ動作し、履歴やファイルには何も保存しません。
- Skill自身や.gitディレクトリには干渉しません。

# 設計方針
- 軽量な動作でコミットフックの邪魔にならないこと
- メッセージ内容は毎回完全ランダム、または--logで日付固定
- タロットカードや運勢の文面は遊び心を重視
- サブコマンドでカードや運勢一覧も確認可能
