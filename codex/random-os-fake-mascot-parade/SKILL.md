# 機能概要
このSkillは、作業中に突如として“謎のOSマスコット”たちがデスクトップ通知で現れるという、全く役に立たないネタ演出を提供します。マスコット名・セリフともに毎回完全ランダムで生成され、カーネルくんやシェルパンダ、メモリねずみなど、架空のOSキャラが意味不明な激励や警告を残して去っていきます。作業の合間に「なぜこれが…」と正気を疑いたくなる体験を演出します。

# 使い方
- 明示呼び出し例: `/skills menu` から本Skillを選択、または `random-os-fake-mascot-parade` をメンション。
- 暗黙発動例: `build`, `compile`, `debug`, `error`, `idle`, `focus` などのイベントやキーワードを含む作業時に自動発動。

# 出力例
```
[通知] カーネルくん: 「ファイルシステムの夢を見てるよ！」
[通知] メモリねずみ: 「バグ撲滅運動中。君も参加しよう！」
[通知] シェルパンダ: 「一服推奨。コマンド履歴は忘れないよ。」
[通知] スワップペンギン: 「メモリが足りない？いや、気合で乗り切ろう！」
[通知] バイナリフクロウ: 「夜更かし注意。バイト列が乱れてるよ。」
```

# 注意点
- 通知内容は完全ランダムで、実用性や生産性は一切ありません。
- 通知頻度は過剰にならないよう内部で制御していますが、長時間の作業では複数回出現します。
- 通知内容やキャラ名は毎回異なり、ローカルに保存されません。
- 一部OSや通知環境によっては表示されない場合があります。
