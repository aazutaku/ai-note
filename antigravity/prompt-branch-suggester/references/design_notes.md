# 概要
prompt-branch-suggesterは、開発現場での作業ブランチやコミット内容の変化をもとに、AIエージェントがより適切なプロンプト指示を自動生成・提案することを目的としています。手動での指示ファイル編集の手間や漏れを減らし、状況に応じた柔軟な分岐指示を実現します。

# 公式ドキュメント抜粋
- Gitブランチやコミット履歴の取得には、gitコマンド（rev-parse, log, diff）を利用しています。
- 出力ファイルは既存の指示ファイルを上書きせず、SUGGESTED_*.mdとして安全に保存されます。

# 利用例
feature/add-loginブランチでlogin.pyを新規追加した場合、自動で「新機能追加に伴う認証ロジックのレビューを優先してください」といった指示案が生成されます。

# 注意点
- 大規模なdiffや複雑なブランチ構成では提案精度が低下する場合があります。
- 本Skillは提案のみを行い、最終的な適用は編集者の判断となります。

# 設計方針
安全性・拡張性を重視し、既存ファイルの上書きを防止。今後は分岐パターンの追加や、AIによる自動要約・高度な指示生成にも対応予定です。
