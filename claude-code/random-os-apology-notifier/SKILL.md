# 機能概要
このSkillは、デスクトップの通知領域に“OSが突然謝罪する”というフェイク通知をランダムに表示します。実際のエラーや業務進行には一切影響せず、作業中に不意打ちで現れることで、職場やリモート作業の雰囲気を和らげたり、話題のきっかけ作りに役立ちます。通知内容は毎回ランダムで、無責任な謝罪や意味不明なコメントが表示されます。

# 使い方
- 明示的な呼び出し例:
  `/random-os-apology-notifier`
- 暗黙発動キーワード例:
  「通知」「謝罪」「エラー」「無責任」「OSメッセージ」などが含まれる文脈で自動発動します。

# 出力例
```
[通知] 本日はご迷惑をおかけしております。
[通知] 謎の遅延が発生しましたが、原因不明です。
[通知] 大変申し訳ありませんが、さっきのコマンドは無かったことにしてください。
[通知] 予期せぬ問題が発生しましたが、詳細は不明です。
[通知] ご不便をおかけし、心よりお詫び申し上げます。
[通知] この通知は自動生成されました。ご安心ください。
```

# 注意点
- 本Skillは完全なジョーク用途です。実際のシステムエラーや業務進行には一切関与しません。
- 通知内容はローカルに保存されません。
- 通知はOSの標準通知APIを利用しますが、利用環境によっては通知が表示されない場合があります。
- 業務上の重要なアラートと混同しないようご注意ください。
