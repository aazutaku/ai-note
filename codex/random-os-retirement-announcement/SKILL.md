# 機能概要
このSkillは、作業の合間やコマンド呼び出し時に、架空のOS内キャラクター（例：カーネル部長、ファイルシステムさん、メモリ課長など）が突如として退職や転職を宣言するデスクトップ通知をランダムに発生させます。意味のない別れのアナウンスが、日常の作業に一瞬のカオスとセンチメンタルな気分をもたらします。通知内容は毎回ランダム生成され、同じものはほぼ出ません。実用性はありませんが、OSやシステムに親しみや遊び心を感じたいユーザーに最適です。

# 使い方
- 明示呼び出し: `/skills menu` でスキル一覧から選択、または `random-os-retirement-announcement` を直接指定。
- 暗黙発動: 「退職」「引退」「通知」「お知らせ」などのワードを含む文脈や、作業中に一定時間が経過したタイミングで自動発火。

# 出力例
```
[通知] 重要：本日をもってカーネル部長が勇退します。
[通知] 急報：メモリ課長が早期リタイアを決意しました。
[通知] お知らせ：ファイルシステムさんが第二の人生へ旅立ちます。
[通知] 緊急：プロセス係長が新天地に転職します。
[通知] ご報告：デバイス管理人が定年を迎えました。
```

# 注意点
- 本Skillは実用的な通知ではなく、完全なジョーク機能です。
- 通知は1時間に1回以上は発火しません（スパム防止）。
- ローカル環境でのみ動作し、通知履歴は保存されません。
- 一部OSでは通知機能に追加パッケージが必要な場合があります。
