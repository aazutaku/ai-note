# 概要
pr-title-omikujiは、開発現場に遊び心を持ち込むためのジョーク系Skillです。PRタイトルに“おみくじ”運勢とコメントを自動付与し、作業の雰囲気を和らげます。

# 公式ドキュメント抜粋
GitHub REST API v3のpullsエンドポイント（GET/PATCH）を利用し、タイトル取得・更新を安全に行います。

# 利用例
- PR作成時に自動でタイトルが「[大吉] 運命のリファクタリング: ...」などに変換される
- /skills menuや明示コマンドで任意タイミングで発動
- dry-runでタイトル変更前の確認も可能

# 注意点
- タイトルの運勢は完全ランダムで内容とは無関係です
- 機密情報や重要なPRには利用を控えることを推奨します
- トークン管理・APIレート制限に注意

# 設計方針
- ユーザー体験を損なわないよう、元タイトルは必ず末尾に残す
- 運勢ワードは10種以上を用意し、バリエーションを確保
- APIエラーやトークン未設定時は明確なエラーメッセージを返す
