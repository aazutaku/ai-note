# 概要
random-os-error-poetryは、OSエラーを詩的に変換することで、開発現場にユーモアと癒しをもたらすことを目的としています。実際のエラーメッセージを損なわず、詩的表現を添えることで、作業効率を維持しつつ気分転換が可能です。

# 公式ドキュメント抜粋
- Python 標準例外: https://docs.python.org/ja/3/library/exceptions.html
- os, socket, argparse 標準APIを利用

# 利用例
- チーム開発時のペアプロやレビューで、エラー発生時に場の雰囲気を和らげる
- 長時間作業や深夜コーディングでの気分転換
- 教育現場でエラー内容の理解促進

# 注意点
- 主要な英語エラーのみ対応。日本語や特殊なエラーは詩化されません。
- クリティカルなシステムエラーやセキュリティ関連例外は詩化対象外です。
- ログや履歴は保存されません。

# 設計方針
- エラー発生時に原文も必ず併記し、通常のデバッグを妨げない
- poetryテンプレートは拡張可能な辞書型で管理
- CLIサブコマンドで実環境テストやカスタムメッセージ詩化も可能
