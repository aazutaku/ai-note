# 機能概要
このSkillは、作業中に突如として“謎のOS退職アナウンス”をデスクトップ通知で発生させる演出系ツールです。通知内容は完全ランダム生成され、「重要：本日をもってカーネル部長が勇退します」「通知：ファイルシステムさんが第二の人生へ」など、実在しないOS内キャラクターの退職や転職が宣言されます。これにより、単調な作業や長時間のコーディングに一瞬のカオスとセンチメンタルな気持ちをもたらします。

# 使い方
このSkillは明示的な呼び出しは不要です。Antigravityが自動的に、"退職" "転職" "勇退" "別れ" などのキーワードや、1時間以上の連続作業を検知した際に発動します。発動頻度は1時間に1回程度までに制限されており、通知はOSのネイティブ通知システム経由で表示されます。

# 出力例
```
[通知] 重要：本日をもってカーネル部長が勇退します。
[通知] 急報：メモリ課長が早期リタイアを決意。
[通知] お知らせ：ネットワーク係長が新天地へ旅立ちます。
[通知] 通知：ファイルシステムさんが第二の人生へ。
[通知] 本日付でプロセス主任が退職されました。
```

# 注意点
- 通知内容は完全にランダム生成され、実在のOSや人物とは一切関係ありません。
- ローカルファイルへの保存や履歴管理は行いません。
- 通知頻度は1時間に1回までに制限されています。
- Linux/macOS/Windowsの主要デスクトップ通知APIに対応しますが、環境によっては通知が表示されない場合があります。
