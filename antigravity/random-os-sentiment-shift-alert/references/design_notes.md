# 概要
本Skillは、OSの人格が突如変わったかのような通知をランダムに表示し、ユーザーに予期せぬユーモアと休憩のきっかけを提供します。実際のOS挙動や作業内容には一切干渉せず、通知のみを純粋に演出目的で出します。

# 公式ドキュメント抜粋
- Linux: `notify-send` コマンド (libnotify)
- macOS: `osascript` (AppleScript経由)
- Windows: `win10toast` Pythonパッケージ

# 利用例
- 長時間作業や集中モード時のリフレッシュ演出
- チーム内での雑談のきっかけ
- 開発現場や勉強会でのデモンストレーション

# 注意点
- 通知APIが動作しない場合、依存パッケージや権限設定を確認してください。
- メッセージ内容はローカル保存されません。
- 実際のOS設定やファイルには影響しません。

# 設計方針
- シュールかつ多様な通知内容を用意し、繰り返し利用しても飽きない工夫を重視
- OS依存部分は自動的に判別し、最適な通知APIを選択
- コマンドラインからも即時通知・一覧・監視モードが利用可能
