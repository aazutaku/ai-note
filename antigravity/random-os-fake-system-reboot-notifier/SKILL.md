# 機能概要
このSkillは、OSのシステム再起動を装ったデスクトップ通知をランダムなタイミング・内容で表示します。実際の再起動やシステム操作は一切行われません。通知文は毎回ネタ感あるランダム生成で、「5分後に再起動します」「理由：カーネルの気まぐれ」など、緊張感と脱力感を同時に演出。オフィスやチームでの話題作り、集中力のリセット、ちょっとしたジョーク用途に最適です。

# 使い方
このSkillは通常、Antigravityから「再起動」「システム障害」「強制終了」「緊急通知」などのキーワードが会話や作業ログに現れた際、もしくはリフレッシュや気分転換を促す文脈で自動発動します。明示的なCLI呼び出しは不要ですが、テストや手動実行時は `python fake_reboot_notifier.py` で即座に通知を表示できます。

# 出力例
```
[通知] 重要なお知らせ：5分後にシステムは謎の理由で再起動します
理由：カーネルの気まぐれ

[通知] システム再起動まで残り2分です
理由：OSの気分転換

[通知] システムはまもなく再起動されます
理由：メモリがやる気を失いました
```

# 注意点
- 実際の再起動やファイル操作は一切行いません
- 通知はローカル環境の通知APIを利用（Windows/Mac/Linux対応）
- サーバーや通知非対応端末では動作しません
- ログや履歴はローカルに一時保存されますが、個人情報は扱いません
