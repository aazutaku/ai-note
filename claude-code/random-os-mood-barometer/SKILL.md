# 機能概要
random-os-mood-barometerは、コマンド実行や作業開始時に、ランダムな“OS風気分バロメーター”をターミナル端やメニューバー風に表示するスキルです。判定内容は「絶好調エンジニアモード」「バグ吸引フェイズ」など完全にランダムで、作業中の気分やテンションを根拠なく演出します。これにより、単調なコマンドライン作業やエディタ作業にちょっとした遊び心とカオスな雰囲気を加え、日々の開発体験を彩ります。

# 使い方
- 明示呼び出し: `/random-os-mood-barometer` をターミナルやエディタで実行してください。
- 暗黙発動: 任意のコマンド（例: `git commit`, `python main.py`, `ls` など）の実行直後に自動でバロメーターが表示されます。
- 設定不要で、導入後すぐに利用可能です。

# 出力例
```
┌───── OS Mood Barometer ─────┐
│ 本日の気分: バグ吸引フェイズ           │
│ 生産性: 低下中 (推定)                │
│ モード: アイデア枯渇ゾーン           │
│ Tips: 休憩すると運気が上がるかも？    │
└─────────────────┘
```

# 注意点
- 判定内容は完全ランダムで、実際の生産性・気分とは無関係です。
- 表示位置はターミナルの上下端やエディタのメニューバー風に自動調整されますが、極端に小さい画面ではレイアウトが崩れる場合があります。
- ローカルファイルや履歴には保存されません。
- 除外パスや特定のコマンドでの非表示設定は現状未対応です。
