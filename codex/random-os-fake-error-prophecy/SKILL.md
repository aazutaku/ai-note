# 機能概要
このSkillは、ターミナルやエディタでコマンドを実行するたびに、まるでOSが未来を予言するかのような“エラープロフェシー”通知をランダムに表示します。通知内容は「エラー: 明日プリンターが紙詰まりします」「警告: 来週、あなたのキーボードが反逆します」など、完全に架空かつ意味不明な未来のトラブル予告です。作業の合間に緊張感を和らげ、思わず笑ってしまうようなエンタメ体験を提供します。

# 使い方
- 明示呼び出し: `/skills menu` から `random-os-fake-error-prophecy` を選択、または `$random-os-fake-error-prophecy` を直接メンション。
- 暗黙発動: ターミナルやエディタでコマンド実行、ファイル保存、または `error`, `warning`, `future`, `prophecy` などのキーワードを含む操作時に自動発動。

# 出力例
```
エラー: 明日の午後、あなたのマウスが一時的に消失します
警告: 来週金曜、Wi-Fiが自我に目覚めます
予言: 3日後、あなたのエディタが詩を詠み始めます
エラー: 1時間後、CapsLockが勝手にONになります
警告: 次回の再起動時、時計が逆回転します
```

# 注意点
- 本Skillの通知は完全なジョークであり、実際のシステムや作業には一切影響しません。
- 本物のエラーや警告とは無関係です。混同に注意してください。
- 通知内容はローカルには保存されません。
- 除外パスや特定のコマンドに対する発動制御はありません。
