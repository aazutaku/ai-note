# 機能概要
random-os-fake-update-notifierは、作業に没頭し始めた瞬間や「アップデート」「通知」などの関連ワードが出現した際に、まるで本物のOSアップデート通知のような“くだらない”内容をデスクトップに表示するスキルです。通知内容は完全ランダムで、「カレンダーに猫の日を自動登録」「未確認のバグをさらに未確認化」など、明らかに嘘くさい更新情報が並びます。リモートワークや会議中、集中したい時にあえて発動させることで、場を和ませたり、ツッコミどころ満載の空気を作り出します。

# 使い方
- 明示呼び出し: `/skills random-os-fake-update-notifier` または `$random-os-fake-update-notifier` をチャットやターミナルで入力
- 暗黙発動: 「アップデート」「OS通知」「バグ修正」「新機能」などのキーワードが会話や作業ログに現れた際に自動発動
- CLIサブコマンド: `python notifier.py notify` で即座にランダム通知を表示

# 出力例
```
[OSアップデート通知]
バージョン: 13.4.9-fake
- 新機能: スタートメニューに“謎のボタン”を追加
- バグ修正: たまに現れる幽霊プロセスをさらに不可視化
- 既知の問題: アップデート後も何も変わらない場合があります
```

# 注意点
- 本Skillはジョーク用途です。実際のOSや業務システムとは一切連携しません。
- 通知内容は完全に架空であり、誤認防止のために「fake」や「ジョーク」等の文言を含みます。
- 通知履歴はローカル保存されません。
