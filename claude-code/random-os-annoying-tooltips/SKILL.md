# 機能概要
`random-os-annoying-tooltips`は、あなたの作業中に突然“懐かしのOS風うざいツールチップ”をデスクトップ通知として表示するスキルです。内容は「CapsLockが押されているかもしれません」「アップデートしませんか？」「入力速度が遅いかも」など、完全に余計なお世話な小言や謎の提案ばかり。集中している時ほど現れ、かつての“助けにならないヘルプ”体験を現代にリバイバルします。職場や自宅で「また出た！」と笑いとイライラをお届けします。

# 使い方
- 明示呼び出し例: `/random-os-annoying-tooltips`
- 暗黙発動キーワード例: 「通知」「アップデート」「CapsLock」「ヘルプ」「うざい」などを含む発言やコード実行時に自動発動します。
- CLIからは `python annoying_tooltips.py` で起動し、バックグラウンドで動作します。

# 出力例
```shell
[通知] CapsLockが押されているかもしれません。
[通知] 今すぐアップデートしませんか？
[通知] 入力速度が先週より5%低下しています。
[通知] 休憩を取るタイミングかもしれません。
[通知] あなたのクリップボードがいっぱいです。
```

# 注意点
- 本Skillは主要なOS（Windows, macOS, Linux）で動作しますが、通知機能はOSの仕様に依存します。
- 通知頻度は過剰にならないよう調整されていますが、長時間実行時はご注意ください。
- ローカル環境にのみ通知し、外部送信や保存は行いません。
- システム設定によっては通知が表示されない場合があります。
