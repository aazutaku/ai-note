# 概要
このSkillは、作業中の文脈（プロジェクトやタスク）切替時の理由や経緯を自動的に記録し、後から参照できるようにするものです。エージェントの作業履歴を透明化し、なぜその時その選択をしたかを振り返ることができます。

# 公式ドキュメント抜粋
Pythonの標準loggingモジュールやファイル操作APIのみを利用し、外部依存を極力排除しています。ログはローカルのユーザーディレクトリ配下に保存されます。

# 利用例
- プロジェクトAからBへの緊急切替時、その理由を即記録
- タスクレビューや顧客要望優先変更の履歴管理
- チーム全体の意思決定フローの可視化

# 注意点
- ログファイルはローカル保存のため、バックアップや共有が必要な場合は手動で対応してください。
- 個人情報や機密事項の記録には十分注意してください。

# 設計方針
シンプルなCLIとファイルベースの記録方式を採用し、導入・運用コストを最小化しています。明示的なコマンド呼び出しと、セマンティックな文脈検知の両方に対応しています。
