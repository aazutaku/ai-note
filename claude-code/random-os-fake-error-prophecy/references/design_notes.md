# 概要
`random-os-fake-error-prophecy`は、作業リズムの中に“根拠のない未来予言エラー”を挿入し、ユーザーに笑いや意外性を提供することを目的としています。実用性はなく、純粋なエンタメ要素です。

# 公式ドキュメント抜粋
- Python random: https://docs.python.org/3/library/random.html
- argparse: https://docs.python.org/3/library/argparse.html

# 利用例
- ターミナルやエディタ操作時に自動発動
- /random-os-fake-error-prophecy コマンドで明示呼び出し
- 履歴やサマリー表示で“未来予言”の傾向を確認

# 注意点
- 本Skillは実際のエラー通知と混同しないよう、現実には存在しない内容・表現を採用しています。
- ローカル保存される履歴はユーザーのホームディレクトリ配下で、個人情報や実作業内容は記録しません。

# 設計方針
- 予言内容・タイミングは完全ランダム
- 履歴・サマリー機能で“予言”の蓄積も楽しめる設計
- 実在API・標準ライブラリのみ利用し、外部依存を排除
