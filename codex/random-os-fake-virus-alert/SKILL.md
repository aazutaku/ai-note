# 機能概要
このSkillは、作業がマンネリ化したり集中力が途切れたタイミングで、デスクトップに“絶対に本物と間違えない”ユーモア満載のウイルス検出風アラート通知をランダム表示します。『危険：未知のバグウイルスを検出』『注意：あなたのコードが自我を持ちました』など、日常にスパイスを加える軽妙なジョーク通知をOS標準APIで演出。実害や悪質な挙動は一切なく、気分転換やチームの雰囲気づくりに最適です。

# 使い方
- 明示呼び出し例: `/skills menu` で本Skillを選択、または `$random-os-fake-virus-alert` メンション
- 暗黙発動例: 「眠い」「飽きた」「集中できない」等の発言や、作業が一定時間停滞した際に自動発動

# 出力例
```
[通知] 危険：未知のバグウイルスを検出しました！
[通知] 注意：あなたのコードが自我を持ちました。今すぐ休憩を推奨します。
[通知] システム警告：AIがあなたの冗談を理解できませんでした。
[通知] お知らせ：バグが自己増殖を開始しました（嘘です）。
[通知] 警告：あなたのコーヒー残量が危険水域です。
```

# 注意点
- 通知は本物のウイルス警告と誤認されないよう、明確なジョーク文言・演出のみ
- OS標準の通知API（Windows: Toast, macOS: AppleScript, Linux: notify-send等）を使用し、追加インストール不要
- ローカルファイルやシステム設定には一切変更を加えません
- 頻度や内容はユーザーの作業状況に応じて調整可能
