# 概要
Terminal BOSS BGM Suggesterは、ユーザーが本日最初にターミナル操作を行った際、完全ランダムに“本日のテーマソング”をテキストで提案するSkillです。BGMは著名なアニメ・ゲーム・クラシック等から選ばれ、日常の作業開始に遊び心を提供します。

# 公式ドキュメント抜粋
本Skillは明示呼び出し不要・alwaysトリガーで、semantic-match-onlyにより「最初のコマンド」状況を検知します。音声再生や外部APIは使用せず、Python標準のrandom・datetimeで実装されています。

# 利用例
- 毎朝の開発作業開始時に、ランダムなBGMタイトルで気分転換
- チームでの朝会前に話題作り
- 「今日は何が来るか」楽しみながら作業開始

# 注意点
- 提案は1日1回のみ
- テーマソングはユーザーの好みや状況を考慮しません
- 職場利用時は周囲への配慮を推奨

# 設計方針
状態管理はローカルJSONファイルで日付のみ記録。CLIサブコマンドでリセット・リスト表示・状態確認も可能。選曲アルゴリズムはrandom.choice()による完全ランダムです。
