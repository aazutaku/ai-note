# 概要
random-os-fake-boot-sound は、作業開始時に“謎のOS起動音”をランダム再生することで、ユーザー体験にユーモアや驚きを追加する演出系スキルです。音源の差し替えや追加が容易な点が特徴です。

# 公式ドキュメント抜粋
サウンド再生にはPythonの `playsound` ライブラリを利用しています。詳細は [playsound GitHub](https://github.com/TaylorSMarks/playsound) を参照してください。wav/mp3形式のローカルファイル再生に対応しています。

# 利用例
- ターミナルやVSCode等のエディタ起動時に自動発動
- チームの朝会やイベント開始時のネタ演出
- 複数の音源を用意し、毎回違う“OS起動”体験を提供

# 注意点
- 音源は `.agent/skills/random-os-fake-boot-sound/sounds/` に配置してください。
- サウンド再生はローカル環境依存です。Dockerやリモート環境では音声出力に制限がある場合があります。
- 音量や出力デバイスはOS側で調整してください。

# 設計方針
Skill本体はシンプルな構成とし、音源の差し替えや追加が容易なディレクトリスキャン方式を採用。CLIサブコマンドで確認やデバッグも可能にしています。
