# 概要
このSkillは、ユーザーの作業アクションごとに完全にランダムな“自己肯定感ハイプバー”を表示することで、作業の合間にユーモラスな演出を提供します。実用性や正確性を一切持たず、気分転換やモチベーションの演出に特化しています。

# 設計方針
- パラメータ名・数値は毎回ランダム生成し、同じ組み合わせが続かないよう工夫しています。
- ログ機能や履歴集計も実装し、過去の“ハイプバー”を振り返ることも可能です。
- システムや実際の進捗とは無関係で、あくまで通知・演出目的です。

# 利用例
- コマンド実行時に自動でハイプバー表示
- `/random-os-motivational-hypebar` で明示呼び出し
- 履歴や集計を活用してカオスな変遷を楽しむ

# 注意点
- 実際の作業進捗や健康状態の指標には一切なりません。
- システムディレクトリや自動バッチ処理時には発動しません。
