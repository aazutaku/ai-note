# 概要
terminal-mood-indicatorは、ターミナル作業時の気分を可視化し、開発現場の雰囲気を和らげるための演出Skillです。環境変数や設定ファイルを利用し、気分の切り替えや表示を柔軟に行います。

# 公式ドキュメント抜粋
- Bash/Zshの環境変数設定方法: https://www.gnu.org/software/bash/manual/html_node/Environment.html
- Python subprocess, argparse 標準ライブラリ

# 利用例
- 朝の作業開始時に `/terminal-mood-indicator set "やる気MAX"` を実行し、以降のコマンドやプロンプトに気分を表示。
- チーム開発で各自の気分を可視化し、コミュニケーションのきっかけに。

# 注意点
- プロンプト自動書き換えには、シェルの設定ファイル編集が必要です。
- 気分はローカルユーザーごとに保存され、他ユーザーの環境には影響しません。

# 設計方針
- シンプルなCLI/環境変数連携を重視し、外部依存を最小化。
- 気分リストは拡張可能。今後はカスタム気分や外部API連携も検討。
