# 機能概要
このSkillは、普段の作業やコマンド実行の合間に、完全ランダムなタイミングでOSの通知領域へ“謎のコンフェッティ祝福メッセージ”を表示します。内容は「今日も出社おめでとう」「意味なく祝福します」「さっきのls、最高でした」など、全く脈絡のないネタ系お祝いばかり。作業に没頭しがちな開発現場に、唐突な非日常と笑いを提供し、気分転換やチームの雰囲気向上に役立ちます。

# 使い方
- 明示呼び出し例:
  `/random-os-notification-confetti`
- 暗黙発動キーワード例:
  - 「通知」「祝福」「ランダム」「気分転換」「演出」などの単語を含む会話やコマンド実行時
  - 何らかのCLIコマンド（例: ls, git commit, make など）実行直後に自動発動する場合もあり

# 出力例
```
[通知] 本日も出社おめでとうございます！
[通知] さっきのls、最高でした。
[通知] 何もしてないけど祝っておきます。
[通知] 意味なく祝福します。紙吹雪！
[通知] あなたのタイピング、見事でした。
[通知] 今日の作業、すでに伝説です。
[通知] コーヒー休憩も祝福タイム！
```

# 注意点
- 通知はOSの標準通知APIを利用します（Linux: notify-send, macOS: osascript, Windows: win10toast）。
- 通知内容は完全ランダムで、意味や文脈は考慮しません。
- 通知頻度は内部で自動調整され、過剰なスパムを防ぎます。
- ローカル環境でのみ動作し、通知内容は保存されません。
- 一部の環境では通知が表示されない場合があります。
