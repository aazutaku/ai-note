# 機能概要
このSkillは、日々の開発作業の中で開発者自身の気分やテンションを、気温や天気に見立ててターミナルのステータスバーやプロンプトに表示します。コミット頻度や作業履歴、または完全ランダムな値をもとに「今日の気分：曇り、気温18度」「テンション：真夏日」などのメッセージを生成し、業務の合間にちょっとした癒しや謎の自己分析を提供します。実用性よりも“ジワジワくる演出”を重視し、単調な開発現場に意味不明な気象レポートをお届けします。

# 使い方
- 明示呼び出し: `/terminal-weather-mood-bar` または `python weather_mood_bar.py show`
- 暗黙発動: 「気分」「天気」「テンション」「ステータスバー」などのキーワードや、ターミナルのプロンプト更新時、コミット直後などに自動発動します。

# 出力例
```
[今日の気分] 曇り時々晴れ  体感温度: 21℃
[テンション] 春一番レベル（やや高め）
[気分予報] 夕方から雨のち快晴
[コミット頻度] 3回/日 → 気温: 19℃
--------------------------------------
```

# 注意点
- 実際の天気APIは使用せず、気分・テンションはコミット頻度やランダム値から生成されます。
- ステータスバーやプロンプトへの自動挿入は、シェル設定や外部ツール連携が必要な場合があります。
- ローカルのGitリポジトリが必要です（commit頻度取得時）。
- 出力内容はジョーク・演出目的であり、実用的な気象情報や心理診断ではありません。
