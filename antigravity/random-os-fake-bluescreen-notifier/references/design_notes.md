# 概要
このSkillは、開発現場やリモートワークの合間に、ブルースクリーン風のジョーク通知をデスクトップに表示することで、緊張感を和らげたり、チーム内にユーモアを提供する目的で設計されています。

# 公式ドキュメント抜粋
- plyer: https://github.com/kivy/plyer
- win10toast: https://github.com/jithurjacob/Windows-10-Toast-Notifications
- notify2: https://github.com/caronc/notify2

# 利用例
- 朝会やペアプロ中のアイスブレイク
- 長時間作業後のリフレッシュ促進
- チームチャットで「ブルースクリーン通知」などの話題が出た際の自動発動

# 注意点
- 本Skillはシステムやファイルに一切変更を加えません。
- GUI環境がないサーバやSSH端末では通知が表示されません。
- 本物のエラーや障害と誤認されないよう“FAKE”明記や明るい文調を徹底しています。

# 設計方針
- OSごとに最適な通知APIを選択し、依存ライブラリが未導入時は標準出力にフォールバックします。
- 通知内容は毎回ランダム生成し、ユーモアと安全性を両立しています。
