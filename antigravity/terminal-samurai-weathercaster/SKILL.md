# 機能概要
terminal-samurai-weathercasterは、ターミナル起動時や新規セッション開始時に、侍が現れてその日の作業気分を天気予報風に“斬り捨て御免”で表現する演出系Skillです。現実の天気やAPIには一切依存せず、完全ランダムな天気・侍コメントが毎回表示されます。真面目な開発作業に、侍の不可解な乱入を加えることで、日々のコーディングにユーモアと新鮮さを提供します。

# 使い方
このSkillはターミナル起動時や新しいシェルセッション開始時に自動で発動します（明示的な呼び出しコマンドは不要）。semantic-match-onlyトリガーにより、"ターミナル" "terminal" "shell" "セッション開始" などのキーワードを含むイベントで常時有効です。明示呼び出しはありません。

# 出力例
```
─── 本日の侍天気予報 ───
快晴なり。コードも冴え渡るでござる！
────────────────────

─── 本日の侍天気予報 ───
雷注意報。バグの嵐、油断召されるな！
────────────────────

─── 本日の侍天気予報 ───
曇天にて集中力やや低下、無理は禁物でござる。
────────────────────
```

# 注意点
- 現実の天気や位置情報、外部APIは一切参照しません。
- 出力内容は毎回完全ランダムです。
- ローカルへの履歴保存や通知連携はありません。
- 明示的なコマンド呼び出しや引数指定は不要です。
