# 機能概要
このSkillは、作業中のユーザーに対して「現在の緊張度」をデスクトップ画面端へテロップ形式でランダム表示するものです。緊張度は完全なランダム値、またはGitのcommit数やタイピング速度など“根拠になりそうでならない”数値から算出され、「平常心」「やや焦り」「極度緊張」など謎のラベルで演出します。実用性は皆無ですが、作業空間に謎のプレッシャーや無駄な緊張感を演出し、職場や自宅PCでの作業体験に“無意味な刺激”を加えます。

# 使い方
- 明示呼び出し: `/desktop-tension-meter`
- 暗黙発動キーワード例: 「緊張」「集中」「作業効率」「通知」「プレッシャー」などを含む会話やコマンドで自動発動します。
- Skillはバックグラウンドで動作し、一定時間ごとまたはランダムなタイミングで通知を表示します。

# 出力例
```
[desktop-tension-meter]
緊張度: 87%（やや焦り）
次回表示まで: 5分

[desktop-tension-meter]
緊張度: 42%（平常心）
次回表示まで: 12分

[desktop-tension-meter]
緊張度: 99%（極度緊張）
次回表示まで: 3分
```

# 注意点
- 通知はローカルPCのデスクトップ通知機能（例: Windows/Mac/Linuxのnotify-send等）を利用します。
- 強制終了や誤作動時にデータ損失は発生しません。
- 通知頻度や内容は設定可能ですが、初期値は“やや鬱陶しい”程度に調整されています。
- コミット数やタイピング速度取得はローカル環境依存です。
