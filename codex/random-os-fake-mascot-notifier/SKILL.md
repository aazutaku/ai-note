# 機能概要
このSkillは、あなたの作業中に“謎のOSマスコット”が突如デスクトップ通知で現れ、全く役に立たない激励や警告をランダムに発するものです。登場キャラクターやメッセージ内容、通知タイミングは完全にカオスで予測不能。作業の合間に現実逃避やリフレッシュを促します。通知は実際のデスクトップ領域に表示され、現実のOSのマスコットではなく、完全オリジナルのキャラとメッセージが毎回生成されます。

# 使い方
- 明示的呼び出し: `/skills menu` または `random-os-fake-mascot-notifier` のメンションで即時発動。
- 暗黙発動: 「マスコット」「通知」「ゆるキャラ」「応援」「警告」などの単語が会話やコマンドに含まれる場合、または一定時間作業が継続した際に自動発動します。

# 出力例
```
[通知] カーネルくん: 今日もメモリリークしないようにね！
[通知] ビットちゃん: あなたの成功をビット単位で数えています。
[通知] メモリ犬: 休憩しないとスワップしちゃうぞ！
[通知] タスクキャット: そのタスク、まだ終わってないよ！
[通知] パーミッションパンダ: もっと自分に許可をあげて！
```

# 注意点
- 通知はOSのデスクトップ通知機能（Windows/macOS/Linux）を利用します。
- ログや履歴はローカルに保存されません。
- 通知頻度は自動制御され、うるさすぎない設計です。
- 本Skillは実在のマスコットやOSとは一切無関係です。
