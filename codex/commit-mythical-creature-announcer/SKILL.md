# 機能概要
commit-mythical-creature-announcerは、Gitコミットのたびにターミナルへ“架空の神獣・幻獣”と、その日の開発運勢をランダムに表示するSkillです。フェニックスやドラゴン、ユニコーンなどの神話生物が日替わりで現れ、理不尽かつ笑える運勢コメントを添えて、開発現場に謎の神話的カオスをもたらします。真面目な作業にちょっとした息抜きや会話のきっかけを提供し、チームの雰囲気を柔らかくします。

# 使い方
- 明示呼び出し例: `/skills menu` または `commit-mythical-creature-announcer` のメンションで発動。
- 暗黙発動: `commit`、`コミット`、`push` などの操作やキーワードを含むイベントで自動的に発動します。

# 出力例
```
🐉 ドラゴンが眠りから目覚めた…今日は進捗が静かに進む予感。
🦄 ユニコーンが首をかしげている…バグは慎重に扱いましょう。
🦅 フェニックスが炎とともに蘇る…本日はリファクタリング吉日！
🦖 ケルベロスが吠えている…コミットコメントは短めに。
🦋 グリフォンが空を舞う…今日はレビュー運が上昇中！
```

# 注意点
- 出力内容は完全にランダムで、実際の運勢や進捗には一切影響しません。
- ローカル環境でのみ動作し、外部サーバーへのデータ送信はありません。
- 長時間の連続コミット時は同じ神獣が出る場合もあります。
