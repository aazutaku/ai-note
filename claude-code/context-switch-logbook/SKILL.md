# 機能概要
このSkillは、エージェントが作業中にプロジェクトやタスクのコンテキストを切り替えた際、その理由や経緯を自然な日本語で自動記録します。記録はローカルの専用ログファイルに追記され、後から「なぜ今この流れ？」を即座に振り返ることができます。これにより、作業の流れや意思決定の履歴が明確になり、チームや個人でのタスク管理が効率化されます。

# 使い方
- 明示呼び出し例: `/context-switch-logbook "バグ修正から新機能開発に切り替え。顧客要望の優先度が上がったため。"`
- 暗黙発動キーワード例: "プロジェクト変更", "タスク切替", "文脈移動", "作業内容変更"
Skillはこれらのキーワードや明示コマンドを検知して自動的にログを記録します。

# 出力例
```
[2024-06-15 14:22:01] プロジェクトAからプロジェクトBへ切り替え。緊急対応の必要が発生したため。
[2024-06-15 15:10:45] タスクXからタスクYへ移動。レビュー依頼が届いたため。
[2024-06-15 16:00:12] バグ修正から新機能開発に切り替え。顧客要望の優先度が上がったため。
```

# 注意点
- ログはローカルの`.claude/skills/context-switch-logbook/context_switch_log.txt`に追記保存されます。
- ファイルの消失・破損に注意してください。
- プライバシー保護のため、個人情報や機密事項の記録には十分注意してください。
- Skillは明示的な呼び出し、または文脈切替に関する発言を検知してのみ発動します。
