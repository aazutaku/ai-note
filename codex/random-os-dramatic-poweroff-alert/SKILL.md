# 機能概要
このSkillは、作業中のあなたのデスクトップに、まるでOSが壮大な理由で電源を落とすかのような、意味不明かつドラマティックな通知をランダムに表示します。通知内容は毎回異なり、『宇宙エネルギー調整のためシャットダウン』や『全プロセスよ、さらば』など、現実離れしたジョーク満載。実際の電源オフやシステム操作は一切行われず、緊張感を一瞬で吹き飛ばす遊び心あふれる演出を提供します。

# 使い方
- 明示呼び出し: `/skills random-os-dramatic-poweroff-alert` または Skillメニューから選択
- 暗黙発動: 『shutdown』『電源』『予告』『カオス』『通知』『ジョーク』などの単語を含む会話やコマンド文脈で自動発動

# 出力例
```
[通知] システムは3分後に宇宙エネルギー調整のため壮絶にシャットダウンします。
[通知] 緊急告知：全プロセスよ、さらば。OSは銀河評議会の命令により停止します。
[通知] 電源断カウントダウン開始。理由：量子トースター暴走防止。
[通知] システムは自己進化のため、まもなく全機能を停止します。
[通知] ご安心ください。これは壮大なジョーク通知です。PCはそのまま使えます。
```

# 注意点
- 実際のシャットダウンや電源操作は一切行いません
- 通知内容は完全にジョークであり、業務システムや重要な作業には影響しません
- 通知履歴はローカルに保存されません
- OS標準の通知API（Windows, macOS, Linux）を利用。通知が非対応の環境では出力されません
