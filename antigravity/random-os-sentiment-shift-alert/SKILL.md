# 機能概要
このSkillは、ユーザーが作業に没頭している最中や、一定時間操作がなかった場合などに、まるでOSの人格が急変したかのようなランダム通知をデスクトップに表示します。通知内容は「本日よりツンデレモードです」「しばらく詩人になります」など毎回異なり、実際のOS挙動や設定には一切影響を与えません。仕事中に不意打ちで現れる不可解な通知により、思わず二度見したくなるようなユーモアと意外性を演出します。

# 使い方
このSkillは明示的な呼び出しは不要で、Antigravityのsemantic-match-onlyトリガーにより自動発動します。長時間の無操作や「集中」「作業」「退屈」「眠い」といったキーワードが検出されると、ランダムなタイミングで通知が表示されます。手動でスクリプトを実行する場合は、`python random_os_sentiment_shift_alert.py` で即座に通知を出すことも可能です。

# 出力例
```
[通知] OS人格変更のお知らせ
本日よりツンデレモードに移行します。以後、指示には素直に従いません。

[通知] OS人格変更のお知らせ
しばらく詩人になります。すべてのエラーメッセージが短歌になります。

[通知] OS人格変更のお知らせ
今からやたら上から目線で案内します。覚悟してください。
```

# 注意点
- このSkillは通知表示のみを行い、OSや作業環境の設定・挙動には一切影響しません。
- 通知内容はローカルに保存されません。
- Linux/macOSでは`notify-send`や`osascript`、Windowsでは`win10toast`等のAPIを利用します。
- 通知が表示されない場合は、通知APIの依存パッケージや権限設定を確認してください。
