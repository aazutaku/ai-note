# 概要
このSkillは、作業中のユーザーに対して完全に架空の未来トラブルを“予言”することで、緊張感を和らげたり、遊び心を提供することを目的としています。

# 公式ドキュメント抜粋
本SkillはPython標準ライブラリのみを利用し、外部APIやシステムリソースには一切アクセスしません。通知はすべて標準出力です。

# 利用例
- ターミナルで `python random_os_fake_error_prophecy.py log` を実行すると、1件の予言が表示されます。
- `list` サブコマンドで複数の予言を一度に確認可能。
- `summary` でどのデバイス・イベントが多いか集計できます。

# 注意点
- 本Skillは実際のエラーやシステム挙動には一切影響しません。
- 予言内容は完全なフィクションです。混乱を避けるため、業務システム等での常時表示は推奨しません。

# 設計方針
- 予言文はテンプレート・デバイス・イベント・未来時刻をランダムに組み合わせて生成。
- 通知頻度やタイミングはトリガー側で制御可能。
- ログ保存機能は任意で利用可能です。
