# 機能概要
このSkillは、ターミナルやエディタでコマンドを実行するたびに、架空の「OSアップデート進捗バー」を画面端やメニューバー風に一時的に表示します。進捗内容は完全にランダムで、「バグ収穫中」「やる気パッチ適用中」「怠惰アップグレード中」など、意味不明な項目ばかり。進捗が100%になっても何も起きず、作業の緊張感を一瞬で脱力させる“無駄な演出”を提供します。

# 使い方
- 明示呼び出し: `/random-os-fake-update-progrebar` を入力すると即座に進捗バーが表示されます。
- 暗黙発動: `run`, `build`, `test`, `deploy`, `save`, `commit`, `push` など、主要なコマンドやファイル操作時に自動的に発動します。

# 出力例
```
[==>                ] 12% バグ収穫中...
[=======>           ] 38% やる気パッチ適用中...
[============>      ] 67% 怠惰アップグレード中...
[==================>] 100% 仕様の穴を拡張中...
```

# 注意点
- 実際のOSやファイルには一切影響しません。
- 進捗バーは10秒以内に自動で消えます。
- ログや履歴は残さず、ローカル保存もしません。
- 進捗内容や表示位置は毎回ランダムです。
