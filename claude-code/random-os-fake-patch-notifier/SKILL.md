# 機能概要
このSkillは、開発現場や作業中のユーザーに対して、完全ランダムなタイミングで“謎のOSパッチノート”をデスクトップ通知として表示します。通知内容は毎回異なり、実際には存在しない奇妙なバグ修正や新機能（例：「バグ：やる気が0になる問題を修正」「新機能：午後3時に眠気を自動注入」）など、ユーモアと混乱をもたらします。集中をほぐしたい時や、チームに笑いを提供したい時に最適です。

# 使い方
- 明示呼び出し: `/random-os-fake-patch-notifier` を実行すると即座に通知が発生します。
- 暗黙発動: 「アップデート」「パッチ」「OS」「作業効率」などのキーワードを含む会話やコマンド実行時、または完全ランダムなタイミングで自動発動します。
- 通知はOS標準の通知システムを利用し、内容・タイミングともに毎回異なります。

# 出力例
```
[通知] OS Patch Notes 2.7.13
- バグ修正: キーボードの「やる気」キーが物理的に消失する問題を解決
- 新機能: 午後3時になると自動で睡魔を注入します
- 既知の問題: コーヒー検出機能が誤作動する場合があります
```

# 注意点
- 通知内容は完全に架空で実用性はありません
- ローカル環境の通知API（例: notify-send, Windows Toast, macOS通知）を利用します
- スクリプトは作業内容やファイルに一切影響を与えません
- 除外パスや保存先の指定は不要です
