# 概要
このSkillは、作業開始時の気分転換やアイスブレイクを目的に、毎回バカバカしいBGMタイトルを通知する設計です。実際の音楽再生は行わず、通知のみで完結します。

# 公式ドキュメント抜粋
OS通知にはPythonの plyer ライブラリ（https://github.com/kivy/plyer）を利用。macOS, Windows, Linuxで安定動作。通知APIが利用できない場合は標準出力にフォールバックします。

# 利用例
- ターミナルやエディタ起動時に自動発動
- `/random-os-soundtrack-notifier --list`で候補を一覧
- `/random-os-soundtrack-notifier --log`で過去通知履歴を確認

# 注意点
- システムディレクトリやCI環境では通知を抑制
- ログはホームディレクトリ直下に保存
- plyer未インストール時は標準出力のみ

# 設計方針
「バカバカしさ」「理不尽さ」を重視し、ジャンル・年代・雰囲気がバラバラなBGMリストを用意。通知だけで完結し、余計なリソース消費や誤動作を防止。
