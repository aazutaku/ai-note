# 機能概要
commit-mood-forecastは、Gitリポジトリでコミット操作を実行するたびに、ターミナル上へ“本日のあなたの作業気分天気予報”をランダムに表示するSkillです。天気アイコン風のテキストとともに「快晴：今日は絶好調」「曇り：ちょっと迷い気味」「雷：やる気ゼロ」など、毎回異なるコメントが出現します。真面目な開発フローに遊び心を加え、チームの雰囲気を和ませる脱力系演出Skillです。実用的な情報ではありませんが、日々の作業にちょっとした笑いをもたらします。

# 使い方
- 明示呼び出し: `/commit-mood-forecast` コマンドでいつでも発動可能。
- 暗黙発動: `git commit` 実行時や「コミット」「push」などのキーワードを含む操作時に自動で発動します。
- 追加設定や外部依存は不要。Skillを有効化するだけで即利用できます。

# 出力例
```
[commit-mood-forecast]
本日のあなたの作業気分天気予報：
  ☀️ 快晴！今日は絶好調。どんどん進めましょう。
```
```
[commit-mood-forecast]
本日のあなたの作業気分天気予報：
  ☁️ 曇り。ちょっと迷い気味？コーヒーブレイクも大事です。
```
```
[commit-mood-forecast]
本日のあなたの作業気分天気予報：
  ⚡ 雷注意報。やる気ゼロの日もあります。
```

# 注意点
- 出力は毎回ランダムで、実用的な意味はありません。
- ローカル環境でのみ表示され、履歴やファイルには保存されません。
- 長文や過度な通知は避け、作業を邪魔しません。
- 除外パスや特定ファイルには影響しません。
