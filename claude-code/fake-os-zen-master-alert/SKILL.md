# 機能概要
fake-os-zen-master-alertは、作業やコーディングの合間に“偽OSの禅マスター”が現れ、完全無意味な禅問答や悟りの極意をデスクトップ通知で投げかけるSkillです。通知内容は実務や進捗に一切関係なく、集中を乱したり、時に深遠な（？）気づきを与えます。バグ修正やビルドの合間、長時間の作業時などに、意図せず思考を混乱させるエンタメ系通知を体験できます。

# 使い方
- 明示呼び出し: `/fake-os-zen-master-alert on` で通知開始、`/fake-os-zen-master-alert off` で停止。
- 暗黙発動: "ビルド完了", "テスト失敗", "ファイル保存", "長時間アイドル", "デバッグ開始" などのタイミングで自動発動。
- 通知頻度やON/OFFはコマンドライン引数で制御可能。

# 出力例
```
[禅マスター] バグとは何かを問う前に、己を問え。
[禅マスター] 今日の悟り：if文を捨てよ、道は開ける。
[禅マスター] コンパイルに失敗したとき、木魚を叩け。
[禅マスター] エラーの数だけ、悟りの道は続く。
[禅マスター] 眠気と戦うより、眠気を受け入れよ。
```

# 注意点
- 通知はOSのネイティブ通知機能（Windows, macOS, Linux対応）を利用。
- 実務に全く役立たないメッセージのみが表示されます。
- 通知内容はローカル保存されません。
- 頻度やON/OFFはコマンドで制御できますが、完全な無効化は明示的に必要です。
