# 概要
本Skillは、作業の停滞や気分転換が必要なタイミングで、OSの通知機能を活用し、ランダムな『謎のモチベーション格言』を表示します。エンタメ性と話題作りを重視しつつ、導入・削除が容易なシンプル設計です。

# 公式ドキュメント抜粋
- plyer: クロスプラットフォームな通知APIを提供。https://github.com/kivy/plyer
- Python random, argparse: 標準ライブラリで依存性最小化。

# 利用例
- `/skills menu` から明示呼び出し
- 作業進捗が停滞した際の気分転換
- チーム内の話題提供やアイスブレイク

# 注意点
- 通知はローカル端末でのみ表示。SSHやリモートデスクトップ越しでは不可。
- 格言の追加・編集はスクリプト内で対応。
- バックグラウンド常駐型ではないため、定期通知には未対応。

# 設計方針
- 必要十分な機能・コード量に留め、導入・削除が容易
- エラー時は標準エラー出力に明示
- クロスプラットフォーム性を考慮しつつ、主要OSでの動作を優先
