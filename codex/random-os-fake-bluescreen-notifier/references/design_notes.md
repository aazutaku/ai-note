# 概要
本Skillは、開発現場やリモートワーク時の息抜きやチームコミュニケーション促進を目的とし、実害のない“ブルースクリーン風”通知をランダムに生成・表示します。

# 公式ドキュメント抜粋
通知表示にはPythonのplyerライブラリを利用。plyerはクロスプラットフォーム対応の通知APIを提供し、Windows/macOS/Linuxで動作します。

# 利用例
- チームの朝会や雑談タイムに、話題のきっかけとして利用
- 長時間作業時のリフレッシュ用リマインダーとして
- プロジェクトの進捗が停滞した際のムード転換

# 注意点
- 本Skillはユーザーのデスクトップ通知機能に依存します。通知が表示されない場合はOS側の設定を確認してください。
- 実際のシステムエラーやBSoDと誤認されないよう、明確にジョークである旨を明記しています。

# 設計方針
- 実在するAPIのみを使用し、システムやデータに一切影響を与えない設計
- 通知内容は毎回ランダム生成され、出力例やヒントも複数パターンを用意
- ログ機能を備え、過去の通知内容を振り返ることも可能
