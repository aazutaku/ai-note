# 機能概要
terminal-weather-mood-barは、開発者の気分やテンションを気象情報に見立て、ターミナルのステータスバーやプロンプトに表示するスキルです。実用性よりも癒しやユーモアを重視し、commit頻度やランダム値をもとに「今日の気分：曇り、気温18度」や「テンション：真夏日」などの“気分天気”を自動生成します。日々の単調な開発現場に、意味不明な気象レポートでちょっとした笑いと自己分析のきっかけを提供します。

# 使い方
このSkillは、ターミナルやシェルのプロンプト描画タイミング、または「気分」「テンション」「天気」などのキーワードを含む会話・コマンド実行時に自動で発動します。明示的な呼び出しは不要です。

**暗黙発動キーワード例:**
- 「今日の気分は？」
- 「テンション表示」
- 「天気みたいに気分を出して」
- 「commit頻度から気分を予報して」

# 出力例
```
[terminal-weather-mood-bar] 今日の気分：小雨、気温17度
[terminal-weather-mood-bar] テンション：真夏日（32度）
[terminal-weather-mood-bar] 気分天気：曇り時々晴れ、21度
[terminal-weather-mood-bar] 本日の気分：快晴、気温28度
[terminal-weather-mood-bar] テンション：冬日（-2度）
```

# 注意点
- 実際の天気や体調とは無関係のランダム/commit頻度ベースの演出です。
- ターミナルやプロンプトのカスタマイズが必要な場合があります。
- ローカルのgitリポジトリがない場合は完全ランダム予報になります。
- 気分天気の内容はジョーク目的であり、業務判断には使えません。
