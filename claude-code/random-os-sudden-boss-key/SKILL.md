# 機能概要
このSkillは、作業中に突如“謎のボスキー”を発動し、現在のターミナルやエディタ画面を一瞬だけ意味不明なダミー画面へ切り替えます。表示される内容は毎回完全ランダムで、「業務資料っぽさゼロの謎グラフ」や「よくわからない警告ウィンドウ」など、むしろ怪しまれる演出です。緊張感のある職場環境を一瞬でカオスに変えたいとき、またはジョークやストレス発散のために活用できます。

# 使い方
明示的な呼び出し例:

```
/random-os-sudden-boss-key
```

暗黙的な発動キーワード例:
- "ボスキー発動"
- "画面を隠したい"
- "急に誰か来た"

発動すると、現在の画面が即座にランダムなダミー画面に切り替わり、任意のキー入力や一定時間経過で元の画面に復帰します。

# 出力例
```
=== WARNING ===
OS Error: 0xDEADBEEF
System integrity check failed.
[OK]

---

┌─────────────┐
│  RANDOM DATA CHART │
├─────┬─────┤
│ Foo │  42 │
│ Bar │  99 │
│ Baz │   7 │
└─────┴─────┘
```

# 注意点
- 本Skillは実験的な演出用です。実際の業務資料や本物の警告画面ではありません。
- ローカルでのみ動作し、外部サーバ等へ情報送信はありません。
- 画面切替は一時的で、復帰後に内容は保存されません。
- X11/Wayland環境またはWindows/macOSの標準Pythonで動作しますが、環境によっては一部機能が制限される場合があります。
