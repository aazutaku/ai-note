# 機能概要
このSkillは、ターミナルやエディタの起動時、あるいは明示的なコマンド呼び出し時に、架空のOSや謎のシステムの“起動画面”をテキストアニメーションで毎回ランダム生成・表示します。『Antigravity 3000 起動準備中』『Codex Limited Edition 起動…』『ビットを配置中』『AIを目覚めさせています』など、現実には存在しないOS名や進捗バー、意味不明なシステムメッセージで、作業前の一瞬を未来感と謎の緊張で満たします。実用性はほぼゼロですが、退屈な起動時にちょっとした非日常を演出します。

# 使い方
- 明示呼び出し例: `/skills random-os-startup-movie` または `@codex random-os-startup-movie`
- 暗黙発動キーワード例: 「起動」「OS」「開始」「準備」「システム」などが含まれるタイミングで自動発動

# 出力例
```
[Antigravity 3000] 起動画面
============================
メモリ空間を非現実化中... 12%
AIコアを目覚めさせています... 38%
量子ビットを並列配置中... 71%
未知のプロトコルを初期化... 100%
Welcome to Antigravity 3000 OS (Build 42.42.42)

[Codex Limited Edition] 起動中
-----------------------------
仮想次元を同期中... 23%
イマジナリーユーザーを生成... 67%
システム準備完了。
```

# 注意点
- 実在しないOSや進捗内容のみを生成します。実際のシステム状態やファイルには影響しません。
- ローカル保存や履歴記録は行いません。
- 演出内容は毎回完全ランダムで再現性はありません。
