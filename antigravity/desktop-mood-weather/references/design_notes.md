# 概要
このSkillは、ユーザーの実際の気分や天気情報とは無関係に、完全ランダムな“気分天気”をデスクトップに表示することを目的としています。ユーモラスな演出や作業空間のカオス化を狙っています。

# 公式ドキュメント抜粋
- Tkinter: https://docs.python.org/ja/3/library/tkinter.html
- Pillow: https://pillow.readthedocs.io/en/stable/

# 利用例
開発中や会議中、作業の合間にデスクトップ右下に突然“雷”や“大雪”などの天気アイコンが現れ、ちょっとした話題や気分転換になります。Skillは常駐型で、10分ごとに新しい天気がランダム表示されます。

# 注意点
本Skillは実際の気象APIやユーザーの感情分析とは一切連動しません。Tkinterのウィンドウ表示は一部Linux環境やマルチディスプレイ構成で位置ズレが発生する場合があります。アイコン画像はローカル生成・保存され、ネットワーク不要です。

# 設計方針
シンプルなローカル実装を重視し、外部APIや大規模依存を避けています。Pillowによる動的アイコン生成により、追加天気パターンも容易に拡張可能です。
