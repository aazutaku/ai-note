# 概要
このSkillは、デスクトップ上に無意味な「緊張度」テロップを表示し、作業環境に謎のプレッシャーを演出するために設計されています。実用性よりも、職場や個人のPCでの“無駄な演出”を重視しています。

# 公式ドキュメント抜粋
- [Tkinter](https://docs.python.org/ja/3/library/tkinter.html): Python標準GUIで、OS間の互換性が高く、手軽にウィンドウ表示が可能
- [psutil](https://psutil.readthedocs.io/ja/latest/): システムリソース取得用（拡張性のため）
- [keyboard](https://github.com/boppreh/keyboard): タイピング速度計測に利用

# 利用例
- チーム内のネタとして、会議や作業時間に“緊張度”を可視化
- 個人の集中力維持や、気分転換のためのジョークツール

# 設計方針
- テロップは画面端に短時間のみ表示し、鬱陶しさを抑制
- 緊張度算出は完全ランダムまたは“根拠になりそうでならない”値を組み合わせ
- ログはローカルファイルのみ、外部送信なし

# 注意点
- 実際の心理状態や生産性とは無関係です
- 通知頻度はパラメータで調整可能
- 強制終了時もデータ損失はありません
