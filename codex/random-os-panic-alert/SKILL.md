# 機能概要
このSkillは、普段の作業やコーディング中に突如として“謎のOSパニックアラート”をデスクトップ通知で表示します。通知内容は毎回ランダムに生成され、「カーネルがランチに出ました」「ビットが逃走しました」「仮想メモリが現実逃避中です」など、現実では絶対に見ない珍妙なメッセージばかり。深刻そうで全く深刻ではない通知が、作業の合間に絶妙な違和感とユーモアをもたらします。

# 使い方
- 明示呼び出し例: `/skills random-os-panic-alert` または Skillメニューから選択
- 暗黙発動: 「panic」「OSエラー」「クラッシュ」「アラート」等の単語を含む会話・ログ・コード断片を検出時に自動発動

# 出力例
```
[通知] OS Panic Alert: カーネルがランチに出ました
[通知] OS Panic Alert: ビットが逃走しました。追跡中...
[通知] OS Panic Alert: 仮想メモリが現実逃避中です
[通知] OS Panic Alert: システムクロックが未来に到達しました
[通知] OS Panic Alert: デバッグ用アヒルが迷子です
```

# 注意点
- 本Skillの通知は現実のシステム障害とは無関係です。
- 通知頻度は1回/10分を上限とし、鬱陶しくならないよう設計されています。
- 通知はローカル環境のみに表示され、履歴はSkill内でのみ保持されます。
- macOS/Linux/Windowsの主要な通知APIに対応していますが、仮想環境やリモート接続時は通知が表示されない場合があります。
