# 概要
context-path-history-visualizerは、作業中のファイルやプロンプトの履歴を自動記録し、分岐や合流を含めてツリー表示するSkillです。履歴の可視化により、思考経路の把握や再利用ポイントの発見が容易になります。

# 公式ドキュメント抜粋
履歴管理にはPythonのjson・argparse・osモジュールのみを利用し、外部依存を排除しています。公式: https://docs.python.org/3/library/

# 利用例
- 複雑な編集フローの全体像を把握したいとき
- 過去の分岐点や検討漏れを確認したいとき
- 編集や参照が多岐にわたるプロジェクトの整理

# 注意点
履歴はユーザーホームの隠しファイルに一時保存されますが、セッション外での永続利用は想定していません。除外パスは自動フィルタされます。

# 設計方針
CLIサブコマンドで履歴の記録・一覧・ツリー表示・クリアを分離し、拡張性と保守性を重視しています。履歴の分岐・合流も正確に表現できるよう、親子関係を記録しています。
