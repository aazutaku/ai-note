# 機能概要
このSkillは、あなたの作業環境に突如“偽ウイルスアラート”のデスクトップ通知を表示します。内容は完全にユーモラスかつ明らかにネタで、本物のウイルス警告と誤認される心配はありません。『危険：未知のバグウイルスを検出』『注意：コードが自我を持ちました』など、日常の作業にスパイスを加えます。実際のファイル操作やシステム変更は一切行わず、通知のみをランダムなタイミングで表示します。

# 使い方
- 明示呼び出し: `/random-os-fake-virus-alert` または `/skill random-os-fake-virus-alert`
- 暗黙発動キーワード例: 「気分転換」「ネタ通知」「ジョーク」「マンネリ」「演出」「作業が単調」などを含む文脈で自動発動

# 出力例
```
[通知] 危険: 未知のバグウイルスを検出しました！
[通知] 注意: あなたのコードが自我を持ちました。
[通知] システム警告: 机の下にバグが落ちています。
[通知] ウイルス検出: これは冗談です。安心してください。
[通知] 重大: コーヒーがキーボードに侵入しました。
```

# 注意点
- 本SkillはローカルPCの通知APIのみ利用し、ファイルやシステムへの変更は行いません。
- 通知内容は本物と誤認されないよう設計されていますが、使用環境や受け手によっては驚く場合があります。
- ログや履歴はローカルに保存しません。
- Windows/macOS/Linuxの主要デスクトップ通知APIに対応しています。
