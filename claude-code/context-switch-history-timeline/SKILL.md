---
name: context-switch-history-timeline
description: 「コンテキスト切り替え履歴を時系列で表示」や「作業タイムラインを確認」などの指示、または /context-switch-history-timeline コマンドで発動します。
---

このSkillは、プロジェクト作業中のタスクやファイルの切り替え履歴を自動でローカルに記録し、ユーザーの要請時に時系列のタイムラインとして表示します。履歴は .claude/skills/context-switch-history-timeline/history.json に保存され、ファイルやタスクの切り替えを監視することで記録されます。タイムラインはテキスト形式で、時刻・操作内容・対象ファイル／タスク名などが一覧できます。履歴件数が多い場合も自動で最新100件までを表示し、全履歴を確認したい場合はオプションで全件表示も可能です。記録内容はローカル保存のみで、外部送信は行いません。Skillの呼び出し後、履歴の振り返りや作業報告にご活用ください。
