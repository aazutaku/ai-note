# 機能概要
このSkillは、開発現場や作業中に突如として“謎のOSパッチノート”をデスクトップ通知で表示します。通知内容は完全にでたらめで、「バグ：やる気が0になる問題を修正」「新機能：午後3時に眠気を自動注入」など、実際には役立たず、OS開発者も絶対に実装しないような珍妙なアップデート情報ばかりです。通知タイミングと内容は完全ランダムで、集中している時ほど思わず笑ってしまうような混乱とユーモアをもたらします。

# 使い方
このSkillは明示的な呼び出しは不要です。Antigravityが「パッチ」「アップデート」「バグ修正」などのキーワードを検出した場合や、semantic-match-onlyトリガーによって自動的に発動します。暗黙的な発動例としては、パッチノートの記述やOS関連の議論中などが挙げられます。

# 出力例
```
[通知] OSパッチノート v13.7.42
- バグ修正: 画面を見つめすぎると現実逃避する問題を修正
- 新機能: 3時のおやつ自動リマインダーを追加
- 既知の問題: 希望が時々消失することがあります
```
```
[通知] OSパッチノート v2.0.1-beta
- バグ修正: キーボードが突然歌い出す問題を修正
- 新機能: 画面の端にランダムな猫が出現します
```

# 注意点
- 通知は完全にジョーク目的であり、実際のOSや作業環境には一切影響を与えません。
- 通知内容やタイミングは毎回異なり、個別に制御できません。
- ローカルに通知履歴は保存されません。
- システム通知APIを利用するため、Linux/macOS/Windowsで動作しますが、通知の見え方は環境依存です。
