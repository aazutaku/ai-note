# 機能概要
本Skillは、作業中や進捗が停滞した際などに、唐突にデスクトップOS通知で『謎のOS偽モチベーション格言』をランダム表示します。格言は「進捗ゼロ、それもまた進化」「本日もバグに感謝せよ」「コードは寝かせて美味しくなる」など、やけに的外れだったり、無駄に深そうな内容です。進捗が出ない時や気分転換が必要な時、周囲の同僚も困惑するようなエンタメ通知を体験できます。

# 使い方
- 明示呼び出し例: `/skills menu` から本Skillを選択、または `$random-os-motivational-quote-notifier` をメンション
- 暗黙発動キーワード例: 「進捗ゼロ」「やる気が出ない」「バグ」「停滞」「気分転換」などの文脈を検知

# 出力例
```
[OS通知] 本日もバグに感謝せよ
[OS通知] 進捗ゼロ、それもまた進化
[OS通知] コードは寝かせて美味しくなる
[OS通知] OSは気まぐれ、あなたも気まぐれ
[OS通知] 仕様変更、それは運命のいたずら
```

# 注意点
- OS通知API（Python: plyer）を利用。通知はローカル端末でのみ表示され、リモート環境や一部Linux環境では動作保証外です。
- 格言はスクリプト内で管理されており、ユーザーによる追加・編集は要スクリプト修正。
- バックグラウンド常駐や定期通知機能はありません（明示呼び出し/トリガーのみ）。
