# 概要
random-excuse-notifierは、ターミナルでのコマンド実行時に、OSの通知領域へ根拠のない“言い訳”をランダム表示するユーモア特化型Skillです。実用性よりも演出・遊び心を重視しています。

# 公式ドキュメント抜粋
- Linux: notify-send (freedesktop.org)
- macOS: osascript (AppleScript)
- Windows: win10toast (PyPI)

# 利用例
- ターミナルで任意のコマンドを実行するたび、通知領域に「今日はPCの機嫌が悪いです」などのメッセージが表示される。
- `python random_excuse_notifier.py notify` で手動発火も可能。

# 注意点
- OS通知APIが利用できない環境では標準出力にフォールバックします。
- 言い訳履歴は最大10件までローカル保存し、同じメッセージの連続出現を抑制します。

# 設計方針
- ユーザー体験重視のため、通知内容は毎回ランダム選択かつ履歴排除で被りを低減。
- サブコマンドで一覧表示や履歴参照も可能とし、拡張性を確保。
- OSごとの通知APIを自動判別し、クロスプラットフォーム対応としています。
