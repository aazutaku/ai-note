# 概要
fake-os-zen-master-alertは、開発者の集中を絶妙に乱すための迷惑系エンタメSkillです。完全に実務無関係な禅問答をOS通知で表示し、作業の合間に“無意味な気づき”を与えることを目的としています。

# 公式ドキュメント抜粋
通知機能はPythonの外部ライブラリ「plyer」を利用しています。plyerはクロスプラットフォームでOS通知を送るためのAPIを提供します。
- plyer: https://pypi.org/project/plyer/
- Python subprocess: https://docs.python.org/3/library/subprocess.html

# 利用例
- 長時間のビルドやテスト待ち時間に、突然禅マスターが現れて通知を表示
- コマンド実行後のアイドル時に、思考を混乱させるメッセージ
- Skill設定から通知頻度やON/OFF切り替えが可能

# 注意点
- 通知内容は完全にジョークです。実務的なヒントや警告は一切含まれません。
- plyerの通知APIはOSによって挙動が異なる場合があります。通知が無効化されていると表示されません。

# 設計方針
- メッセージは完全に無意味かつ禅問答風であることを重視
- 実際のOS通知APIのみ使用し、架空のCLIや関数は使わない
- ユーザーが頻度やON/OFFを調整できるよう設計
