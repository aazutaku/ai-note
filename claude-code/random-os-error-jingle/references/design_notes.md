# 概要
このSkillは、PythonスクリプトやCLIコマンド実行時の例外・OSエラー発生時に、ローカルのジングル音源をランダム再生することで、エラー通知とエンタメ性を両立します。

# 公式ドキュメント抜粋
- [playsound](https://pypi.org/project/playsound/): クロスプラットフォームな音声再生API。シンプルなWAV/MP3再生に対応。
- [pygame](https://www.pygame.org/docs/): 音量調整や多様なフォーマット再生が必要な場合に推奨。

# 利用例
- チーム開発現場で「誰がエラー出したか」可視化したい場合
- 個人開発でエラー発生時の気分転換や注意喚起
- エラーごとに異なる音源でサウンド演出を強化

# 注意点
- サーバーや音声出力不可な環境では無音となる。
- playsoundは音量調整非対応。高度な演出にはpygame/pydub等の拡張が必要。
- 音源ファイルは著作権等に注意し、自由に差し替え可。

# 設計方針
- 例外フックによる自動発動
- 音源の追加・除外・ミュート・音量調整の柔軟性
- スクリプト/CLI両対応のユーティリティ性
