# 機能概要
このSkillは、作業中のウィンドウ（ターミナルやエディタ）をワンアクションで“謎のダミー画面”に即座に切り替えます。従来のボスキーのように業務画面を装うのではなく、毎回完全ランダムに生成される「意味不明なグラフ」「OS風の警告」「謎の進捗バー」など、逆に怪しまれるカオスな演出を実現します。仕事の緊張感を一瞬で混沌に変えたい時や、ジョーク用途に最適です。

# 使い方
Antigravityが「ボスキー」「画面隠し」「一時退避」「上司が来た」などのキーワードや、急なウィンドウ切り替えを要求する文脈を検知すると自動発動します。明示的な呼び出しコマンドはありません。Skill発動時は現在のウィンドウを隠し、ダミー画面を表示。エスケープキーや一定時間経過で元の画面に復帰します。

# 出力例
```
[=== OS WARNING ===]
System anomaly detected!
Error code: XJ-42A
Please contact your administrator.

[ランダムグラフ]
CPU Utilization: ████▒▒▒▒▒▒ 43%
Memory: ██████▒▒▒▒ 62%

Press ESC to return.
```

# 注意点
本SkillはローカルのX11/Waylandウィンドウ制御API（pyautogui, tkinter等）を利用します。リモート環境やCLI専用端末では一部機能が制限されます。ダミー画面の内容は毎回完全ランダム生成。Skill発動中の作業内容は保存されませんのでご注意ください。
