# 概要
このSkillは、作業中に突発的な“謎のOSアップデート進捗バー”をランダム表示し、ユーザーの緊張を和らげる演出を目的としています。実用性はなく、完全に遊び心に特化した設計です。

# 公式ドキュメント抜粋
Python標準ライブラリのrandom, argparse, time, sysのみを利用し、外部依存を排除しています。

# 利用例
- ターミナルで `python random_os_fake_update_progressbar.py run` を実行すると、ランダムな進捗バーが表示されます。
- `--task` で進捗内容を指定可能、`--fast` で高速表示も可能です。

# 注意点
- 実際のOSやシステムには一切影響を与えません。
- ログや履歴への保存は行いません。
- 進捗内容は毎回異なり、100%になっても何も起きません。

# 設計方針
冗長な演出を避け、1回の表示は2〜3秒以内で完結します。進捗内容は日本語で完全ランダムに生成され、ユーザーの作業を邪魔しない範囲で表示されます。
