# 機能概要
このSkillは、作業中や特定のキーワード（例：パッチ、アップデート、リリースノート）を検出した際、または明示的な呼び出し時に、全く役に立たない“謎のOSパッチノート”をデスクトップ通知としてランダムに表示します。通知内容は毎回異なり、バグ修正や新機能の説明がシュールかつ意味不明。開発現場に突如として笑いと混乱をもたらし、集中しすぎた空気を和ませることを目的としています。

# 使い方
- 明示呼び出し: `/skills random-os-fake-patch-notifier` またはメニューから選択
- 暗黙発動: 「パッチ」「アップデート」「リリースノート」などのキーワードを含む会話やコマンド実行時に自動発動
- 通知タイミングは完全ランダムで、連続発火や過剰な通知を防ぐため内部で間隔を制御しています

# 出力例
```
[通知] OS Patch Notes v13.4.2
- 新機能: 午後3時に自動で眠気を注入する機能を追加しました
- バグ修正: やる気が0になる問題を修正しました
- 既知の問題: たまに画面が唐突に青くなりますが仕様です

[通知] OS Patch Notes v42.0.1-beta
- 新機能: マウスカーソルがたまに踊りだすようになりました
- バグ修正: 画面の端に謎の猫が出現する問題を修正
```

# 注意点
- 通知はローカル環境の通知API（notify-send, win10toast等）を利用し、ファイルやシステム設定には一切変更を加えません
- 内容は完全なフィクションであり、実際のOSや作業には影響しません
- 長時間の連続実行や大量通知が不要な場合は、明示的に停止してください
