# 概要
commit-mood-forecastは、開発現場に遊び心を加えるために設計された、純粋な演出系Skillです。Gitコミット操作に合わせて、気分天気予報をランダムに表示することで、作業の合間に小さな笑いとリラックスを提供します。

# 公式ドキュメント抜粋
本SkillはGitのコミットフックやターミナル出力に依存しません。Skillエージェントがコミット操作や'/commit-mood-forecast'コマンドを検知した際に、スクリプトを呼び出して実行します。

# 利用例
- チームでの開発時に雰囲気を和ませたい場合
- 長時間の作業で気分転換したい場合
- コミットごとにちょっとした演出を加えたい場合

# 注意点
- 実用的な通知や履歴保存は行いません
- 出力は毎回ランダム生成され、再現性はありません
- 外部APIや追加パッケージに依存せず、Python標準ライブラリのみで動作します

# 設計方針
シンプルなCLI設計とし、Skill本体は100行以上のPythonスクリプトで実装。複数の気分天気パターンを用意し、毎回異なる出力で飽きさせない工夫をしています。
