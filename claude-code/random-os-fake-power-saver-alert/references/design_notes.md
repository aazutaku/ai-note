# 概要
このSkillは、作業中に突然“偽OS省エネ警告”をデスクトップ通知で表示し、ユーザーや周囲の集中を意図的に中断させるジョーク系演出ツールです。実際の電源管理やシステム状態には一切影響を与えません。

# 公式ドキュメント抜粋
通知表示にはPythonの`plyer`パッケージの`notification`モジュールを利用しています。plyerはクロスプラットフォームで、Windows/macOS/Linuxの標準通知APIをラップします。

# 利用例
- チームのリフレッシュタイムや雑談ネタに
- 長時間作業の合間に強制的な気分転換として
- オフィスやリモートワーク環境でのジョーク用途

# 注意点
- 通知内容は完全なフィクションであり、実際の作業やPCには影響しません。
- plyerの通知は環境によって表示形式や動作が異なる場合があります。
- スクリプトはローカル通知のみ対応。リモート通知やモバイル通知はサポート外です。

# 設計方針
- シンプルなCLI構成とし、run/list/summaryサブコマンドで柔軟に操作可能
- 通知内容は完全ランダムかつ毎回異なるよう十分なバリエーションを用意
- 実害ゼロ・ネタ度MAX・安全設計を徹底
