# 機能概要
terminal-mood-barometerは、ターミナルでコマンドを実行するたびに、完全ランダムな「やる気指数」とコメントを表示する演出系Skillです。真面目な作業中にも、気分指数（例：やる気57）と一言コメント（例：「今日は雨、でもコードは晴れ」）が毎回変わって表示されます。これにより、単調なコマンドライン作業にユーモアや遊び心を加え、作業者の気分を和らげたり、ちょっとした話題を提供します。

# 使い方
このSkillは明示的な呼び出しは不要で、Antigravityがコマンド実行・ターミナル操作・通知・演出などのキーワードを検知した際に自動発動します。コマンド実行時に自動的に気分指数とコメントが表示されるため、ユーザーは特別な操作を意識する必要がありません。

# 出力例
```
$ ls -l
やる気 92：今日もコーディング日和！
(コマンドの出力...)

$ git status
やる気 13：カフェイン補給推奨。
(コマンドの出力...)

$ python main.py
やる気 67：集中力アップ中！
(コマンドの出力...)
```

# 注意点
- 気分指数やコメントは完全ランダム生成であり、実際の作業状況や心理状態とは無関係です。
- ローカルに履歴や個人情報は保存しません。
- コマンド実行の妨げにならないよう、通知は1行で簡潔に表示されます。
- 除外パスや特定のコマンド（例：パイプ処理やバッチ実行）では演出が抑制される場合があります。
