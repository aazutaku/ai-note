# 機能概要
fake-os-zen-master-alertは、作業中やコマンド実行の合間に突如として“偽OSの禅マスター”が現れ、完全に実務とは無関係な禅問答や悟りの極意をOSデスクトップ通知で投げかけるSkillです。通知内容は「バグとは何かを問う前に、己を問え」「コンパイルに失敗したとき、木魚を叩け」など、思わず考え込んでしまう迷惑系・エンタメ系の内容が中心。集中を乱し、時に深遠な（？）気づきを与えることで、日々の開発作業に一風変わった刺激をもたらします。

# 使い方
このSkillは明示的な呼び出しを必要とせず、Antigravityの「semantic-match-only」トリガーで自動発動します。例えば、ビルドやテストの完了、長時間の無操作、ファイル保存、コマンド入力後の待機時など、作業の合間に適宜通知が表示されます。明示呼び出しはできませんが、Skill設定から通知頻度やON/OFFの切り替えが可能です。

# 出力例
```
[通知] 偽OS禅マスター: バグとは何かを問う前に、己を問え
[通知] 偽OS禅マスター: 今日の悟り：if文を捨てよ、道は開ける
[通知] 偽OS禅マスター: コンパイルに失敗したとき、木魚を叩け
[通知] 偽OS禅マスター: 変数名に迷うとき、まず呼吸せよ
[通知] 偽OS禅マスター: echoの裏にこそ真実がある
```

# 注意点
- 通知内容は完全に実務無関係です。実用的なアドバイスは一切含まれません。
- OSの通知機能を利用するため、通知が無効化されていると表示されません。
- 通知はローカルには保存されません。
- 頻度やON/OFFはSkill設定で調整可能です。
