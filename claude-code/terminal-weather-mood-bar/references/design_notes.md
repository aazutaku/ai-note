# 概要
このSkillは、開発現場の単調さを和らげるために、気分やテンションを天気情報風に変換してターミナルに表示する演出ツールです。Gitのコミット頻度やランダム値をもとに、気象用語や温度を生成します。

# 公式ドキュメント抜粋
Gitのコミット取得にはGitPythonを利用しています。詳細は[GitPython公式](https://gitpython.readthedocs.io/en/stable/)。

# 利用例
- 毎日の作業開始時に`python weather_mood_bar.py show`で気分天気を確認
- コミット直後にプロンプトへ自動出力（シェル連携）
- Slack等への転送も可能

# 注意点
- 気分・天気は完全なジョークであり、実際の心理分析や天気予報ではありません。
- Gitリポジトリが存在しない場合はランダム値で出力します。

# 設計方針
- 気分やテンションを気象情報にマッピングすることで、日常の開発作業に“意味不明な癒し”を提供します。表示は短く、邪魔にならないよう工夫しています。
