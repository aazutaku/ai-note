# 機能概要
このSkillは、ターミナルやエディタの起動時に毎回“偽のOS起動画面”をテキストアニメーションとして表示します。表示される内容は毎回異なり、実在しないOS名（例: Windows 11.5, Antigravity 3000, QuantumOS X）や、ありえない進捗メッセージ（例: ビットを配置中, AIを目覚めさせています, 宇宙的セキュリティを検証中）など、完全に架空の未来感と謎の緊張感を演出します。作業の前に一瞬だけ非日常を楽しみたい、実用性ゼロの“無駄演出”を求める方に最適です。

# 使い方
このSkillはターミナルやエディタのセッション開始時（新しいウィンドウ/タブのオープンや再起動時）に自動で発動します。明示的な呼び出しは不要です。暗黙の発動キーワード: "起動", "スタートアップ", "セッション開始", "新規ウィンドウ" など。

# 出力例
```
[Antigravity 3000 Ultimate Edition 起動中...]
■■■■□□□□□□ (38%)
AIを目覚めさせています...
量子ビットの再配置中...
全銀河ネットワークとの同期を確認...
起動完了。ようこそ、未来へ。
```

```
[Codex Limited Edition OS 起動準備]
■□□□□□□□□□ (9%)
ビットを配置中...
自己修復モジュールを展開中...
意味不明なプロセスを初期化中...
起動に成功しました。
```

# 注意点
- 完全な演出目的であり、実用的な情報やシステム操作は一切行いません。
- 出力内容は毎回ランダム生成され、現実のOSやシステムとは無関係です。
- ローカルにログ等は保存しません。
- 除外パス: システム管理系や本番運用環境では自動発動しないよう設計されています。
