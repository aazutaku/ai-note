# 概要
このSkillは、ユーザーの作業中に突然現れる“謎の再起動進捗バー”で、気分転換やジョークを提供するために設計されています。実際のシステム操作は一切行わず、心理的なリセット感を演出します。

# 公式ドキュメント抜粋
Tkinterを利用してクロスプラットフォームなGUI進捗バーを実装。Python標準ライブラリのみを活用し、外部依存を最小限に抑えています。

# 利用例
- 長時間作業後のリフレッシュ演出
- 「再起動」や「リセット」関連の話題時に自動発動
- オンライン会議やペアプロ時のジョーク

# 注意点
本Skillは進捗バーの表示のみを行い、システムやファイルには一切影響を与えません。Tkinterが利用できる環境でのみ動作します。

# 設計方針
ユーザー体験を損なわず、笑いと驚きの演出を重視。進捗メッセージは毎回ランダム生成され、繰り返しでも飽きない工夫をしています。
