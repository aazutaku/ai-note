# 機能概要
このSkillは、あなたがターミナルやエディタでコマンドを実行するたびに、OS画面端やメニューバーに「やればできる度」「バグ耐性ゲージ」「今日のやる気残量」など、まったく意味不明かつランダムな自己肯定感パラメータを持つ“ハイプバー”を表示します。実用性や正確性は一切なく、ただあなたの気分を謎に盛り上げる演出を提供します。進捗や成果に関係なく、毎回異なる内容で自己肯定感をカオスに演出します。

# 使い方
このSkillは明示的な呼び出しは不要です。Antigravityが「コマンド実行」「ファイル保存」「新規ターミナル起動」などのアクションを検知すると自動で発動します。暗黙発動キーワード例: "run", "save", "build", "commit", "open terminal" など、ほぼ全ての操作が対象です。

# 出力例
```
[OS Motivational Hypebar]
やればできる度: 87%
バグ耐性ゲージ: 41%
今日のやる気残量: 102%
コードカオス指数: 73%
謎の達成感: 99%
```

# 注意点
- 表示される数値やパラメータ名は完全にランダムで、実際の状況や進捗とは無関係です。
- 実用的な通知や進捗管理には利用できません。
- ローカル保存や履歴管理は行いません。
- 除外パスや特定のファイルタイプによる無効化はありません。
