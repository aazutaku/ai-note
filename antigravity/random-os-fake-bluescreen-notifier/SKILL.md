# 機能概要
このSkillは、デスクトップ上に本物そっくりだが全く害のない“ブルースクリーン風”の通知をランダムなネタメッセージで表示します。例えば「STOP CODE: 眠気の暴走」や「エラー: コーヒーブレイク不足」など、思わずクスッとする内容が毎回変化します。開発現場やリモートワーク中の雰囲気を和ませたり、ちょっとした気分転換を提供します。

# 使い方
このSkillは明示的な呼び出しは不要で、"ブルースクリーン"、"エラー通知"、"OSクラッシュ"、"ジョーク通知"などのキーワードが会話やリクエストに含まれると自動で発動します。例えば「面白いブルースクリーン通知を出して」や「今日は眠気がひどいので警告してほしい」などの文脈で動作します。

# 出力例
```text
[FAKE BLUE SCREEN]
STOP CODE: 眠気の暴走
原因: コーヒーブレイク不足
アクション: 立ち上がってストレッチしてください

[FAKE BLUE SCREEN]
STOP CODE: キーボードに猫が乗りました
原因: 入力デバイスが占拠されています
アクション: 猫をなでてください
```

# 注意点
- 実際のシステムやファイルには一切影響を与えません。
- 通知はデスクトップ上に一時的に表示され、ログや履歴は残しません。
- 本物のブルースクリーンと誤認されないよう、明確に“FAKE”や“ジョーク”である旨を表示します。
- Linux/macOS/Windowsの主要なデスクトップ通知APIに対応していますが、サーバ環境やGUIのない端末では動作しません。
