# 概要
context-switch-notes Skillは、複数のプロジェクトやタスクを並行して扱う現場で「直前の流れ」を記録・復元することを目的としています。作業の切り替え時に自動で要約・TODOを保存し、再開時に即座に表示することで、思考の断絶を防ぎます。

# 公式ドキュメント抜粋
本Skillは https://ai-note.tech/docs/context-switch-notes に準拠し、semantic-match-onlyトリガーで暗黙的に発動します。保存先はローカルの .context_notes ディレクトリです。

# 利用例
- プログラミング中にバグ修正と新機能開発を交互に進める場合
- 複数の顧客プロジェクトを切り替える場合
- 長期間放置したタスクの再開時

# 注意点
- メモ内容はユーザーが明示的に編集・削除可能です。
- 機密情報の自動記録は行いませんが、念のため内容確認を推奨します。

# 設計方針
- 冗長な記録を避けるため、直前と同じ内容のメモは自動でスキップします。
- データ損失防止のため、定期的なバックアップ機能を提供しています。
