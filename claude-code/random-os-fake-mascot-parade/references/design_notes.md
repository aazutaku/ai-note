# 概要
このSkillは、作業中に突如現れるランダムなOS風マスコット通知によって、ユーザーの気分転換やネタ的な驚きを演出することを目的としています。実用性はなく、完全にジョーク・エンタメ用途です。

# 公式ドキュメント抜粋
- Python random: https://docs.python.org/3/library/random.html
- plyer通知API: https://plyer.readthedocs.io/en/latest/

# 利用例
- 長時間の作業やコーディング中に、突然「カーネルくん」や「シェルパンダ」などのキャラクターが応援や謎の警告を通知。
- `/random-os-fake-mascot-parade` コマンドで即時発動。

# 注意点
- 通知はローカル環境でのみ表示され、履歴は保存されません。
- plyerが未導入の場合は標準出力にフォールバックします。
- 実在のOSやキャラクターとは一切関係ありません。

# 設計方針
- 通知内容・キャラ名・セリフは毎回完全ランダム生成。
- 通知頻度は5～20分間隔で自動調整し、うるさすぎない設計。
- スクリプトはCLIサブコマンド形式で柔軟に呼び出せます。
