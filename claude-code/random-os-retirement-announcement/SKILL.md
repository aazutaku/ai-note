# 機能概要
このSkillは、作業中に突如として“謎のOS退職アナウンス”をデスクトップ通知で表示します。通知内容は完全ランダムで、架空のOS内キャラクター（例：カーネル部長、ファイルシステムさん、メモリ課長など）が、なぜか勝手に退職や転職を宣言します。全く意味はありませんが、不条理な演出を通じて一瞬だけセンチメンタルな気分を味わえます。通知は1時間に1回程度までの頻度で発火し、作業の合間にカオスな癒やしを提供します。

# 使い方
- 明示呼び出し例: `/random-os-retirement-announcement`
- 暗黙発動キーワード例: 「退職」「勇退」「転職」「お知らせ」「通知」などが会話やコマンド履歴に現れた際、自動発動します。
- 通知はランダムなタイミングで自動表示されるため、特別な操作は不要です。

# 出力例
```
通知: 本日をもってカーネル部長が勇退します。
重要: ファイルシステムさんが第二の人生へ旅立ちます。
急報: メモリ課長が早期リタイアを決意しました。
速報: プロセス係長が退職し、シェル芸人を目指します。
連絡: ネットワーク主任が新天地へ転職しました。
```

# 注意点
- 通知内容は完全に架空で、実際のシステムやデータには影響しません。
- ローカルに通知履歴は保存されません。
- 1時間に1回以上は通知が発火しません（スパム防止）。
- デスクトップ通知API（notify-send等）を利用しているため、環境によっては通知が表示されない場合があります。
