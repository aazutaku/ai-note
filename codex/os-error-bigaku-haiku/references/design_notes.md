# 概要
このSkillは、CLIで発生するエラーを和風俳句に変換し、エラー内容ごとに異なる雰囲気を持たせることで、開発者の心理的負担を軽減し、笑いや癒しを提供することを目的としています。

# 公式ドキュメント抜粋
- Python subprocess: https://docs.python.org/3/library/subprocess.html
- argparse: https://docs.python.org/3/library/argparse.html

# 利用例
例えば `os-error-bigaku-haiku run cat missing.txt` のようにコマンドをラップ実行し、エラーが発生した場合は即座に俳句化されたメッセージが表示されます。また、`haiku-log` サブコマンドで過去の俳句履歴を一覧できます。

# 注意点
- 俳句生成はテンプレートベースですが、今後AI生成やカスタマイズも検討可能です。
- すべてのエラーに完全対応するわけではありません。未知のエラーは汎用俳句で対応します。
- 履歴はユーザーホームディレクトリにJSON形式で保存されます。

# 設計方針
- シンプルなPythonスクリプトで、CLIラッパーとして機能。
- エラー分類はキーワードマッチングで実装。
- 履歴管理は最大100件で自動ローテーション。
