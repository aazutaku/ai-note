# 機能概要
このSkillは、作業中のユーザーに対し、まるで本物のOSセキュリティスキャンが突然始まったかのようなフェイク通知をデスクトップに表示します。通知内容は「未知のやる気ウイルスを発見」「マウスの動きが怪しい」など完全なジョークで構成され、進捗バーもランダムに増加。真面目な空気を一瞬で和らげ、職場やリモートワーク環境での息抜きや話題作りに最適です。

# 使い方
- 明示呼び出し: `/skills menu` で一覧から選択、または `/random-os-fake-security-scan` コマンドを直接実行
- 暗黙発動: 「scan」「virus」「セキュリティ」「random」「進捗」などの単語が会話やコマンドに含まれると自動発動します

# 出力例
```
[セキュリティスキャン開始] 進捗: 7%
警告: 未知のやる気ウイルスを検出しました
[セキュリティスキャン進行中] 進捗: 33%
警告: マウスの動きが怪しいです
[セキュリティスキャン進行中] 進捗: 61%
警告: あなたの進捗は安全です
[セキュリティスキャン完了] 進捗: 100%
結果: 重大な問題は見つかりませんでした
```

# 注意点
- 通知はローカル環境でのみ表示されます（サーバー・リモート環境では不可）
- 本Skillは完全なジョーク用途であり、実際のセキュリティ診断は行いません
- 通知内容や進捗は毎回ランダム生成されます
- 履歴やログはローカル保存されません
