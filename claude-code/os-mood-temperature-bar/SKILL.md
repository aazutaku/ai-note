# 機能概要
このSkillは、あなたの作業中の“気分温度”をOSのステータスバーやメニューバー風に表示します。コマンド実行やエディタ操作のたび、気分が「絶好調39度」「冷え冷え16度」「炎上中42.5度」など、完全ランダムな温度とコメントで演出されます。仕事の空気が一瞬ゆるみ、チーム内の会話やリフレッシュのきっかけにもなります。

# 使い方
- 明示的呼び出し: `/os-mood-temperature-bar` を入力
- 暗黙的発動: 「気分」「温度」「テンション」「やる気」などの単語を含む操作や会話の直後
- CLIサブコマンド: `python os_mood_temperature_bar.py log` で温度記録、`list` で履歴表示、`summary` で統計も可能

# 出力例
```
[OS-Mood-Bar] 気分温度: 42.5度 炎上中！
[OS-Mood-Bar] 気分温度: 16.0度 冷え冷え…
[OS-Mood-Bar] 気分温度: 39.0度 絶好調！
[OS-Mood-Bar] 気分温度: 27.3度 まあまあ
[OS-Mood-Bar] 気分温度: 21.8度 低気圧注意
```

# 注意点
- 温度とコメントは完全ランダムで、実際の気分や体調とは無関係です
- ローカル環境に記録ファイル（.os_mood_log.json）が生成されます
- メニューバーやターミナル出力を模倣しますが、GUI統合はありません
- 業務用途での真面目な利用には不向きです
