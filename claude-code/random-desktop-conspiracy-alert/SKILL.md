# 機能概要
random-desktop-conspiracy-alertは、あなたの開発現場に突如として謎の陰謀論アラートをデスクトップ通知で表示するSkillです。Wi-Fiの波動や宇宙人の監視、バグの覚醒など、根拠のない怪しさ満点の通知を完全ランダムで生成し、作業中の気分転換やチームの笑いを誘発します。集中力が途切れがちな長時間作業や、雑談のネタが欲しいときに最適です。

# 使い方
- 明示呼び出し: `/random-desktop-conspiracy-alert` を実行すると即座にアラートが表示されます。
- 暗黙発動: 「集中力」「バグ」「監視」「Wi-Fi」「宇宙人」などのキーワードが会話やコード内に現れると自動で発動します。

# 出力例
```
[通知] Wi-Fiの波動があなたの思考を妨害しています。
[通知] コードレビューは宇宙人の監視下にあります。
[通知] 本日15時、全てのバグが覚醒します。
[通知] あなたのキーボードは秘密組織により記録されています。
[通知] 開発環境が量子もつれ状態に突入しました。
```

# 注意点
- 通知はローカルマシンのデスクトップ通知機能を利用します。
- 頻度は1時間に最大3回まで自動発動（明示呼び出しは制限なし）。
- 通知内容は完全にフィクションであり、実際のセキュリティや業務には影響しません。
- Linux/macOS/Windowsで動作確認済み。
