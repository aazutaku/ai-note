# 概要
このSkillは、作業開始時の“謎のOS起動音”による演出を目的とし、ユーザーのローカル環境でwav/mp3音源をランダム再生します。Python標準と実績ある音声再生ライブラリ（playsound/pydub）を用い、依存関係を最小限に抑えつつ多環境対応を意識しています。

# 公式ドキュメント抜粋
- [playsound](https://github.com/TaylorSMarks/playsound): シンプルなクロスプラットフォーム音声再生
- [pydub](https://github.com/jiaaro/pydub): 多機能な音声処理と再生

# 利用例
- ターミナル/エディタ起動時の気分転換やネタ演出
- チーム開発での“OSごっこ”やアイスブレイク
- サウンド差し替えによるカスタム演出

# 注意点
- サウンドはローカル保存必須、ネット経由再生は非対応
- 音量・再生デバイスはOS依存
- Python 3.7+と必要なライブラリのインストールが前提

# 設計方針
- シンプルなコマンド/自動発動両対応
- サウンド候補の一覧・概要表示機能
- 音源追加・差し替えの容易さ
- エラー時の明示的なガイド
