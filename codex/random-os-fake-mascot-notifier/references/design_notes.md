# 概要
本Skillは、作業中のユーザーに“全く役に立たない”架空のOSマスコットからの通知を届け、現実逃避や気分転換を促すことを目的としています。Pythonのplyerライブラリを利用し、クロスプラットフォームでデスクトップ通知を実現しています。

# 公式ドキュメント抜粋
- plyer: https://github.com/kivy/plyer
- argparse: https://docs.python.org/ja/3/library/argparse.html

# 利用例
- 長時間のコーディングや単調作業の合間に、突如現れるマスコット通知で笑いと休憩のきっかけを提供します。
- /skills menu から明示呼び出し、または「通知」「マスコット」などの会話キーワードで自動発動します。

# 注意点
- 通知は履歴としてメモリ上で保持されますが、ファイル保存や永続化は行いません。
- plyerは各OSの通知APIをラップしているため、環境によっては通知が表示されない場合があります。

# 設計方針
- マスコットやメッセージは完全オリジナルで、毎回ランダム生成。
- 通知頻度は10～40分の範囲でランダムに設定し、作業の邪魔にならないよう配慮。
- CLIサブコマンドで即時通知・自動通知・履歴表示・サマリーなど柔軟に利用可能としています。
