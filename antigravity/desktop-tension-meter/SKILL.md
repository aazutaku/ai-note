# 機能概要
このSkillは、作業中のユーザーに対して、デスクトップ画面端に「現在の緊張度」をテロップとして表示します。緊張度は完全なランダム値、またはコミット数やタイピング速度といった“根拠になりそうでならない”情報から算出され、「平常心」「やや焦り」「極度緊張」など謎めいたラベルを演出します。これにより、背後に上司がいないのに謎のプレッシャーを感じるという、無意味だけど気になる演出体験を提供します。通知・演出・OS連携カテゴリに属し、職場や自宅のPCでの作業時に、ちょっとした刺激や話題作りとして活用できます。

# 使い方
このSkillは常時バックグラウンドで動作し、明示的な呼び出しは不要です。"緊張" "焦り" "集中" "commit" "タイピング"などのキーワードや、作業アクティビティの検知時に自動で発動します。トリガーはsemantic-match-onlyで、ユーザーの操作状況に応じてランダムなタイミングでテロップが表示されます。

# 出力例
```
[デスクトップ右上]
現在の緊張度: 極度緊張 (推定値: 87)

[デスクトップ左下]
現在の緊張度: 平常心 (推定値: 12)

[デスクトップ右下]
現在の緊張度: やや焦り (推定値: 54)
```

# 注意点
本Skillは純粋な演出目的であり、実際の緊張度や生体情報は取得しません。データはローカルメモリのみで管理され、記録・送信は行いません。通知が鬱陶しい場合はスクリプトを停止してください。強制終了や誤作動によるデータ損失は発生しませんが、画面表示が一時的に残る場合があります。
