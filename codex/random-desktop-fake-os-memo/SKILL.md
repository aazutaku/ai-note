# 機能概要
このSkillは、ユーザーの作業中や任意のタイミングで、まるで謎のOSからの警告やメモのような意味不明な通知をデスクトップに表示します。通知内容は完全ランダムで「冷蔵庫にプリンあり」「バグは空から降ってくる」「謎の引き出しを開けるな」など、実用性ゼロの怪文書ばかり。集中している時ほど思わず二度見してしまう、シュールな混乱と遊び心をデスクトップに演出します。

# 使い方
- 明示呼び出し例: `/skills menu` で一覧から選択、または `$random-desktop-fake-os-memo` と入力して即時発動
- 暗黙発動キーワード例: 「通知」「演出」「デスクトップ」「遊び」「意味不明」「OSメモ」などを含むタスクや会話中に自動発動

# 出力例
```terminal
[通知] 謎のOSメモ: 冷蔵庫にプリンあり
[通知] 謎のOSメモ: 上司が見ている
[通知] 謎のOSメモ: バグは空から降ってくる
[通知] 謎のOSメモ: 謎の引き出しを開けるな
[通知] 謎のOSメモ: 今日は何曜日でもない
[通知] 謎のOSメモ: そのファイルはもう存在しない
```

# 注意点
- 通知内容は毎回完全ランダムで、実用性や意味は一切ありません。
- 通知履歴や内容はローカル保存されません。
- 通知APIのみを利用し、システム設定やファイルには影響しません。
- OSごとに通知の見え方が異なる場合があります。
