# 機能概要
terminal-samurai-weathercasterは、ターミナルを開くたびに“侍”が現れ、その日の作業気分を天気予報風に斬新なコメントで伝えるスキルです。現実の天気やAPIには一切依存せず、完全にランダムな天気・侍コメントを生成します。作業開始の緊張感やマンネリを打破し、日々のコーディングにユーモアと意外性をもたらします。

# 使い方
- 明示呼び出し: `/terminal-samurai-weathercaster` をターミナルで実行
- 暗黙発動: ターミナル起動時や「天気」「侍」「気分」「始業」などのキーワードを含む文脈で自動発動

# 出力例
```
【侍天気予報】
本日快晴、コードも冴え渡るでござる！
---
【侍天気予報】
曇り空、集中力もやや薄れ申す。
---
【侍天気予報】
雷注意報、バグの嵐到来か？油断召されるな！
---
【侍天気予報】
春の陽気、心穏やかに進軍でござる。
---
【侍天気予報】
嵐の予感、仕様変更に備えよ！
```

# 注意点
- 現実の天気やAPIには一切アクセスしません
- 侍コメント・天気は完全ランダムで、同じ内容が続く場合もあります
- ローカルに履歴等は保存しません
- シェルのプロファイル等に組み込むことで自動発動可能です
