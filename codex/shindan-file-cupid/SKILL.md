# 機能概要
shindan-file-cupidは、プロジェクト内のファイル名リストをもとに、なぜか『相性のいい同僚ファイル』を診断・推薦するジョーク系Skillです。ファイルの内容や本当の関連性には一切基づかず、独自の謎ロジックやランダム要素でペアと診断理由を生成します。実用性はゼロですが、チーム内のアイスブレイクや会話のネタ、ちょっとした息抜きに最適です。

# 使い方
明示的な呼び出しは「/skills menu」や「shindan-file-cupidを使って」などのコマンド、または「ファイルの相性診断」「おすすめペアを教えて」などのキーワードを含む発話で発動します。暗黙発動例: 「main.pyと相性のいいファイルは？」「このプロジェクトで仲良しなファイルペアを教えて」

# 出力例
```
[shindan-file-cupid]
main.py ❤️ settings.yaml
理由: どちらもプロジェクトの心臓部。お互いがいないと動きません。
README.md 💔 LICENSE
理由: 価値観が合わず、会話が盛り上がらないタイプ。
data.csv 💘 analysis.ipynb
理由: データを渡してもらうのが日課。まさに理想の関係です。
```

# 注意点
本Skillはファイル内容や本番環境には一切影響しません。診断は完全なジョークであり、実務的な意味はありません。除外パスや隠しファイルも診断対象となる場合があります。ローカル保存や履歴管理は行いません。
