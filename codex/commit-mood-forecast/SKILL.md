# 機能概要
commit-mood-forecastは、Gitのコミット操作時に、ターミナルへ“本日のあなたの作業気分天気予報”をランダムで表示するSkillです。天気風のアイコンと共に「快晴：今日は絶好調」「曇り：ちょっと迷い気味」「雷：やる気ゼロ」など、毎回異なるコメントが出現。開発フローに遊び心を添え、チームや個人の気分転換・雰囲気づくりに役立ちます。実用性はありませんが、真面目な作業にちょっとした脱力感をプラスします。

# 使い方
- 明示呼び出し例: `/skills menu` から `commit-mood-forecast` を選択、または `$ commit-mood-forecast` を直接実行。
- 暗黙発動: Gitで`commit`コマンド実行時（例: `git commit -m "fix bug"`）に自動で発動。

# 出力例
```
[commit-mood-forecast]
本日の作業気分天気予報：
  ☀️ 快晴！今日は絶好調。どんどん進めましょう。
---
[commit-mood-forecast]
本日の作業気分天気予報：
  ☁️ 曇り。ちょっと迷い気味。無理せず進めてください。
---
[commit-mood-forecast]
本日の作業気分天気予報：
  ⚡ 雷。やる気ゼロ…コーヒーでも飲みましょう。
```

# 注意点
- 出力はターミナル標準出力のみ。ファイルやリモート通知は行いません。
- メッセージは毎回ランダム、内容は短く抑えています。
- ローカル環境で即利用可能、追加の依存パッケージ不要。
- 実用的な通知や履歴保存は行いません。
