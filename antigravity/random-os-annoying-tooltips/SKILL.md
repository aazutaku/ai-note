# 機能概要
本Skillは、ユーザーの作業中に突如として“謎のOS風うざいツールチップ”をデスクトップ通知として表示します。内容は「CapsLockが押されているかもしれません」「アップデートしませんか？」「あなたの入力速度、気づいてますか？」など、懐かしの“助けにならないヘルプ”を現代に再現。職場や作業現場に、ささやかな笑いと軽いイライラを提供します。通知は主要OS（Windows/macOS/Linux）で安全に動作し、頻度や内容も適度に調整されています。

# 使い方
このSkillは明示的な呼び出しは不要で、Antigravityのsemantic-matchトリガーによって自動発動します。たとえば「通知」「OS風」「作業妨害」「ヘルプが邪魔」などのキーワードが会話や作業ログに現れると、一定確率でツールチップ通知が表示されます。ユーザーが長時間操作していない場合や、集中しているタイミングでも発動することがあります。

# 出力例
```
[通知] あなたのCapsLockキー、今押されていますか？
[通知] システムアップデートの準備ができています。今すぐ再起動しますか？
[通知] 入力速度が平均より遅いようです。タイピング練習をおすすめします。
[通知] 画面の明るさが最適化されていません。自動調整しますか？
[通知] しばらく休憩していません。ストレッチの時間です。
```

# 注意点
- 本Skillはローカル通知のみを表示し、ユーザーのデータや設定を変更することはありません。
- 通知頻度は自動調整され、過剰な連発は抑制されます。
- 一部のLinux環境では通知APIの事前インストールが必要な場合があります。
- ログや履歴はローカルには保存されません。
