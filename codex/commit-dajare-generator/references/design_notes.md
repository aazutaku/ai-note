# 概要
commit-dajare-generatorは、Gitのステージ済みファイル名や差分内容からキーワードを抽出し、それに基づいた日本語ダジャレコミットメッセージを自動生成するSkillです。エンジニアのチームコミュニケーション活性化や開発現場の雰囲気改善を狙っています。

# 公式ドキュメント抜粋
- Gitコミットメッセージのベストプラクティス: https://www.git-scm.com/book/ja/v2/Git-%E3%81%AE%E5%9F%BA%E6%9C%AC-%E3%82%B3%E3%83%9F%E3%83%83%E3%83%88%E3%83%A1%E3%83%83%E3%82%BB%E3%83%BC%E3%82%B8

# 利用例
- 通常のコミット時に自動でダジャレメッセージが提案される
- `/skills menu` から明示的に呼び出し、生成内容を確認・編集可能

# 注意点
- ダジャレの品質はファイル名・差分内容に依存し、必ずしも秀逸なものばかりではありません
- 英語ファイル名への対応は限定的です
- 生成メッセージはローカルでのみ利用され、外部送信はありません

# 設計方針
- 実在するGitコマンドのみを利用し、外部APIや架空関数は不使用
- メンテナンス性を重視し、パターン追加・カスタマイズも容易な設計
- コミットメッセージの自動生成は手動編集を妨げないよう配慮
