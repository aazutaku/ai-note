# 概要
このSkillは、Gitコミット時の“儀式”に神話的な遊び心を加えることを目的としています。毎回異なる神獣と運勢コメントを表示することで、飽きずに楽しめる設計です。

# 公式ドキュメント抜粋
- Git公式: https://git-scm.com/doc
- Python argparse: https://docs.python.org/ja/3/library/argparse.html

# 利用例
- チーム開発のコミット時に自動実行し、ターミナルに神獣が登場
- `/skills menu`や明示コマンドで履歴や出現回数を確認

# 注意点
- 運勢や神獣は完全にランダムで、実際の作業結果や運には影響しません。
- 履歴はローカルJSONファイルに保存され、他のユーザーとは共有されません。

# 設計方針
- 神獣・コメントの組み合わせは容易に拡張可能
- コマンドラインから `log`（通常発動）、`list`（履歴表示）、`summary`（統計）の3サブコマンドを提供
- 外部APIやネットワーク通信は一切行わず、ローカルで完結
