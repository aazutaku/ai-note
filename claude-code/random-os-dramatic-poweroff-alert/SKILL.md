# 機能概要
このSkillは、あなたの作業中や長時間の集中状態に突如「OSが壮大な理由で電源を切る」といった、意味不明かつ無責任なデスクトップ通知を爆誕させます。通知内容は毎回ランダムで、例えば「全プロセスよ、さらば…」や「宇宙エネルギー調整のためシャットダウンします」など、現実には何も起こらない“壮大な嘘”を演出します。緊張感を一撃で破壊し、遊び心と笑いを提供します。

# 使い方
- 明示呼び出し例: `/random-os-dramatic-poweroff-alert`
- 暗黙発動キーワード例: 「シャットダウン」「電源」「再起動」「OSクラッシュ」などの単語が会話やコード内に出現した場合、自動で通知が発動します。
- どのOS環境でもPythonスクリプト単体で動作し、追加インストール不要です。

# 出力例
```
[通知] システムはあと3分で壮絶にシャットダウンします（理由：宇宙エネルギー調整のため）
[通知] 全プロセスよ、さらば…
[通知] 管理者の気まぐれにより、OSは間もなく伝説となります。
[通知] あなたの作業は今、歴史的電源オフの瞬間を迎えます。
[通知] システムはブラックホールに吸い込まれます。全員退避！
```

# 注意点
- 実際のシャットダウンや電源オフは一切行われません。
- 通知内容は毎回ランダム生成され、現実のシステム操作には影響しません。
- 通知の履歴や内容はローカルに保存されません。
- 一部Linux環境では通知表示にlibnotifyが必要な場合があります。
