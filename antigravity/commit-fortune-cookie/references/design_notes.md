# 概要
commit-fortune-cookieは、コミットごとに開発者へユーモラスな運勢とアドバイスを表示することで、日々のコーディングにちょっとした楽しさを提供するSkillです。

# 公式ドキュメント抜粋
本SkillはGitのコミットフックやAntigravityのトリガー機構と連携し、コミット時にのみ発動します。公式Gitドキュメント: https://git-scm.com/docs/git-commit

# 利用例
- チーム開発時に、コミットごとに運勢が表示され、会話のきっかけやリラックス効果を生みます。
- 個人開発でも、日々の作業にちょっとしたスパイスを加えられます。

# 注意点
- 運勢やアドバイスはランダム生成であり、実際の開発判断には利用しないでください。
- ログはローカルファイル(.fortune_cookie_log)に記録されますが、容量やセキュリティには留意してください。

# 設計方針
- ワークフローを邪魔しないよう、出力はターミナル表示のみに限定。
- commitメッセージや時刻をシード値に用い、毎回異なる結果を保証。
- ログ機能で過去の運勢を振り返ることも可能です。
