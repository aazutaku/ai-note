# 概要
このSkillは、開発現場に突如として現れる“謎のOSパッチノート”通知で、集中しすぎた空気を和ませる目的で設計されています。通知内容は完全なフィクションで、実際のOSや作業には一切影響を与えません。

# 公式ドキュメント抜粋
- Linux: [notify2](https://pypi.org/project/notify2/)
- Windows: [win10toast](https://pypi.org/project/win10toast/)

# 利用例
- チーム開発中の雑談ネタやリフレッシュタイムに
- 「パッチ」「アップデート」などの単語が会話に登場した際の自動発火
- 明示的に `/skills random-os-fake-patch-notifier` で手動発火

# 注意点
- 通知はローカルAPIのみ利用し、ファイルやシステム設定には一切変更を加えません
- 長時間の連続実行時は、通知頻度に注意してください

# 設計方針
- 完全なランダム性と多様なネタ生成を重視
- OSや開発環境に影響を与えない安全設計
- CLIサブコマンドで柔軟な使い分けを実現
