# 概要
commit-mood-forecastは、開発現場にちょっとした遊び心を持ち込むためのSkillです。実用的な通知や履歴保存を行わず、あくまで気分転換・雰囲気づくりが主目的です。

# 公式ドキュメント抜粋
本SkillはGitのコミット時に発動するため、GitのフックやCI/CDパイプラインとの連携も可能です。詳細は[Git公式](https://git-scm.com/book/ja/v2/Customizing-Git-Git-Hooks)を参照してください。

# 利用例
- チーム開発でコミットごとに気分天気が表示され、Slack等で話題に。
- 個人開発で作業の合間にリラックス効果。

# 注意点
- 出力は標準出力のみ、他サービス連携なし。
- 実用的な作業ログや通知ではないため、業務用途には不向き。

# 設計方針
- シンプルな構成で環境依存なし。
- メッセージは短く、邪魔にならない分量に限定。
- 拡張性を考慮し、気分天気やコメントはリストで管理。
