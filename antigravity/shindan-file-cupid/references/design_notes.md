# 概要
shindan-file-cupidは、プロジェクト内のファイル名リストをもとに“相性診断”を行うジョーク・パロディSkillです。実用性よりも、チームの雰囲気づくりやアイスブレイク、会話のネタ提供を目的としています。

# 設計方針
- 診断ロジックは完全にランダムまたは根拠のないものとし、ファイル内容や実際の依存関係は一切考慮しません。
- 除外パスやシステムファイルは自動的に無視される設計です。
- 実行結果は標準出力のみで、ファイルの変更や保存は行いません。

# 利用例
- チームミーティングの冒頭で、プロジェクトの「相性ペア」を紹介し場を和ませる
- 新人歓迎会やプロジェクトキックオフ時の話題作り

# 注意点
- 診断結果はジョークであり、プロジェクト運用や設計判断には一切使えません。
- 本SkillはAntigravity専用であり、他のAI Coding Agentとの互換性は保証しません。
