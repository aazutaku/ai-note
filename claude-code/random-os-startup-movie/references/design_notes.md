# 概要
このSkillは、ユーザーがターミナルやエディタを起動するたびに、毎回異なる“架空OSの起動画面”をテキストアニメーションとして表示することで、作業前の一瞬にユーモアや未来感を演出します。実用性はほぼゼロですが、日々の開発や作業の“儀式”として楽しめる設計です。

# 公式ドキュメント抜粋
- Python標準ライブラリのみ使用し、外部依存を排除
- argparseでCLIサブコマンド・オプションを実装
- 毎回ランダム生成されるため、出力の再現性はありません

# 利用例
- ターミナルのプロファイル起動時に自動実行
- /random-os-startup-movie コマンドで手動発動
- VSCode等のエディタ拡張と連携して“起動演出”を差し込む

# 注意点
- 実在のOSやシステムには一切影響しません
- ログ保存や履歴機能はありません
- システムクリティカルな自動化スクリプトでは発動しない設計です

# 設計方針
- 完全な“無駄演出”を追求し、実用性より体験性を重視
- テキストアニメーションはシンプルながらも毎回違う内容に
- コードは100行以上で拡張性・可読性を確保
