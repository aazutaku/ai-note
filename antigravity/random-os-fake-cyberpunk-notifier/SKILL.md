# 機能概要
このSkillは、あなたのデスクトップに近未来SF世界の架空OS通知をランダムに表示します。通知内容は「ニューロン接続不良」や「シンセ脳波警告」など、現実には存在しないがサイバーパンク世界観に没入できるものばかりです。作業に行き詰まった時や、現実逃避したい時に気分転換として最適です。通知はOSの標準通知APIを用いて表示され、実用性はありませんが、無駄にかっこいい演出を提供します。

# 使い方
このSkillは明示的な呼び出しを必要とせず、「サイバーパンク」「通知」「演出」「現実逃避」などのキーワードや、長時間作業時・集中力低下時に自動発動します。skillの設定で通知頻度や稼働時間帯を調整可能です。

# 出力例
```
[CYBER OS NOTIFIER]
タイトル: ニューロン接続不良を検出
内容: システムは脳内ニューロン接続の不安定化を検出しました。バイオウェア再起動を推奨します。

タイトル: シンセ脳波警告
内容: 異常なシンセ脳波パターンを検出。メンタル・ファイアウォールの強化を推奨。

タイトル: クローム義手アップデート推奨
内容: 最新のクローム義手ドライバが利用可能です。アップデートをお勧めします。
```

# 注意点
本Skillは完全な演出目的であり、通知内容に実用的な意味はありません。通知頻度が高すぎる場合は設定で調整してください。ローカルに通知履歴は保存されません。セキュリティや業務用途には使用しないでください。
