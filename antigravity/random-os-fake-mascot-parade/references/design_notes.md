# 概要
本Skillは、作業中のユーザーに対し、完全ランダムなタイミングと内容で架空のOSマスコットによる通知を表示するジョーク機能です。実用性はなく、主に「唐突さ」「謎の愛嬌」「OS風味のセリフ」に重点を置いて設計されています。

# 公式ドキュメント抜粋
- Python標準ライブラリ random: https://docs.python.org/3/library/random.html
- plyer通知API: https://plyer.readthedocs.io/en/latest/

# 利用例
- 長時間作業中の気分転換や話題作りに
- チーム内でのネタ共有やイベント演出に
- 開発環境のデモや勉強会でのアイスブレイク

# 注意点
- 通知頻度は1回目1-3分後、以降3-10分間隔で調整
- システム通知APIの制限により、CUIやリモート環境では通知が表示されません
- 履歴やローカル保存は行いません

# 設計方針
- キャラ名・セリフともに毎回完全ランダム化
- 過剰な通知を避けるため、間隔は十分に長めに設定
- 追加キャラやセリフは容易に拡張可能な構造
