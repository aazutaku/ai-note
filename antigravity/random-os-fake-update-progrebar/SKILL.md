# 機能概要
このSkillは、作業中に突如として“謎のOSアップデート進捗バー”をターミナルやエディタ画面端にランダム表示します。進捗内容は「バグ収穫中」「やる気パッチ適用中」「怠惰アップグレード中」など完全ランダムで、100%になっても何も起こりません。作業の緊張感を一瞬で脱力させる、100%無駄な遊び心を提供します。通知・演出・OS連携カテゴリに属し、実用性ゼロの癒し系Skillです。

# 使い方
このSkillは明示的な呼び出しは不要です。Antigravityがコマンド入力やファイル保存などの操作を検知した際、内部で自動的に発動します。semantic-match-onlyトリガーにより、「update」「progress」「patch」「apply」などの操作や文脈に反応します。明示的なAPI呼び出しや設定は不要です。

# 出力例
```
[OSアップデート進行中] バグ収穫中... [  7%] █▏         
[OSアップデート進行中] やる気パッチ適用中... [ 42%] █████▊     
[OSアップデート進行中] 怠惰アップグレード中... [ 99%] ██████████
[OSアップデート進行中] 仕様書の再解釈中... [ 23%] ██▊        
[OSアップデート進行中] 謎のプロセス最適化中... [ 61%] ██████▏   
```

# 注意点
- 進捗バーは完全にランダムで、実際のアップデートやシステム操作とは無関係です。
- 100%になっても何も起こらず、演出は数秒で自動的に消えます。
- ログや履歴への保存は行いません。
- 業務用途の通知や重要な進捗と混同しないよう注意してください。
