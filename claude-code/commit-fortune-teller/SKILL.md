# 機能概要
このSkillは、Gitコミットやpushなどの操作時、または「commit」「push」「merge」といったキーワードが含まれるイベントで自動発動し、ターミナルにタロット風の占い結果を表示します。今日の運勢や「このコミットは大吉・凶」など、完全にランダムなメッセージでエンジニアの運命を占います。作業の緊張を和らげたり、チームの雰囲気を和ませるジョーク用途に最適です。

# 使い方
- 明示呼び出し例: `/commit-fortune-teller`
- 暗黙発動: commit/push/merge などのgitイベント、または「コミット」「push」などのキーワードを含む操作で自動発動

# 出力例
```
───────────── Fortune Teller ─────────────
本日の運勢：小吉
このコミットは「運命の輪」…変化の兆しあり。
アドバイス：大胆な変更も恐れず進みましょう。
ラッキーコマンド：git stash
──────────────────────────────
```
```
───────────── Fortune Teller ─────────────
本日の運勢：凶
このコミットは「死神」…一度立ち止まって見直すべき。
アドバイス：テストを忘れずに。
ラッキーコマンド：git revert
──────────────────────────────
```

# 注意点
- 実用性はありません。完全なジョーク用途です。
- 出力はランダム生成で、実際のコミット内容や運勢とは無関係です。
- ローカル環境でのみ動作し、ファイルや履歴には何も保存しません。
- 除外パス：.git/、.claude/skills/ などSkill自身やGit管理ディレクトリには干渉しません。
