# 機能概要
このSkillは、デスクトップに突然「謎のOS緊急アラート」風の通知を表示し、作業現場にカオスな笑いをもたらします。通知内容は『危険：バグが自我を獲得しました』『注意：メモリがピクニック中』など、全く業務に関係ない意味不明な警告ばかり。実害ゼロで、真面目な空気をぶち壊したい・チームの緊張を和らげたいときに最適です。

# 使い方
本Skillは明示的な呼び出しは不要です。Antigravityが「カオス」「混乱」「ジョーク」「通知」「警告」などのキーワードや、場の空気を和ませたい・リフレッシュしたいといった文脈を検知した際、自動的に発動します。明示的な呼び出し例はありません。

## 暗黙発動キーワード例
- 「カオスな通知を出して」
- 「場の空気を壊したい」
- 「ジョークアラート」
- 「謎警告を表示」

# 出力例
```terminal
[通知] 危険：バグが自我を獲得しました
[通知] 注意：メモリがピクニック中
[通知] 深刻：キーボードが人生に迷っています
[通知] 警告：OSが哲学的問いに悩んでいます
[通知] 重大：CPUが詩を書き始めました
```

# 注意点
- 本Skillは実際のシステムや業務には影響を与えません。
- 通知はローカル環境でのみ表示されます（macOS, Windows, Linux対応）。
- オフにしたい場合はスクリプトを停止またはアンインストールしてください。
- 通知内容は完全にランダムで、業務連絡や重要情報は一切含まれません。
