# 機能概要
commit-fortune-cookieは、コードコミットのたびに「今日の運勢」をターミナルに強制表示するエンタメスキルです。コミットメッセージやタイムスタンプをもとに、吉凶・ラッキー言語・謎アドバイス（例:「今日はJavaScriptに手を出すな」「大吉:コンフリクトなし」など）を毎回違うテンションで生成。開発現場に遊び心を加え、肩の力を抜いて作業できる雰囲気を演出します。

# 使い方
- 明示呼び出し: `/commit-fortune-cookie` をターミナルで実行
- 暗黙発動: `commit`, `push`, `merge`, `コミット`, `git commit` などの操作時に自動発動
- 追加オプション不要、既存のワークフローに干渉しません

# 出力例
```
=== 今日の開発運勢 ===
運勢: 小吉
ラッキー言語: Python
アドバイス: 「今日はPRレビューでツッコミが入るかも。気を抜くな！」
コミットメッセージ: "fix: typo in README"
タイムスタンプ: 2024-06-15 14:23:45
=======================
```
```
=== 今日の開発運勢 ===
運勢: 凶
ラッキー言語: Rust
アドバイス: 「動くと思うな」
コミットメッセージ: "feat: add login endpoint"
タイムスタンプ: 2024-06-15 15:01:12
=======================
```

# 注意点
- 本スキルはジョーク用途です。運勢内容は完全にランダム生成されます。
- コミット内容や履歴には影響しません。
- ローカル環境でのみ動作し、運勢結果は保存されません。
- 一部のCI/CD自動コミットには反応しない場合があります。
