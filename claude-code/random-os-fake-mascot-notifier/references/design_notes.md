# 概要
このSkillは、作業中に架空のOSマスコットが唐突に現れて全く役に立たない激励や警告をデスクトップ通知で発するという、遊び心に特化した演出系ツールです。現実逃避や気分転換、集中のリフレッシュを目的としています。

# 公式ドキュメント抜粋
通知APIにはPythonの[plyer](https://plyer.readthedocs.io/en/latest/)を採用。Windows, macOS, Linuxの主要デスクトップ環境で動作し、クロスプラットフォームな実装が可能です。

# 利用例
- 長時間のコーディングや単調作業の合間に、作業者の気分を和ませる
- チームの雑談やペアプロ時のアイスブレイク
- 開発環境の演出やイベント用途

# 注意点
通知は一時的な演出のみで、履歴や個人情報の保存は行いません。通知頻度は10〜30分間隔で自動調整され、過剰な通知を避ける設計です。plyerの仕様により一部環境で通知が表示されない場合があります。

# 設計方針
- マスコット名・メッセージともに完全ランダムでバリエーション豊富
- 実在のOSやキャラクターとは無関係な創作マスコットのみを採用
- コマンドラインから明示的呼び出しや、常駐型のループ実行に対応
- コードは100行以上でメンテナンス性と拡張性を重視
