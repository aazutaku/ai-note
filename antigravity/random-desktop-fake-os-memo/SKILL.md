# 機能概要
random-desktop-fake-os-memoは、デスクトップ上に突然“謎のOSメモ”を通知として表示し、作業中のユーザーにシュールな混乱と遊び心を提供するスキルです。通知内容は完全ランダムで、実用性ゼロの怪文書や意味不明なメッセージばかり。退屈な作業や単調なデスクトップ環境に、意図的な違和感や笑いをもたらします。集中しすぎている時ほど、思わず二度見してしまうような体験を演出します。

# 使い方
このSkillは明示的な呼び出しは不要です。Antigravityが「デスクトップ通知」「OS演出」「作業中の気分転換」などのキーワードや状況を検知した際、自動的に発動します。ユーザーが何も操作しなくても、定期的またはランダムなタイミングで怪文書メモをデスクトップ通知として表示します。

# 出力例
```
[通知] OSメモ: 冷蔵庫にプリンあり
[通知] OSメモ: 上司が見ている
[通知] OSメモ: バグは空から降ってくる
[通知] OSメモ: 謎の引き出しを開けるな
[通知] OSメモ: 今日は何曜日でもない
[通知] OSメモ: システムがあなたを見つめています
[通知] OSメモ: 画面の右下を見てはいけない
[通知] OSメモ: コードに名前をつけてはいけない
```

# 注意点
- 通知内容は完全に意味不明で、実用的な情報は一切含まれません。
- スキルはローカル環境でのみ動作し、通知内容の履歴や保存機能はありません。
- 通知API（notify-send等）に依存するため、環境によっては通知が表示されない場合があります。
- 明示的なトリガーやコマンドは用意されていません。
