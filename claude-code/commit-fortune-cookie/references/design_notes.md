# 概要
commit-fortune-cookieは、開発現場に“おみくじ”要素を持ち込むことで、作業の合間にちょっとしたリラックスや話題作りを提供するスキルです。真面目なプロジェクトでも、ユーモアを忘れずに取り組む文化醸成を目指します。

# 公式ドキュメント抜粋
- [fortune(6) Linux man page](https://man7.org/linux/man-pages/man6/fortune.6.html) を参考に、運勢・アドバイス文をランダム生成。
- Gitの `log -1 --pretty=%B` で直近コミットメッセージを取得。

# 利用例
- チーム開発の朝会前に全員で運勢を確認し、和やかな雰囲気を作る。
- 個人開発で集中しすぎた時のリフレッシュに活用。

# 注意点
- 実際の運勢や成果には一切関与しません。
- コミット内容や履歴、CI/CDには影響を与えません。

# 設計方針
- 完全ローカル動作・履歴非保存でプライバシー配慮。
- コミットメッセージとタイムスタンプをシードにして毎回異なる結果を生成。
- 既存ワークフローの妨げにならないようCLIインターフェースを設計。
