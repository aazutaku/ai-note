# 機能概要
このSkillは、普段のコマンドライン作業やファイル操作の合間に、完全ランダムで“謎のコンフェッティ（紙吹雪）祝福メッセージ”をOSの通知領域へ表示します。内容は「本日も出社おめでとう」「意味なく祝福します」「さっきのls、最高でした」など、全く脈絡のないお祝いばかり。作業に没頭しがちな開発現場へ、唐突な非日常を演出し、気分転換やユーモアを提供します。通知は過剰にならないよう、一定間隔・確率で発動します。

# 使い方
このSkillは明示的な呼び出しは不要です。Antigravityが `ls`、`git`、`make`、`build`、`run` などの作業系コマンドや、ファイル編集・保存などのイベントを検知した際、ランダムなタイミングで自動発動します。暗黙発動キーワード例：`ls`、`git commit`、`make build`、`vim` など。

# 出力例
```
[通知] 本日も出社おめでとう！
[通知] さっきのls、最高でした。
[通知] 意味なく祝福します。
[通知] そのgit add、素晴らしい判断です。
[通知] 今日は何もなくても祝っておきます。
[通知] runコマンド、見事な実行力！
```

# 注意点
- 通知はOSの標準通知API（Windows: win10toast, macOS: pync, Linux: notify2等）を利用します。
- 頻度は過剰にならないよう制御されますが、環境によっては通知が重複・遅延する場合があります。
- ローカル環境でのみ動作し、通知内容は保存されません。
- サーバー・リモート環境では通知が表示されない場合があります。
