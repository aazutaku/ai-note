# 機能概要
このSkillは、作業中に突如として「OS公式っぽい」コーヒーブレイク推奨通知をデスクトップに表示します。通知内容は毎回ランダムで、まるで本当にOSから推奨されたかのような説得力のあるメッセージが飛び出します。ユーザーの集中力や生産性を測るのではなく、理不尽かつユーモラスなタイミングで休憩を促すことで、作業空間にちょっとした遊び心とリフレッシュを提供します。

# 使い方
このSkillは明示的な呼び出しは不要です。Antigravityが「コーヒー」「休憩」「集中力」「疲労」などのキーワードや、長時間の作業継続を検知した際に自動で発動します。ユーザーは特別な操作をせずに、ランダムなタイミングで通知を受け取ることができます。

# 出力例
```
[通知] 重要：カフェイン補給の時刻です。
[通知] タスク進捗が停滞中。コーヒーを飲めとの神託。
[通知] AI的診断：あと5分で集中力が消滅します。
[通知] システムガイドライン：短い休憩を推奨します。
[通知] OSリソース監視：コーヒーブレイクが必要です。
```

# 注意点
- 本Skillは実際のOSやシステムの状態を監視・制御するものではありません。
- 通知は完全にランダムなタイミング・内容で表示され、実害やデータ損失は発生しません。
- ローカルに通知履歴や個人情報を保存することはありません。
- 通知内容はジョークを含みますが、業務上の重要なアラートと誤認しないようご注意ください。
