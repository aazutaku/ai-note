# 概要
本Skillは、作業中に突発的かつ意味不明な“OSキャラクター退職通知”を生成し、ユーザーの注意を一瞬だけ別世界に誘う演出を目的としています。

# 公式ドキュメント抜粋
通知には plyer.notification を利用し、クロスプラットフォームでデスクトップ通知を実現します。詳細: https://plyer.readthedocs.io/en/latest/

# 利用例
長時間のコーディングや「退職」「転職」などの単語を含む作業時、1時間に1回程度の頻度で、通知が自動発火します。CLIから `python os_retirement_announcement.py log` で即時発火も可能です。

# 注意点
- 通知内容は完全ランダム生成で実在の人物・組織とは無関係です。
- 通知履歴はホームディレクトリの隠しファイルに保存されます。
- 通知APIの制限で、環境によっては表示されない場合があります。

# 設計方針
頻度制限(1時間1回)と履歴管理によって、過剰な通知やスパム化を防止。テンプレート・役職・文面を随時追加することで拡張性も確保しています。
