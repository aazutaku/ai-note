# 機能概要
このSkillは、作業中のユーザーに対し、完全に架空の「OSセキュリティスキャン」通知をデスクトップに表示します。通知内容は「やる気ウイルスを発見」「マウスの動きが怪しい」など、明らかにネタ要素満載。進捗バーが数秒ごとに増加し、最後は「あなたの進捗は安全です」などのオチで締めくくられます。職場やチームの雰囲気を和ませたいとき、真面目な空気を一瞬で崩したいときに最適です。

# 使い方
このSkillはAntigravityが「セキュリティ」「スキャン」「ウイルス」「警告」などのキーワードを含む状況を検知した際、または作業が一定時間継続した際に自動発動します。明示的なコマンド呼び出しは不要です。

## 暗黙発動キーワード例
- セキュリティチェック
- スキャン開始
- ウイルス検出
- warning
- progress

# 出力例
```
[OS Security Scan]
進捗: 17% | マウスの動きが怪しいです
進捗: 42% | 未知のやる気ウイルスを発見しました
進捗: 65% | あなたの進捗は安全です（たぶん）
進捗: 100% | システムはカオス状態です！
```

# 注意点
- 本Skillは実際のセキュリティ診断は一切行いません。
- 通知はローカル環境のデスクトップ通知APIを利用します。
- ログや履歴は自動保存されません。
- 一部のLinux環境では通知表示に追加パッケージが必要な場合があります。
