# 概要
terminal-samurai-weathercasterは、作業開始時にユーモアと意外性を提供するための演出系スキルです。実際の天気やAPIには一切依存せず、完全にローカルでランダムな侍コメントを生成します。

# 公式ドキュメント抜粋
- Python標準ライブラリ (random, argparse, datetime) のみ利用
- 外部APIやネットワークアクセスは一切行いません

# 利用例
- ターミナルのプロファイルや.zshrc/.bashrcに `python .claude/skills/terminal-samurai-weathercaster/samurai_weathercaster.py run` を追加することで、毎回自動発動可能
- チームの朝会や始業スクリプトに組み込むことで、雰囲気づくりやアイスブレイクにも活用できます

# 注意点
- 出力内容は完全ランダムで、現実の天気や気象情報とは無関係です
- ログ記録機能は任意で、履歴を残したい場合のみ利用してください

# 設計方針
- シンプルな構成と可搬性を重視し、依存パッケージを極力排除
- コメントや天気のバリエーションは今後も容易に拡張可能な設計
