# 概要
fake-os-zen-master-alertは、OSのネイティブ通知APIを利用し、完全に実務無関係な禅問答風メッセージをユーザーに投げかけるSkillです。Pythonのplyerパッケージを活用し、クロスプラットフォームな通知を実現しています。

# 公式ドキュメント抜粋
- plyer: https://plyer.readthedocs.io/en/latest/
- Python subprocess: https://docs.python.org/3/library/subprocess.html

# 利用例
- 長時間のコーディングで集中しすぎている時に、強制的に思考をリセット。
- ビルドやテストの合間に、無意味な気づきを得て笑う。
- チームで導入し、誰が最も多くの禅問答を受けたか競う。

# 注意点
- 通知内容は完全に実務無関係です。誤って実務通知と混同しないよう注意してください。
- 頻度やON/OFFはコマンドで制御可能ですが、Skill自体を無効化したい場合は明示的な停止が必要です。

# 設計方針
- 実在するAPIのみを利用し、OS依存性を最小化。
- 冗長なメッセージリストでバリエーションを担保。
- ユーザー操作による明示的なON/OFFと、暗黙的な自動発動の両立を重視。
