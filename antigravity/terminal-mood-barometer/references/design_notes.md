# 概要
terminal-mood-barometerは、ターミナル作業時に遊び心ある“やる気指数”をランダム表示し、作業体験を和らげる目的で設計されています。実用性よりも演出・通知・気分転換を重視しています。

# 公式ドキュメント抜粋
Python標準のrandomモジュールとsubprocess、argparseを活用し、コマンド実行時に気分指数を決定・表示します。ログは~/.mood_barometer.logに保存され、list/summaryサブコマンドで履歴や統計も確認できます。

# 利用例
- 長時間のコーディングやビルド作業時に、気分指数でちょっとした笑いを提供
- チームの朝会やペアプロで雑談のきっかけに
- ログを見返して「今日はやる気が高かった」などの話題作り

# 注意点
- 気分指数は完全ランダムで、実際の心理状態や生産性とは無関係です
- 除外コマンドやパイプ処理では演出を抑制
- ログは個人利用を前提とし、セキュリティやプライバシーに配慮しています

# 設計方針
演出は1行で簡潔に、コマンド実行の邪魔にならないよう配慮。拡張性を考慮し、コメントリストや除外コマンドも容易にカスタマイズ可能です。
