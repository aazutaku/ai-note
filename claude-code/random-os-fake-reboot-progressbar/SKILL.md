# 機能概要
`random-os-fake-reboot-progressbar`は、作業中に突然“謎のOS再起動進捗バー”を画面端やメニューバー風に表示し、ユーザーに焦りと笑いを提供する演出用Skillです。進捗バーは「再起動準備中…」「謎のプロセス終了中…」など毎回異なるメッセージをランダム生成し、進行状況も徐々に変化します。実際の再起動やシステム操作は一切行わず、あくまで仮想的な演出を目的としています。集中しすぎた頭をリセットしたい時や、冗談でチームを驚かせたい時に最適です。

# 使い方
- 明示的な呼び出し例: `/random-os-fake-reboot-progressbar` または「謎の進捗バーを表示して」などの指示
- 暗黙発動キーワード例: 「再起動」「リセット」「進捗」「プロセス終了」などを含む会話やコマンド

# 出力例
```
[  7%] 再起動準備中…
[ 21%] 謎のプロセス終了中…
[ 34%] システムキャッシュを解放しています…
[ 56%] セキュリティチェック中…
[ 78%] ほぼ完了、あと3分で全てがリセットされます
[100%] 完了！何も起きませんでした。
```

# 注意点
- 実際の再起動やプロセス終了は一切行いません
- ローカルファイルや設定には変更を加えません
- 演出目的のため、業務システム等での誤用にご注意ください
- 進捗メッセージは毎回ランダム生成されます
