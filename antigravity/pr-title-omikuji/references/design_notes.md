# 概要
このSkillはPRタイトルを“おみくじ”風に変換し、開発現場にエンタメ要素を提供します。運勢ワードとフレーズは複数用意し、バリエーションを確保しています。

# 公式ドキュメント抜粋
GitHub Pull Request API: https://docs.github.com/en/rest/pulls/pulls

# 利用例
- PR作成時に自動でタイトルが「大吉: 運命のリファクタリング」等に変換される
- previewサブコマンドでローカルでタイトル案を確認可能

# 注意点
- PR本文や他フィールドは変更しません
- タイトルが既に運勢ワードで始まる場合は再変換しません
- GitHubアクセストークンが必要です

# 設計方針
タイトルの運勢ワードとフレーズはランダムで組み合わせ、毎回異なるタイトルを生成。APIエラー時は詳細をstderrに出力し、exit codeで通知します。
