# 機能概要
commit-fortune-cookieは、Gitのコミット時や明示呼び出しで、あなたの作業にエンタメ要素を加えるスキルです。コミットメッセージや日時からランダム性を生成し、吉凶・ラッキー言語・謎のアドバイスなどを毎回違うテンションで表示します。シリアスな開発現場にちょっとした笑いとリフレッシュを提供し、コミットごとに肩の力を抜くきっかけを作ります。

# 使い方
- 明示呼び出し例: `/skills menu` からcommit-fortune-cookieを選択、または `/commit-fortune-cookie` と入力
- 暗黙発動キーワード例: `git commit` 実行時、または「コミット」や「push」等のワードを含む操作

# 出力例
```
=== 今日の開発運勢 ===
運勢: 大吉
ラッキー言語: Python
アドバイス: 今日はJavaScriptに手を出すな。
コミットメッセージ: fix: バグ修正
タイムスタンプ: 2024-06-28 14:23:10
----------------------
運勢: 凶
ラッキー言語: Rust
アドバイス: 動くと思うな。
コミットメッセージ: feat: 新機能追加
タイムスタンプ: 2024-06-28 15:02:51
```

# 注意点
- 出力は標準出力のみで、ファイルやGit履歴は変更しません。
- コミットごとに異なる結果を出すため、メッセージや時刻をもとに乱数を生成します。
- ローカル環境でのみ動作し、CI/CDやリモートでは発動しません。
- 除外パスや特定のファイルには影響しません。
