# 機能概要
このSkillは、あなたの作業環境に“謎のOS再起動進捗バー”をランダムに表示します。進捗バーは「再起動準備中…」「謎のプロセス終了中…」などの意味不明なメッセージとともに進行し、「あと3分で全てがリセットされます」など、焦りと笑いを誘う演出を提供します。実際のシステム再起動やプロセス終了は一切行われません。集中しすぎた時のリフレッシュや、会話・作業フローに混乱とユーモアを加えたい場面に最適です。

# 使い方
- 明示呼び出し例: `/skills random-os-fake-reboot-progressbar` または `@codex skill:random-os-fake-reboot-progressbar`
- 暗黙発動キーワード例: 「再起動」「進捗バー」「OSリセット」「プロセス終了」「冗談」「リフレッシュ」「混乱」などが会話に含まれると自動発動。

# 出力例
```
[==>                  ] 12% 再起動準備中…
[=====>               ] 28% 謎のプロセス終了中…
[========>            ] 41% システムキャッシュを解放中…
[=============>       ] 67% 仮想デバイスを初期化中…
[==================>  ] 92% あと3分で全てがリセットされます
[====================] 100% 完了！実際には何も起きませんでした。
```

# 注意点
- 本Skillは視覚的な演出のみで、実際の再起動やシステム操作は一切行いません。
- 進捗バーやメッセージは毎回ランダム生成されます。
- ローカル保存や履歴記録は行いません。
- 長時間の進捗バー表示や、複数同時実行には非対応です。
