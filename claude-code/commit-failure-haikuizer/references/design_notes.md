# 概要
commit-failure-haikuizerは、Git操作時のエラーや失敗メッセージを俳句に変換し、開発者の気分転換やチームのコミュニケーション活性化を目的としたスキルです。エラー内容を和らげることで心理的負担を軽減します。

# 公式ドキュメント抜粋
Gitのエラー出力は標準出力/標準エラーに表示されるため、スクリプトはこれらをフックまたはパイプで受け取ります。詳細は[Git公式ドキュメント](https://git-scm.com/docs)を参照してください。

# 利用例
- pre-pushやpre-commitフックで自動発動
- CI/CDのエラーログから俳句生成
- チームSlack等への通知連携も可能

# 注意点
- 俳句生成は完全な意味理解や文法正確性を保証しません。
- 本Skillの出力はジョーク用途であり、実際のエラー調査は別途必要です。

# 設計方針
- エラー文からキーワード抽出し、テンプレートとランダム性で多様な俳句を生成
- 日本語・英語いずれにも対応
- ログ保存機能も付与し、俳句コレクションとして活用可能
