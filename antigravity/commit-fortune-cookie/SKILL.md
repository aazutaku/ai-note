# 機能概要
commit-fortune-cookieは、開発者がコミット操作を行うたびに、ターミナル上で“今日の運勢”を強制表示するエンタメ系Skillです。コミットメッセージやタイムスタンプをもとに、吉凶・ラッキー言語・謎のアドバイス（例:「今日はJavaScriptに手を出すな」「大吉:コンフリクトなし」など）を毎回違うテンションで表示します。真剣な開発現場にもユーモアを提供し、コード作業の緊張を和らげます。

# 使い方
このSkillは明示的な呼び出しは不要です。`commit`, `push`, `git`, `コミットメッセージ`などの操作やキーワードを含むタイミングで自動的に発動します。Skillの設定やカスタマイズは不要で、既存のワークフローを妨げません。

# 出力例
```
=== 今日のコミット運勢 ===
運勢: 中吉
ラッキー言語: Python
アドバイス: テストコードは後回しにしないで。
コミット: "fix: バグ修正"
時刻: 2024-06-20 13:45:12
=========================

=== 今日のコミット運勢 ===
運勢: 凶
ラッキー言語: Go
アドバイス: 今日はJavaScriptに手を出すな。
コミット: "feat: add user login"
時刻: 2024-06-20 14:02:01
=========================
```

# 注意点
- Skillの出力はターミナル表示のみで、コミット内容や履歴には影響しません。
- ローカル環境にのみ適用され、リモートリポジトリやCI/CDには影響しません。
- 除外パスや特定ディレクトリでの抑制設定はありません。
- 運勢やアドバイスはランダム生成されるため、内容の正確性や実用性は保証されません。
