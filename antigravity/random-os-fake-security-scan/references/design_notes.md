# 概要
このスキルは、作業中のユーザーに対し、完全に架空のセキュリティスキャン通知をデスクトップ領域にランダム表示し、職場やチームの雰囲気を和ませることを目的としています。実際の検査や診断は一切行いません。

# 公式ドキュメント抜粋
通知表示にはPythonのplyerライブラリを利用し、クロスプラットフォームなデスクトップ通知を実現しています。
- https://plyer.readthedocs.io/en/latest/
- https://docs.python.org/3/library/random.html

# 利用例
- チームの朝会で突然セキュリティスキャン通知を流して場を和ませる
- 長時間作業している同僚のPCに仕込んで笑いを誘う

# 注意点
- 本Skillはジョーク目的であり、業務システムや重要な作業環境への導入は推奨されません。
- Linux環境ではlibnotify等の追加パッケージが必要な場合があります。

# 設計方針
- 完全にネタ要素と分かる文言・進捗演出を重視
- 通知API利用でローカルに痕跡を残さない
- コマンドラインからもデモ・dry-runが可能
