# 機能概要
`random-os-fake-error-prophecy`は、ターミナルやエディタでの操作時に、まるでOSが未来を予言するかのような“エラープロフェシー”通知をランダム生成して表示するエンタメSkillです。例えば「エラー: 明日は会議中にWi-Fiが落ちます」や「警告: あなたのマウスが反逆します」など、完全に架空かつ無責任な未来のトラブル予告が毎回異なる内容で現れます。作業の緊張感を和らげ、思わず笑ってしまうようなカオスな演出を提供します。

# 使い方
- 明示呼び出し: `/random-os-fake-error-prophecy` を実行すると即座に予言通知が現れます。
- 暗黙発動: `run`, `build`, `commit`, `save`, `deploy` などの操作や、コマンド実行時に自動で発動します。

# 出力例
```
エラー: 明日の午後、あなたのキーボードが自動でCapsLockになります
警告: 来週金曜、プリンタが紙詰まりを予言しています
エラー: 次のアップデート時、OSがコーヒーをこぼします
警告: 3日後、USBメモリが自我に目覚めるかもしれません
エラー: 近日中にディスプレイが突然哲学的な質問をします
```

# 注意点
- 本Skillは完全にジョーク目的であり、実際のエラーやシステム挙動には一切影響を与えません。
- 通知内容は毎回ランダム生成され、同じ内容が続くことはありません。
- ログやファイル保存は行いません。通知はその場限りです。
- 重要な作業中の混乱を避けるため、明示的な除外パス設定も可能です。
