# 概要
terminal-samurai-weathercasterは、ターミナル起動時に侍風の天気コメントをランダム表示する演出系Skillです。現実の天気や外部APIには一切依存せず、完全に内部で定義した天気・侍台詞からランダム生成します。

# 公式ドキュメント抜粋
Pythonのrandomモジュール（https://docs.python.org/ja/3/library/random.html）を用いて、天気・台詞の選択を行います。ターミナル出力には標準出力（sys.stdout）を使用。

# 利用例
ターミナルを開くたびに、侍コメントが自動で表示され、作業開始時の気分転換やユーモア演出に活用できます。コマンドライン引数でログ記録や履歴参照も可能です。

# 注意点
現実の天気や位置情報、外部APIは一切参照しません。Skillの動作は完全にローカルで完結します。ユーザーの作業内容や気分とは無関係に、毎回ランダムな侍コメントが降臨します。

# 設計方針
Skillは明示的な呼び出し不要で、ターミナル起動時に常時発動する設計です。出力内容の多様性を担保するため、天気・台詞ともに複数パターンを用意し、組み合わせで毎回変化を生み出します。
