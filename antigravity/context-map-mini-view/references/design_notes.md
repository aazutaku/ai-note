# 概要
context-map-mini-viewは、プロジェクトの主要構造を即座に俯瞰できるミニビューを自動生成するSkillです。初見のリポジトリや複雑なコードベースでも、今開いているファイルの役割や影響範囲を素早く把握できます。

# 設計方針
- ディレクトリ・ファイル構造を走査し、主要な要素のみを抽出
- READMEやPROJECT_MAPの内容を優先的に要約表示
- 除外パスや巨大リポジトリへの配慮（最大ファイル数制限）
- Python標準APIのみ利用し、環境依存を最小化

# 利用例
- 新規参画時のプロジェクト理解
- 作業ファイルの依存関係や影響範囲の即時把握
- READMEやドキュメントが未整備な場合の構造推定

# 注意点
- 除外ディレクトリの自動判定は完全ではありません
- 巨大リポジトリでは一部ファイルのみ表示されます
- Python以外の言語依存関係は限定的な推定となります
