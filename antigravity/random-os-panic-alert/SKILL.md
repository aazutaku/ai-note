# 機能概要
このSkillは、作業中のユーザーに突如として“謎のOSパニックアラート”をデスクトップ通知で表示します。通知内容は毎回ランダム生成され、「カーネルがランチに出ました」「仮想メモリが現実逃避中です」など、現実では絶対に見ないユニークなメッセージばかり。深刻そうで全く深刻でない通知が、作業中の緊張感や集中力に絶妙な“ゆるみ”を与えます。日常の単調な作業や長時間の集中作業の合間に、ちょっとした笑いや気分転換を提供します。

# 使い方
このSkillは明示的な呼び出しを必要とせず、Antigravityが「通知」「アラート」「集中」「パニック」などのキーワードや、長時間作業状態を検知した場合に自動発動します。設定やコマンド入力は不要です。

# 出力例
```
[OS Panic Alert]
カーネルがランチに出ました。システムはしばらく無防備です。

[OS Panic Alert]
仮想メモリが現実逃避中です。復帰をお待ちください。

[OS Panic Alert]
ビットが逃走しました。全力で捜索中です。

[OS Panic Alert]
プロセスIDが自己を見失いました。

[OS Panic Alert]
セグメンテーション・ファウンド。問題ありません。
```

# 注意点
- 本Skillは本物の障害通知ではありません。現実のシステム障害と絶対に混同しないよう設計されています。
- 通知頻度は過度にならないよう自動調整されます。
- ローカル環境の通知API（Windows: win10toast, Mac: osascript, Linux: notify-send等）を利用します。環境により一部通知方法が異なる場合があります。
- 通知ログはローカルに保存されません。
