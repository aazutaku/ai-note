# 機能概要
このSkillは、作業に集中し始めたその瞬間や「アップデート」「通知」「バージョン」などの単語が現れた際に、まるで本物のOSアップデート通知のような見た目で、完全ランダムかつ明らかに嘘くさい更新内容をデスクトップ通知として表示します。通知内容は「新たなバグの追加」「意味不明な機能強化」など、思わずツッコミたくなるものばかり。職場やオンライン会議中に仕込むことで、周囲の笑いと戸惑いを誘い、堅苦しい雰囲気を和らげます。

# 使い方
このSkillは明示的な呼び出しコマンドは不要です。下記のようなキーワードが会話やコードに現れると自動発動します：
- アップデート
- バージョン
- 通知
- OS
- update
- upgrade

また、Skillの設定で通知頻度や発動タイミングを細かく調整可能です。

# 出力例
```
[OS Update Notifier]
新機能: ウィンドウを閉じるときに謎の効果音が鳴ります。
バグ修正: ごく稀にマウスが逆方向に動く問題を未確認で修正。
既知の問題: 画面が突然ピンク色になることがありますが仕様です。
詳細: https://os.fakeupdates.example.com/notes/2024.06.01
```

# 注意点
- 実際のOS通知と誤認しないよう、明らかに架空と分かる内容を生成します。
- 通知はローカル環境でのみ表示され、外部サーバーへの送信は行いません。
- 頻度や発動条件は設定ファイルで調整可能です。
- 本Skillはジョーク用途であり、実業務への影響や誤解を招かないようご注意ください。
