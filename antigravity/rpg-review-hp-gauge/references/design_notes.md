# 概要
このSkillは、コードレビューの指摘数をRPGのHPゲージ風に可視化することで、開発現場に遊び心と緊張感をもたらします。指摘数が多いほどHPが減り、0になると「力尽きた」演出が表示されます。

# 公式ドキュメント抜粋
HPゲージの表示や指摘数のカウントは、Pythonスクリプトで自動化されています。履歴はJSONでローカル保存され、過去のレビュー状況も確認可能です。

# 利用例
- PRレビュー時に指摘数を入力し、HPゲージで可視化
- チームのレビュー傾向をサマリーで把握
- HPゼロ演出で場を和ませる

# 注意点
指摘数のカウント基準はプロジェクトごとに調整可能ですが、レビューコメントの形式によっては誤集計が発生する場合があります。最大HPや減少量もカスタマイズできます。

# 設計方針
シンプルなCLIとJSON保存で、他ツールとの連携や拡張も容易です。UIはRPGっぽさを重視しつつ、実用的なレビュー支援を目指しています。
