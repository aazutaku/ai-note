# 概要
このSkillは、作業中に意図的に脱力感を演出するための“無駄な進捗バー”を表示するものです。心理的な緊張緩和や、チーム内でのちょっとしたジョーク用途を想定しています。

# 公式ドキュメント抜粋
Pythonの標準ライブラリ（sys, time, random, argparse）だけで実装されており、外部依存はありません。進捗バーはターミナル互換の環境で動作します。

# 利用例
- `/random-os-fake-update-progrebar` を明示的に呼び出して、会議や作業の合間に表示
- コマンド実行時に自動発動させて、緊張を和らげる演出

# 注意点
このスキルは実務的な意味を一切持たず、進捗バーが100%になっても何も起きません。進捗内容は毎回ランダムで、実在のOSやファイルには一切影響を与えません。履歴やログも残りません。

# 設計方針
- 10秒以内に自動消去し、作業の邪魔にならない
- 進捗内容・表示タイミングは完全ランダム
- コードは100行以上で可読性と拡張性を重視
- 標準出力のみを使用し、外部ライブラリ不使用
