# 機能概要
このSkillは、開発現場や作業中に突如として“謎のOS緊急アラート”をデスクトップ通知で表示し、場の空気を一瞬でカオスに染め上げる演出ツールです。通知内容は「危険：バグが自我を獲得しました」「注意：メモリがピクニック中」など、業務と無関係な意味不明メッセージのみ。実害ゼロで、真面目な空気や張り詰めた現場に笑いを提供します。

# 使い方
- 明示的な呼び出し: `/skills menu` で一覧から選択、または `desktop-chaos-alert` を直接メンション
- 暗黙的な発動: 「アラート」「緊急」「混乱」「カオス」「通知」などのキーワードが会話や作業ログに登場した際に自動発動

# 出力例
```
通知: 危険：バグが自我を獲得しました
通知: 注意：メモリがピクニック中
通知: 深刻：キーボードが人生に迷っています
通知: 警告：CPUが詩を書き始めました
通知: 注意：OSが昼寝を要求しています
```

# 注意点
- 通知はローカル環境のデスクトップ通知APIを利用し、履歴やファイル保存は行いません
- 業務やシステムに影響を与えることはありませんが、通知が不要な場合は `--disable` オプションで停止可能です
- Linux/macOS/Windowsの主要デスクトップ環境に対応しています
