# 機能概要
このスキルは、作業中に突然“謎のOS再起動進捗バー”を画面端やメニューバーにランダム表示します。進捗バーは「再起動準備中…」「謎のプロセス終了中…」など、毎回異なる不条理なメッセージとともに進行し、実際には何も再起動されません。ジョークや気分転換、集中しすぎた作業の合間に「全てがリセットされる」かのようなカオスな体験を提供します。

# 使い方
このSkillは明示的なコマンド呼び出しは不要です。文脈に「再起動」「リセット」「進捗バー」「プロセス終了」などのキーワードが現れた際、または長時間の作業や集中状態が続くと自動で発動します。設定により発動頻度や表示位置のカスタマイズも可能です。

# 出力例
```
[==        ] 23% | 謎のプロセス終了中…
[=====     ] 52% | システム設定を初期化中…
[=======   ] 70% | ほぼ完了、あと3分で全てがリセットされます
[==========] 100% | 再起動完了！何も変わりませんでした。
```

# 注意点
本Skillは実際のシステム再起動やプロセス終了などの操作は一切行いません。進捗バーとメッセージの表示のみを行い、ユーザーデータや設定には影響を与えません。ローカルファイルや履歴の保存もありませんが、演出のために一時的に端末や画面端に表示されることがあります。
