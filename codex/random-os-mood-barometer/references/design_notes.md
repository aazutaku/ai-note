# 概要
random-os-mood-barometerは、コマンド実行時に気分や生産性を“完全ランダム”で演出するSkillです。ユーザーの作業リズムに遊び心を加えることで、単調な作業や長時間の開発の合間にちょっとした変化をもたらします。

# 公式ドキュメント抜粋
本SkillはPython標準のrandomモジュールのみを利用し、外部依存を持ちません。コマンドライン引数処理にはargparseを用いています。

# 利用例
- ターミナルで毎回違う気分モードが表示されることで、作業の合間に小さな楽しみや話題を提供
- /skills menuから明示的に呼び出して、その場の雰囲気を和らげる

# 注意点
- 判定内容は完全ランダムで、実際の作業内容や生産性とは無関係です
- 除外コマンド（ls, cdなど）では表示されません

# 設計方針
- 1行表示で邪魔になりすぎないこと
- 気分モード・テンプレートの拡張性を確保
- ログ保存や一覧表示など、CLIツールとしての実用性も考慮
