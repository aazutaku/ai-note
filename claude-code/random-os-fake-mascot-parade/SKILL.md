# 機能概要
このSkillは、架空のOSマスコットたちがあなたの作業中に突如デスクトップ通知で現れ、意味不明な激励や警告を残して去っていく演出を提供します。マスコット名やセリフは毎回完全ランダムで生成され、カーネルくんやメモリねずみ、シェルパンダなど、OS風味のキャラクターが登場します。作業の合間に「なぜこれが…」と正気を疑いたくなる、実用性ゼロのネタ系Skillです。

# 使い方
- 明示呼び出し: `/random-os-fake-mascot-parade` で即座に通知を発生させます。
- 暗黙発動: 「マスコット」「応援」「OS」「通知」などのキーワードを含む文脈や、一定時間作業が続いたタイミングで自動発動します。

# 出力例
```
[通知] カーネルくん: 「システムの安定は君の努力次第！」
[通知] メモリねずみ: 「バグ撲滅運動中。応援よろしく！」
[通知] シェルパンダ: 「一服推奨。コマンド入力しすぎ注意！」
[通知] スワップタコ: 「仮想記憶の海で泳いでます」
[通知] デバイスきつね: 「新しいUSBを認識しました（嘘）」
```

# 注意点
- 通知内容は毎回完全ランダムです。
- 通知頻度は自動調整され、うるさすぎないよう設計されています。
- ローカルに通知履歴等は保存されません。
- 実在のOSやマスコットとは一切関係ありません。
