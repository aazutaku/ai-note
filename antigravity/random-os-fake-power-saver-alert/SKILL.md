# 機能概要
このSkillは、作業中のユーザーに対して、あたかもOSが発したかのような“パワーセーバー警告”をデスクトップ通知でランダムに表示します。全く根拠のない省エネ指示（例：「CPU温存のため10分間タイピング禁止」など）を炸裂させ、職場や自宅で謎の笑いと混乱を演出します。実害ゼロ、ネタ度MAXの暇つぶし・気分転換に特化した演出Skillです。

# 使い方
このSkillは明示的な呼び出しは不要です。Antigravityが「省エネ」「パワーセーブ」「通知」「ジョーク」「暇つぶし」などのキーワードや、集中作業中の雰囲気を検知すると自動的に発動します。手動テストやデバッグ時は `python fake_power_saver_alert.py --test` で即時通知を表示できます。

# 出力例
```
[通知] OSパワーセーバー警告: CPU温存のため10分間タイピング禁止
[通知] 電力節約のため思考も控えめにお願いします
[通知] マウス移動過多: 省エネモード発動中
[通知] ディスプレイ輝度自動減光中。目を細めてご利用ください
[通知] 無意味なクリックが多発しています。省エネのため自重を推奨
```

# 注意点
- このSkillはユーザー操作やPCの実際の省電力設定には一切影響しません
- 通知内容は完全なジョークであり、実効性はありません
- 通知履歴や内容はローカルには保存されません
- Windows/macOS/Linuxの主要デスクトップ環境で動作しますが、通知APIの制限がある場合は表示されないことがあります
