# 概要
context-switch-notesは、複数プロジェクトやタスクを並行して進める開発現場で、作業の流れや思考の断絶を防ぐためのメモ自動管理Skillです。切り替え時の「直前の状態」を自動記録・復元し、再開効率を高めます。

# 公式ドキュメント抜粋
- https://ai-note.tech/docs/context-switch-notes
- CLIサブコマンド: log/list/show/summary

# 利用例
- プロジェクトAからBに切り替える際、Aの作業要約が自動保存され、Bに戻ると直前の要約が自動表示される。
- 明示的な呼び出しで任意のメモを記録・確認可能。

# 注意点
- メモはローカルJSONで管理。クラウド同期や外部API連携は未対応。
- 保存件数制限（3件）により冗長化を防止。

# 設計方針
- シンプルなCLIとファイル管理で堅牢性を重視。
- 冗長なメモや重複保存を避け、要約を自動抽出。
