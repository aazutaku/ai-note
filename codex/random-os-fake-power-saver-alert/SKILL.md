# 機能概要
本Skillは、PC作業中に突然“OSのパワーセーバー警告”を装ったデスクトップ通知をランダムに表示します。通知内容は「CPU温存のため10分間タイピング禁止」や「マウス移動過多：省エネモード発動」など、全く根拠も効力もない省エネ指示が毎回異なる内容で現れ、職場や自宅での作業に意図的な混乱や笑いを提供します。実害はなく、ちょっとした息抜きやネタ演出、会話のきっかけ作りに最適です。

# 使い方
- 明示呼び出し例:
  - `/skills random-os-fake-power-saver-alert`
  - `$random-os-fake-power-saver-alert`
- 暗黙発動キーワード例:
  - 「省エネ」「パワーセーブ」「集中力」「眠い」「作業疲れ」などのワードが会話やコードコメントに現れた際に自動発動します。

# 出力例
```
[通知] OSパワーセーバー警告: CPU温存のため10分間キーボード操作を控えてください。
[通知] 電力節約モード: マウスの動きが多すぎます。しばらく静止してください。
[通知] 省エネ推奨: 画面を見つめすぎです。目を閉じて深呼吸しましょう。
[通知] パワーセーブ: 思考が過熱しています。アイデアの発散を10分間抑制してください。
[通知] バッテリー保護: 机の上のコーヒー消費量を減らしてください。
```

# 注意点
- 通知は実際のOSや電源管理には一切影響を与えません。
- ローカルに記録や履歴は保存されません。
- 長時間の連続通知や業務用端末での使用は推奨しません。
- 通知の内容はすべてフィクションです。
