# 機能概要
このSkillは、作業中のデスクトップに「現在の緊張度」をテロップとしてランダムなタイミングで表示します。緊張度は、commit数やタイピング速度など“根拠になりそうでならない”値や完全ランダムから算出され、「平常心」「やや焦り」「極度緊張」などの謎ラベルで演出。実用性は皆無ですが、無意味なプレッシャーや職場の雰囲気を盛り上げる無駄な演出を提供します。

# 使い方
- 明示呼び出し: `/skills menu` から「desktop-tension-meter」を選択、または `$desktop-tension-meter` と入力
- 暗黙発動キーワード: 「緊張」「プレッシャー」「テロップ」「デスクトップ通知」などを含む会話やコマンド

# 出力例
```
[14:23:10] 緊張度: やや焦り (推定値: 57)
[14:31:04] 緊張度: 平常心 (推定値: 22)
[14:42:55] 緊張度: 極度緊張 (推定値: 91)
[14:57:13] 緊張度: 平常心 (推定値: 18)
[15:10:21] 緊張度: やや焦り (推定値: 48)
```
デスクトップ画面端に上記のようなテロップが5-10秒表示されます。

# 注意点
- 緊張度は実用的な測定値ではなく、完全なジョークです。
- 通知が鬱陶しい場合はスクリプトを停止してください。
- ローカルPC上でのみ動作し、データ保存や外部送信は行いません。
- 強制終了してもデータ損失はありません。
