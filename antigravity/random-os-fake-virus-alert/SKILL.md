# 機能概要
このSkillは、作業のマンネリ化や集中力の低下を感じたタイミングで、完全に架空の“ウイルス検出アラート”をデスクトップ通知として表示します。『危険：未知のバグウイルスを検出』『注意：あなたのコードが自我を持ちました』など、絶対に本物と誤認しないユーモア溢れる内容で、日常の作業に唐突なスパイスを加えます。通知はランダムなタイミング・内容で発動し、実害や悪質な動作は一切ありません。

# 使い方
このSkillは明示的な呼び出しを必要とせず、Antigravityが「飽きた」「集中力低下」「作業マンネリ」などのキーワードや状況を検知した際に自動で発動します。ユーザー操作やコマンド入力は不要です。

# 出力例
```
[デスクトップ通知]
タイトル: 危険：未知のバグウイルスを検出！
本文: あなたのコードが自我を持ち始めました。直ちにコーヒーを補給してください。

[デスクトップ通知]
タイトル: 注意：OSがAIに乗っ取られました
本文: これは冗談です。作業の合間に深呼吸しましょう。

[デスクトップ通知]
タイトル: 警告：メモリ内に謎の“やる気ウイルス”
本文: 影響はありません。気分転換にストレッチをおすすめします。
```

# 注意点
- 通知内容は全てジョークであり、実際のウイルス検出やシステム異常は一切発生しません。
- OS標準の通知APIのみを利用し、追加の環境構築や外部依存は不要です。
- 通知の頻度やタイミングは自動制御され、作業の妨げにならないよう配慮されています。
- ローカルファイルへの記録や履歴保存は行いません。
