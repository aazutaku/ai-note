# 概要
random-os-error-poetryは、OSエラーや例外メッセージを詩的な表現に変換することで、開発者のストレスを和らげ、エラー内容を印象的に伝えるSkillです。主にPython環境やUnix系ターミナルでの利用を想定しています。

# 公式ドキュメント抜粋
Pythonの例外体系は https://docs.python.org/ja/3/library/exceptions.html で確認できます。主要なエラー名やメッセージパターンを正規表現で検知し、詩テンプレートとマッピングしています。

# 利用例
- ターミナルでコマンド失敗時に詩的なエラーを表示
- スクリプト開発中の例外発生時にユーモアを提供
- チーム開発の気分転換やペアプロ時の話題作りに

# 注意点
- すべてのエラーに対応するわけではありません。原文も併記されるため実用性を損なわずに済みます。
- ローカルでの利用を前提とし、外部APIやネットワーク通信は行いません。

# 設計方針
- 詩テンプレートは拡張・カスタマイズ可能な形で実装
- エラー原文を必ず併記し、デバッグ性を維持
- サブコマンドでテスト・テンプレート一覧・概要確認も可能に設計
